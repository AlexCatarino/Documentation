Mia is an LLM-powered chatbot that can write QuantConnect algorithms, debug your existing code issues, and has deep knowledge of the QuantConnect API. It boasts several powerful features:

Context Aware

Mia knows the project contents and recent error messages during compiling, backtests, or live algorithms. When asked she can interpret the error message and quickly debug the relevant code that needs fixing.

Code Validation Tools

Mia has a suite of tools to write high-quality code. When asked to write strategies, she uses these to validate (and fix) code before returning the strategies to users. She even runs quick backtests to ensure the code works as expected.
