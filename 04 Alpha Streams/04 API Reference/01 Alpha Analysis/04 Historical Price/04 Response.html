The /alpha/{id}/prices API accepts requests in the following format:

200 Success

Price Model - Record of the price of an Alpha or a bid made for it.
time number
example: 1520785125.21
Unix timestamp of the last time the price was updated.
price-type string Enum
The type of price update. Can be bid, ask or trade. Only ask supported at this time. Options : ['bid', 'ask', 'trade']
shared-price number
example: 1300
Shared price for this moment of time.
exclusive-price number
example: 4500
Exclusive price for this moment of time.
Example 
[
  {
    "time": 1520785125.21,
    "price-type": "bid",
    "shared-price": 1300,
    "exclusive-price": 4500
  }
]

401 Authentication Error

UnauthorizedError Model - Unauthorized response from the API. Key is missing, invalid, or timestamp is too old for hash.
www_authenticate string
Header

Default Generic Error

Error Model
success boolean
example: False
Boolean indicating true-false; for error models this is always false.
messages string Array
Array of error messages from the API.
Example 
{
  "success": False,
  "messages": [
    "string"
  ]
}
