The /alpha/{id}/errors API accepts requests in the following format:

Error Model
success boolean
example: False
Boolean indicating true-false; for error models this is always false.
messages string Array
Array of error messages from the API.
Example 
{
  "success": False,
  "messages": [
    "string"
  ]
}
/alpha/{id}/errors Method
id string
Unique id hash of an Alpha published to the marketplace.
start number
Optional. Starting position for the search query. With very large datasets the results can be paginated and fetched in chunks starting from start. Default: 0.
Example 
{
  "id": "string",
  "start": 0,
}
