The /alpha/{id} API accepts requests in the following format:

200 Success

Alpha Model - Snapshot of a Project at the time it was deployed to the marketplace, written by an Author on QuantConnect, generating Insights about Assets.
id string
example: 78fc2972b9ac597369d86a0f98d9316e
Unique hash identifier for this published Alpha.
authors Author Array
Creators of the Alpha. Sometimes users collaborate on a Project or have one trader paired with a coder. Authors can be added and removed from projects.
asset-classes string Array
Asset classes predicted in this stream.
accuracy Point Array
Daily rolling accuracy of the Alpha module over the last 30 days of insight direction score.
analyses-performed number
Number of backtests/analysis trials used to generate this Alpha model.
author-trading boolean
Boolean indicating whether the author is actively trading this alpha on their own account.
description string
Author supplied paragraph description of the Alpha behavior.
estimated-depth number
Estimated market depth available for this asset based on trading volumes at the time of Insights.
exclusive-available boolean
Flag to indicate if the author is open to selling the signal exclusively.
exclusive-subscription-fee number
Monthly fee for exclusive access to the Alpha signal.
estimated-effort number
Estimated number of hours the user was working on this Alpha.
listed-time number
Unix timestamp the Alpha was listed into the marketplace.
project Project object
Project object where the Alpha source resides. One Project can have multiple generated Alphas.
uniqueness number
Indicator of uniqueness in in the QuantConnect marketplace based on the assets traded and the overlap of signal with other Alphas in the Client portfolio, where percentage of overlap with other Alphas (100% is identical).
sharpe-ratio number
Sharpe ratio of the Alpha with a $1M portfolio, equal weighting portfolio construction and immediate execution models. NOTE: This is currently not implemented.
subscription-fee number
Monthly fee levied by the author on the Alpha signal.
status string Enum
Alphas are forced-running by default but after 10 production runtime errors they are taken offline. Options : ['running', 'stopped']
tags string Array
Array of string tags to describe the alpha and its algorithm properties.
version string
example: 1.0.2
Version of the Alpha. The production version of the Alpha.

Author Model - Author user of QuantConnect responsible for creation of an Alpha.
id string
example: 623b06b231eb1cc1aa3643a46
Unique string hash id for Author.
alphas string Array
String array of Alpha-Id hashes the Author has created.
alphas-listed integer
example: 4
Number of Alphas Author has listed in the marketplace.
analysis-average-length integer
example: 520
Number of days of the average analysis(backtest) length for the Author.
backtests integer
example: 1300
Lifetime number of analysis(backtests) conducted by the Author.
biography string
example: Physics professor from North East USA with background in nuclear fusion.
Profile auto-biography (100-500 words).
forum-discussions integer
example: 5
Number of forum discussions started by the Author.
forum-comments integer
example: 34
Number of forum comments made by the Author.
language string Enum
example: Py
Author preferred programming language. Options : ['Py', 'C#']
last-online-time number
example: 1520785125.21
Unix timestamp of the last time the Author was online (Updated daily).
location string
example: Brooklyn, New York, US
Best known estimate of the Author geographic location.
projects integer
example: 34
Total count of the number of projects the Author has in account.
signup-time number
example: 1520785125.21
Unix timestamp of the user sign up.
social-media string Array
Array of social media profile links.

last-modified-time number

parent-id number
