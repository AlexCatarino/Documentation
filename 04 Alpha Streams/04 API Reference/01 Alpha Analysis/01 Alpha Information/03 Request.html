The /alpha/{id} API accepts requests in the following format:

ReadDataLinkResponse Model - Response from reading purchased data.
link string
Link to the data.
success boolean
Indicate if the API request was successful.
errors string Array
List of errors with the API call.
Example 
{
  "link": "string",
  "success": true,
  "errors": [
    "string"
  ]
}
/alpha/{id} Method
id string
Unique id hash of an Alpha published to the marketplace.
