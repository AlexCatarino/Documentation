The /alpha/{id}/conversation/create API accepts requests in the following format:

/alpha/{id}/conversation/create Method
id string
Unique id hash of an Alpha published to the marketplace.
from string
Email that is going to be used to send the replies from the author(s).
subject string
The subject of the thread, this is going to be used for the email.
message string
Message to be sent to the author(s).
cc string
Optional. Comma separated list of emails that are going to be copied into the author(s) replies.
Example 
{
  "id": "string",
  "from": "string",
  "subject": "string",
  "message": "string",
  "cc": "string",
}
