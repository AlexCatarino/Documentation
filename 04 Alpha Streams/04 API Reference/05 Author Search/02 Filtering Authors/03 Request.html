The /alpha/author/search API accepts requests in the following format:

/alpha/author/search Method
alphas-minimum integer
Optional. The minimum number of Alphas the Author has listed. Minimum: 0
alphas-maximum integer
Optional. The maximum number of Alphas the Author has listed.
biography string
Optional. Search the Author biography information for this text.
forum-comments-minimum integer
Optional. Minimum number of comments created on QuantConnect. Minimum: 0
forum-comments-maximum integer
Optional. Maximum number of comments created on QuantConnect.
forum-discussions-minimum integer
Optional. Minimum number of discussions started on QuantConnect. Minimum: 0
forum-discussions-maximum integer
Optional. Maximum number of discussions started on QuantConnect.
last-login-minimum integer
Optional. Minimum of the unix timestamp of the Author last login on QuantConnect. Minimum: 0
last-login-maximum integer
Optional. Maximum of the unix timestamp of the Author last login on QuantConnect.
languages array Enum
Optional. Preferred programming language for the primary Author. Options: ['C#', 'Py']
location string
Optional. Best guess geographic location of the Author based on the IP address.
projects-minimum integer
Optional. Minimum number of projects the user has created. Minimum: 0
projects-maximum integer
Optional. Maximum number of projects the user has created.
signed-up-minimum integer
Optional. The minimum unix timestamp of the Author registration on QuantConnect.
signed-up-maximum integer
Optional. The maximum unix timestamp of the Author registration on QuantConnect.
start number
Optional. Starting position for the search query. With very large datasets the results can be paginated and fetched in chunks starting from start. Default: 0
