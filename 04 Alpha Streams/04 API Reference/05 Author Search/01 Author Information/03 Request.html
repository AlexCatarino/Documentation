The /alpha/author/{id} API accepts requests in the following format:

Error Model
success boolean
example: False
Boolean indicating true-false; for error models this is always false.
messages string Array
Array of error messages from the API.
Example 
{
  "success": False,
  "messages": [
    "string"
  ]
}
/alpha/author/{id} Method
id string
Unique id hash of an Author.
Example 
{
  "id": "string",
}
