The /alpha/{id}/subscribe API accepts requests in the following format:

Successfully searched for passed query parameters; returns an array of Alpha objects.
value string
/
Example 
[
  "fd4bc7ce493c54b7453f2fd2a72eaa29"
]
/alpha/{id}/subscribe Method
id string
Unique id hash of an Alpha published to the marketplace.
exclusive boolean
Optional. Try and subscribe to the Alpha with an exclusive preference.
Example 
{
  "id": "string",
  "exclusive": true,
}
