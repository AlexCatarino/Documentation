The / API provides a response in the following format:

200 Success

InsightPacket Model - Wrapper packet to hold collection of Insight objects
alpha-id object
example: 623b06b231eb1cc1aa3643a46
Unique Alpha Id which generated this Insight.
algorithm-id object
example: A-78fc2972b9ac597369d86a0f98d9316e
Deploy id for this algorithm.
insights Insight Array
Array of insights emitted at this timestep.
Example 
[
  {
    "alpha-id": "623b06b231eb1cc1aa3643a46",
    "algorithm-id": "A-78fc2972b9ac597369d86a0f98d9316e",
    "insights": [
      {
        "id": "dd9527c4dfde4b54b4c46fd95fe39049",
        "confidence": 0.21,
        "created-time": 1520785125.21,
        "close-time": 1520785125.21,
        "direction": "up",
        "type": "price",
        "source": "in sample",
        "period": 600,
        "reference-value": 0,
        "symbol": "AAPL R735QTJ8XC9X",
        "ticker": "AAPL"
      }
    ]
  }
]
Insight Model - Individual prediction from an Alpha.
id string
example: dd9527c4dfde4b54b4c46fd95fe39049
Unique hash id of this Insight.
confidence number
example: 0.21
Confidence of the Insight as a percentage.
created-time number
example: 1520785125.21
Unix timestamp for when the Alpha Insight was created.
close-time number
example: 1520785125.21
Unix timestamp for when the Alpha Insight was closed (start + period).
direction string Enum
Direction of the insight; flat, up or down. Options : ['up', 'down', 'flat']
type string Enum
Insight prediction type category: price or volatility. Options : ['price', 'volatility']
source string Enum
Description of the state of the model emissions (backtest, live or out of sample). Options : ['in sample', 'out of sample', 'live trading']
period number
example: 600
Prediction period of the insight in seconds.
reference-value number
Sample of the asset price at the time the insight was created.
symbol string
example: AAPL R735QTJ8XC9X
Uniquely identified securities-contract based on QuantConnect.Symbol type. All information to uniquely identify a contract.
ticker string
example: AAPL
Current ticker for this asset.
Example 
{
  "id": "dd9527c4dfde4b54b4c46fd95fe39049",
  "confidence": 0.21,
  "created-time": 1520785125.21,
  "close-time": 1520785125.21,
  "direction": "up",
  "type": "price",
  "source": "in sample",
  "period": 600,
  "reference-value": 0,
  "symbol": "AAPL R735QTJ8XC9X",
  "ticker": "AAPL"
}
AlphaMode Model - Description of the state of the model emissions (backtest, live or out of sample)
AlphaMode string Enum
Description of the state of the model emissions (backtest, live or out of sample). Options : ['in sample', 'out of sample', 'live trading']
Example 
{
  "AlphaMode": "in sample"
}

Uniquely identified securities-contract based on QuantConnect.Symbol type. All information to uniquely identify a contract.
Example 
{

}
