The /alpha/search API accepts requests in the following format:

/alpha/search Method
accuracy-minimum number
Optional. Search for Alphas which have a specific accuracy minimum. Minimum: 0
accuracy-maximum number
Optional. Search for Alphas which have a specific accuracy maximum. Maximum: 1
asset-classes array Enum
Optional. Comma separated value array of asset classes to search. Options: ['equity', 'forex', 'future', 'option', 'cfd', 'crypto']
author string
Optional. Hash author id identifier to locate.
estimated-value-minimum number
Optional. Search for Alphas which have the specified minimum estimated-value. Minimum: 0
estimated-value-maximum number
Optional. Search for Alphas which have the specified maximum estimated-value.
exclude string
Optional. Comma separated list of tags to exclude from the search results.
fee-minimum number
Optional. Fee minimum filter on the listed Alphas. Minimum: 0
fee-maximum number
Optional. Fee maximum filter on the listed Alphas.
include string
Optional. Comma separated list of tags to include in the search results.
project-id integer
Optional. Search for Alphas created by a specific Project.
sharpe-minimum number
Optional. Search for Alphas which have the specified maximum sharpe ratio.
sharpe-maximum number
Optional. Search for Alphas which have the specified maximum sharpe ratio.
symbols Insight Array
Optional. Search for Alphas which utilize specific symbols. QuantConnect symbol identifier code.
start number
Optional. Starting position for the search query. With very large datasets the results can be paginated and fetched in chunks starting from start. Default: 0
uniqueness-minimum number
Optional. Search for Alphas which meet the specified minimum uniqueness value relative to existing portfolio. Uniqueness is a measure of correlation of the return curve approximation. Minimum: 0
uniqueness-maximum number
Optional. Search for Alphas which meet the specified maximum uniqueness value relative to existing portfolio. Uniqueness is a measure of correlation of the return curve approximation. Minimum: 1
