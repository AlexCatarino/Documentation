To create a project, call the CreateProject method with the project name, the project programming language, and the organization name.

#load "../Initialize.csx"
#load "../QuantConnect.csx"
using QuantConnect;

var projectResponse = api.CreateProject(name: "Project", 
                                        language: Language.CSharp,
                                        organizationId: "<yourOrganizationName>");
project_response = api.CreateProject(name="Project", 
                                     language=Language.Python,
                                     organizationId="<your_organization_name>")

The organizationId argument is optional. If you omit it, the CLI adds the project to your preferred organization.
