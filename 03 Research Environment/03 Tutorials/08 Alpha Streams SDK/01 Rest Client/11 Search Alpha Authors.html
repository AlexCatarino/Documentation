Knowing who created the Alpha can be an important search tool to improve your filtering. Participating Author information can be searched in a similar manner to the Alpha Search. Each supplied criteria is a logical AND on the query.

  1. Call SearchAuthors with list of programming language(s) and location, or more selection criteria.
    2. var authors = client.SearchAuthors( new SearchAuthorsRequest{ Languages=new List<string>{"C#"}, Location="Virginia" } );
    authors = client.SearchAuthors( SearchAuthorsRequest( languages=["C#"], location="Virginia" ) )
  2. Display the results.

    3. In this tutorial, we display the last 5 insights.

    Console.WriteLine($"Located {authors.Count()} authors matching your criteria.");

}
