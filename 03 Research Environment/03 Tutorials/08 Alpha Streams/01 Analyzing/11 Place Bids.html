Create New Bid

  1. Call CreateBid with CreateBidPriceRequest method that wrapped the alpha ID, bid price, allocation amount, period for liscensing and valid time for the bid. The method will return a status of your bid.

        Id = "d0fc88b1e6354fe95eb83225a",
        Bid = 3,
        Allocation = 10000,
        Period = 1,
        GoodUntil = ((DateTimeOffset)DateTime.Today + TimeSpan.FromDays(1)).ToUnixTimeSeconds()
    }
);
