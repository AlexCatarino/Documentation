You need to subscribe to all of the Equities for which you want fundamental historical data. You can request historical data for a subset of the Equities or all of the Equities.

Subset of Equities

Create subscriptions for multiple Equities and then call the GetFundamental method with a list of Symbols to request fundamental historical data for those specific securities.

var fb = qb.AddEquity("FB").Symbol;
var amzn = qb.AddEquity("AMZN").Symbol;
var aapl = qb.AddEquity("AAPL").Symbol;
var nflx = qb.AddEquity("NFLX").Symbol;
var goog = qb.AddEquity("GOOG").Symbol;

var startTime = new DateTime(2017, 1, 1);
var endTime = DateTime.Now;
var marketCaps = qb.GetFundamental(new []{fb, amzn, aapl, nflx, goog}, "CompanyProfile.MarketCap",
    startTime, endTime);
var fb = qb.AddEquity("FB").Symbol
var amzn = qb.AddEquity("AMZN").Symbol
var aapl = qb.AddEquity("AAPL").Symbol
var nflx = qb.AddEquity("NFLX").Symbol
var goog = qb.AddEquity("GOOG").Symbol

start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
marketCaps = qb.GetFundamental([fb, amzn, aapl, nflx, goog], "CompanyProfile.MarketCap",
    startTime, endTime);

All Equities

Call the GetFundamental method with qb.Securities.Keys to request fundamental historical data for those specific securities for all of the Equities.

var marketCaps = qb.GetFundamental(qb.Securities.Keys, "CompanyProfile.MarketCap",
    startTime, endTime);
marketCaps = qb.GetFundamental(qb.Securities.Keys, "CompanyProfile.MarketCap",
    start_time, end_time)
