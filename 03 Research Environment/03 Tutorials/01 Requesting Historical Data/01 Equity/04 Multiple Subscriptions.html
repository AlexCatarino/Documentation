Follow the steps of the previous sections to request historical data for multiple securities:

Subscribe to Equity

Instantiate a QuantBook and then call AddEquity to subscribe to an Equity security for multiple tickers by using the following snippet:

var qb = new QuantBook();
var spy = qb.AddEquity("SPY").Symbol;
var tlt = qb.AddEquity("TLT").Symbol;
var gld = qb.AddEquity("GLD").Symbol;

qb = QuantBook()
spy = qb.AddEquity("SPY").Symbol
tlt = qb.AddEquity("TLT").Symbol
gld = qb.AddEquity("GLD").Symbol

Make Historical Data Request

The three ways to make historical data requests are available. The first argument of the History method can be a list of Symbol for a single request as follows:

var start = new DateTime(2021, 1, 1);
var end = new DateTime(2021, 2, 1);
var definedPeriodHistory = qb.Historyv(new []{spy, tlt, gld}, start, end, Resolution.Daily);
var timespanHistory = qb.History(new []{spy, tlt, gld}, TimeSpan.FromDays(10), Resolution.Daily);
bar barCountHistory = qb.History(new []{spy, tlt, gld}, 10, Resolution.Daily);

start = datetime(2021, 1, 1)
end = datetime(2021, 2, 1)
defined_period_history = qb.History([spy, tlt, gld], start, end, Resolution.Daily)
timespan_history = qb.History([spy, tlt, gld], timedelta(days=10), Resolution.Daily)
bar_count_history = qb.History([spy, tlt, gld], 10, Resolution.Daily)

Alterativelly, the History method can make a historical data request that incluses all securities that has been susbcribed as follows:

History method can be a list of Symbol for a single request as follows:

var timespanHistory = qb.History(TimeSpan.FromDays(10), Resolution.Daily);
var barCountHistory = qb.History(10, Resolution.Daily);

timespan_history = qb.History(qb.Securities.Keys, timedelta(days=10), Resolution.Daily)
