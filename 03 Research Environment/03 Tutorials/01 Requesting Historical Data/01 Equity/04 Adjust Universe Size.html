You need to subscribe to all of the Equities for which you want historical data. You can request historical data for a subset of the Equities or all of the Equities.

Subset of Equities

Create subscriptions for multiple Equities and then call the History method with a list of Symbols to request historical data for those specific securities.

var qb = new QuantBook();
var spy = qb.AddEquity("SPY").Symbol;
var tlt = qb.AddEquity("TLT").Symbol;
var gld = qb.AddEquity("GLD").Symbol;

var startTime = new DateTime(2021, 1, 1);
var endTime = new DateTime(2021, 2, 1);
var definedPeriodHistory = qb.History(new []{spy, tlt, gld}, startTime, endTime, Resolution.Daily);
var timespanHistory = qb.History(new []{spy, tlt, gld}, TimeSpan.FromDays(10), Resolution.Daily);
bar barCountHistory = qb.History(new []{spy, tlt, gld}, 10, Resolution.Daily);
qb = QuantBook()
spy = qb.AddEquity("SPY").Symbol
tlt = qb.AddEquity("TLT").Symbol
gld = qb.AddEquity("GLD").Symbol

start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History([spy, tlt, gld], start_time, end_time, Resolution.Daily)
timespan_history = qb.History([spy, tlt, gld], timedelta(days=10), Resolution.Daily)
bar_count_history = qb.History([spy, tlt, gld], 10, Resolution.Daily)

All Equities

Call the History method with no Symbols to request historical data for all of the Equities.

Call the History method with qb.Securities.Keys, time argument(s), and resolution to request historical data for all of the Equities.

var startTime = new DateTime(2021, 1, 1);
var endTime = new DateTime(2021, 2, 1);
var definedPeriodHistory = qb.History(startTime, endTime, Resolution.Daily);
var timespanHistory = qb.History(TimeSpan.FromDays(10), Resolution.Daily);
var barCountHistory = qb.History(10, Resolution.Daily);
start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History(qb.Securities.Keys, start_time, end_time, Resolution.Daily)
timespan_history = qb.History(qb.Securities.Keys, timedelta(days=10), Resolution.Daily)
bar_count_history = qb.History(qb.Securities.Keys, 10, Resolution.Daily)

The History method returns an IEnumerable<Slice> for multiple-security requests.
