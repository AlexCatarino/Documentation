QuantConnect Research Environment has several Python packages for plotting data. In this tutorial, we will show two popular chart types.

Candlestick

The candlestick chart is a popular chart in technical analysis that displays the high, low, open, and closing prices of a security for a specific period.

import plotly.graph_objects as go

fig = go.Figure(data=[go.Candlestick(x=spy_history.index,
                                    open=spy_history['open'],
                                    high=spy_history['high'],
                                    low=spy_history['low'],
                                    close=spy_history['close'])],
                                    layout=go.Layout(
                                        title=go.layout.Title(text='SPY OHLC'),
                                        xaxis_title='Date',
                                        yaxis_title='Price',
                                        xaxis_rangeslider_visible=False
                                    ))

fig.show()
