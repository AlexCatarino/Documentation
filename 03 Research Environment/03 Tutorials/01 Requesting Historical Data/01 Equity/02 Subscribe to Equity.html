Follow these steps to subscribe to a security of type Equity to your QuantBook:

  1. Create an instance of QuantBook
  2. Call the AddEquity method to add a subscription for an Equity object
  3. Save an instance of the Symbol object for the Equity object
var qb = new QuantBook();
var equity = qb.AddEquity("SPY");
var symbol = equity.Symbol;

qb = QuantBook()
equity = qb.AddEquity("SPY")
symbol = equity.Symbol

By default, the susbcribed data is adjusted for splits and dividends, reflecting the historical request data output. We can change the normalization mode by using the following snippet:

equity.SetDataNormalizationMode(DataNormalizationMode.Raw);
