Follow these steps to susbcribe to a security of type Equity to your QuantBook:

  1. Create an instance of QuantBook
  2. Call the AddEquity method
var qb = new QuantBook();
var equity = qb.AddEquity("SPY");
var symbol = equity.Symbol;

qb = QuantBook()
equity = qb.AddEquity("SPY")
symbol = equity.Symbol

By default, this method susbcribes to data adjusted for dividends and splits which reflects on the historical request data output. We can change the normalization mode as follows:

equity.SetDataNormalizationMode(DataNormalizationMode.Raw);
