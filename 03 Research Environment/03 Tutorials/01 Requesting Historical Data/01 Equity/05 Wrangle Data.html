Run a cell in a Jupyter Notebook with the pandas.DataFrame as the last line to display the historical data.

Call the Console.WriteLine method in a Jupyter Notebook to display the historical data.

Select One Security

Iterate through the IEnumerable and index the Slice with a Symbol to get the historical data of the security.

Index the pandas.DataFrame with a Symbol to select the historical data of the security.

var history = history.Select(data => data[symbol]);
foreach(var bar in history.Take(5))
{
    Console.WriteLine(bar.ToString());
}

history.loc[symbol]

Select a Property of the Security

Iterate through the IEnumerable, index the Slice with a Symbol, and select a property of the Slice to get the historical values of the property.

Index the pandas.DataFrame with a Symbol to select the historical data of the security and then select a property column to get the historical values of the property.

var closePrices = history.Select(data => data[symbol].Close);
foreach(var closePrice in closePrices.Take(5))
{
    Console.WriteLine(closePrice.ToString());
}

history.loc[symbol]['close']

Unstack the Dataframe

If you request historical data for multiple securities, you can transform the pandas.DataFrame so that it is a time series of closing prices for all of the securities. Select the column that you want to display for each security and then call the unstack method to transform the pandas.DataFrame into the desired format. In the new format, the column indices are the Symbol of each security.

history['close'].unstack(level=0)
