The format of the returned object from the History method call depends on the language our Research Environment is set. In C#, this method returns an IEnumerable for a single-security request and an IEnumerable for a multi-security index. In Python, it returns a multi-index pandas.DataFrame.

Jupyter Notebooks are prepared to show the value of any variable using the Python Kernel. For C#, we need to rely on Console.WriteLine to visualize the values

Select one security:

var spyHistory = history.Select(data => data[spy]);
foreach(var bar in spyHistory.Take(5))
{
    Console.WriteLine(bar.ToString());
}

spy_history = history.loc[spy]

Select a specific property of the security:

var spyClosePrices = history.Select(data => data[spy].Close);
foreach(var closePrice in spyClosePrices.Take(5))
{
    Console.WriteLine(closePrice.ToString());
}

spy_close_prices = history.loc[spy]['close']

Reorganize the dataframe

We can a dataframe with a single property, e.g. "close", where the Symbol objects are the columns by unstacking the dataframe.

close_prices_by_symbols = history['close'].unstack(level=0)
