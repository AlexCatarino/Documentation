Follow these steps to subscribe to an Equity security:

  1. Instantiate a QuantBook.
    2. var qb = new QuantBook();
    qb = QuantBook()
  2. Call the AddEquity method with a ticker and resolution.
    3. var equity = qb.AddEquity("SPY", Resolution.Daily);
    equity = qb.AddEquity("SPY", Resolution.Daily)

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  3. Save a reference to the Equity Symbol.
    4. var symbol = equity.Symbol;
    symbol = equity.Symbol
  4. (Optional) Call the SetDataNormalizationMode method with a data normalization mode.
    5. equity.SetDataNormalizationMode(DataNormalizationMode.Raw);
    equity.SetDataNormalizationMode(DataNormalizationMode.Raw)

    If you don't pass a data normalization mode argument, DataNormalizationMode.Adjusted is used by default.

    The call to SetDataNormalizationMode above throws an error because it is obsolete. Use the following code snippet to avoid the warning and ensure all subscriptions have the same data normalization mode:

    qb.SubscriptionManager
  .SubscriptionDataConfigService
  .GetSubscriptionDataConfigs(equity.Symbol)
  .SetDataNormalizationMode(DataNormalizationMode.Raw);
