Follow these steps to subscribe to an Equity Option Chain security:

  1. Instantiate a QuantBook.
    2. var qb = new QuantBook();
    qb = QuantBook()
  2. Call the AddEquity method with a ticker and resolution.
    3. var equity = qb.AddEquity("SPY");
    equity = qb.AddEquity("SPY")

    If you do not pass a resolution argument, Resolution.Minute is used by default

  3. Save a reference to the Equity Symbol.
    4. var underlying = equity.Symbol;
    underlying = equity.Symbol
  4. Call the AddOption method with a ticker and resolution.
    5. var option = qb.AddOption("SPY");
    option = qb.AddOption("SPY")

    If you do not pass a resolution argument, Resolution.Minute is used by default

  5. (Optional) Call the SetFilter method with a specified expiration range value.
    6. option.SetFilter(-2, 2, 0, 90);
    option.SetFilter(-2, 2, 0, 90)

    If you don't call the SetFilter method, the default values of -1, 1, 0, and 35 will be used respectively.
