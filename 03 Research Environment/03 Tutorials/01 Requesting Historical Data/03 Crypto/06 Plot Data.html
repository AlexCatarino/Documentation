Jupyter Notebooks don't currently support packages to plot historical data, but we are working on adding the functionality. Until the functionality is added, use Python to plot historical Crypto data.

You need to subscribe to Cryptocurrencies and get their historical data to plot their historical data. You can use many of the supported libraries to plot data in various formats. For example, you can plot candlestick and line charts.

Candlestick Chart

Follow these steps to plot candlestick charts:

  1. Import the plotly package.
    2. import plotly.graph_objects as go
  2. Create a Candlestick.
    3. candlestick = go.Candlestick(x=history.index,
                             open=history['open'],
                             high=history['high'],
                             low=history['low'],
                             close=history['close'])
  3. Create a Layout.
    4. layout = go.Layout(title=go.layout.Title(text='BTCUSD OHLC'),
                   xaxis_title='Date',
                   yaxis_title='Price',
                   xaxis_rangeslider_visible=False)
  4. Create the Figure.
    5. fig = go.Figure(data=[candlestick], layout=layout)
  5. Show the Figure.
    6. fig.show()

    Candlestick charts display the open, high, low, and close prices of the security.

Line Chart

Follow these steps to plot line charts using built-in methods:

  1. Select data to plot.
    2. volume = history['volume'].unstack(level=0)
  2. Call the plot method on the pandas object.
    3. volume.plot(title="Volume", figsize=(15, 10))
  3. Show the plot.
    4. plt.show()

    Line charts display the value of the property you selected in a time series.
