You need to subscribe to all of the Cryptocurrencies for which you want historical data. You can request historical data for a subset of the Cryptocurrencies or all of the Cryptocurrencies.

Subset of Cryptocurrencies

Create subscriptions for multiple Cryptocurrencies and then call the History method with a list of Symbols to request historical data for those specific securities.

var btc = qb.AddCrypto("BTCUSD", market: Market.Bitfinex).Symbol
var eth = qb.AddCrypto("ETHUSD", market: Market.Bitfinex).Symbol
var sol = qb.AddCrypto("SOLUSD", market: Market.Bitfinex).Symbol

var startTime = new DateTime(2021, 1, 1);
var endTime = new DateTime(2021, 2, 1);
var definedPeriodHistory = qb.History(new []{btc, eth, sol}, startTime, endTime,
    Resolution.Daily);
var timespanHistory = qb.History(new []{btc, eth, sol}, TimeSpan.FromDays(10),
    Resolution.Daily);
var barCountHistory = qb.History(new []{btc, eth, sol}, 10, Resolution.Daily);
btc = qb.AddCrypto("BTCUSD", market=Market.Bitfinex).Symbol
eth = qb.AddCrypto("ETHUSD", market=Market.Bitfinex).Symbol
sol = qb.AddCrypto("SOLUSD", market=Market.Bitfinex).Symbol

start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History([btc, eth, sol], start_time, end_time,
    Resolution.Daily)
timespan_history = qb.History([btc, eth, sol], timedelta(days=10),
    Resolution.Daily)
bar_count_history = qb.History([btc, eth, sol], 10, Resolution.Daily)

All Cryptocurrencies

Call the History method with no Symbols to request historical data for all of the Cryptocurrencies.

Call the History method with qb.Securities.Keys, time argument(s), and resolution to request historical data for all of the Cryptocurrencies.

var timespanHistory = qb.History(TimeSpan.FromDays(10), Resolution.Daily);
var barCountHistory = qb.History(10, Resolution.Daily);
start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History(qb.Securities.Keys, start_time, end_time,
    Resolution.Daily)
timespan_history = qb.History(qb.Securities.Keys, timedelta(days=10),
    Resolution.Daily)
bar_count_history = qb.History(qb.Securities.Keys, 10, Resolution.Daily)

The History method returns an IEnumerable<Slice> for multiple-security requests.
