Jupyter Notebooks don't currently support packages to plot historical data, but we are working on adding the functionality. Until the functionality is added, use Python to plot historical Index Option data.

You need to subscribe to Index Options and get their historical contract data to plot their historical data. You can use many of the supported libraries to plot data in various formats. For example, you can plot candlestick and line charts.

Candlestick Chart

Follow these steps to plot candlestick charts:

  1. Call the History method with an OptionContract Symbol, start datetime, and end datetime.
    2. history = qb.History(symbol, start_time, end_time)
  2. Drop the first four index levels of the returned DataFrame.
    3. history.index = history.index.droplevel([0,1,2,3])
  3. (Optional) Select just one day of the returned history.
    4. history = history.loc[(history.index < start_time+timedelta(1))]
  4. Import the plotly package.
    5. import plotly.graph_objects as go
  5. Create a Candlestick.
    6. candlestick = go.Candlestick(x=history.index,
                             open=history['open'],
                             high=history['high'],
                             low=history['low'],
                             close=history['close'])
  6. Create a Layout.
    7. layout = go.Layout(title=go.layout.Title(text=f'{symbol.Value} OHLC'),
                   xaxis_title='Date',
                   yaxis_title='Price',
                   xaxis_rangeslider_visible=False)
  7. Create the Figure.
    8. fig = go.Figure(data=[candlestick], layout=layout)
  8. Show the Figure.
    9. fig.show()

    Candlestick charts display the open, high, low, and close prices of the contract.

Line Chart

Follow these steps to plot line charts using built-in methods:

  1. Call the History method with the OpenInterest class, OptionContract Symbols, and a timedelta.
    2. history = qb.History(OpenInterest, symbols, timedelta(days=10))
  2. Drop the first three index levels of the returned DataFrame.
    3. history.index = history.index.droplevel([0,1,2])
  3. Select the data to plot.
    4. open_interests = history['openinterest'].unstack(level=0)
  4. Rename the columns to the Symbol of each contract.
    5. open_interests.columns = [
    Symbol.GetAlias(SecurityIdentifier.Parse(x), index.Symbol)
        for x in open_interests .columns]
  5. Call the plot method on the pandas object.
    6. open_interests.plot(title="Open Interest", figsize=(15, 4))
  6. Show the plot.
    7. plt.show()

    Line charts display the value of the property that you selected in a time series.
