Follow these steps to subscribe to alternative data security from QuantConnect datasets:

  1. Instantiate a QuantBook.
    2. var qb = new QuantBook();
    qb = QuantBook()
  2. Call the AddData method with a ticker and resolution.
    3. var cboe = qb.AddData<CBOE>("VIX", Resolution.Daily);
    cboe = qb.AddData(CBOE, "VIX", Resolution.Daily)

    If you do not pass a resolution argument, the default resolution of the alternative data is used by default. On the other hand, if you pass a resolution that is not supported, it will be ignored.

  3. Save a reference to the Alternative data Symbol.
    4. var symbol = cboe.Symbol;
    symbol = cboe.Symbol
