Follow these steps to subscribe to a Forex security:

  1. Instantiate a QuantBook.
    2. var qb = new QuantBook();
    qb = QuantBook()
  2. Call the AddForex method with a ticker and resolution.
    3. var forex = qb.AddForex("EURUSD", Resolution.Daily);
    forex = qb.AddForex("EURUSD", Resolution.Daily)

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  3. Save a reference to the Forex Symbol.
    4. var symbol = forex.Symbol;
    symbol = forex.Symbol
