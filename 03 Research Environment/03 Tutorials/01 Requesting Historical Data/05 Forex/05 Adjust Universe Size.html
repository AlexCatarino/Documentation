You need to subscribe to all of the Forex pairs for which you want historical data. You can request historical data for a subset of the Forex pairs or all of the Forex pairs.

Subset of Forex pairs

Create subscriptions for multiple Forex pairs and then call the History method with a list of Symbols to request historical data for those specific securities.

var qb = new QuantBook();
var eur = qb.AddForex("EURUSD").Symbol;
var nzd = qb.AddForex("NZDUSD").Symbol;
var gbp = qb.AddForex("GBPEUR").Symbol;
var startTime = new DateTime(2021, 1, 1);
var endTime = new DateTime(2021, 2, 1);
var definedPeriodHistory = qb.History(new []{eur, nzd, gbp}, startTime, endTime, Resolution.Daily);
var timespanHistory = qb.History(new []{eur, nzd, gbp}, TimeSpan.FromDays(10), Resolution.Daily);
var barCountHistory = qb.History(new []{eur, nzd, gbp}, 10, Resolution.Daily);
qb = QuantBook()
eur = qb.AddForex("EURUSD").Symbol
nzd = qb.AddForex("NZDUSD").Symbol
gbp = qb.AddForex("GBPEUR").Symbol

start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History([eur, nzd, gbp], start_time, end_time, Resolution.Daily)
timespan_history = qb.History([eur, nzd, gbp], timedelta(days=10), Resolution.Daily)
bar_count_history = qb.History([eur, nzd, gbp], 10, Resolution.Daily)

All Forex pairs

Call the History method with no Symbols to request historical data for all of the Forex pairs.

Call the History method with qb.Securities.Keys, time argument(s), and resolution to request historical data for all of the Forex pairs.

var timespanHistory = qb.History(TimeSpan.FromDays(10), Resolution.Daily);
var barCountHistory = qb.History(10, Resolution.Daily);
start_time = datetime(2021, 1, 1)
end_time = datetime(2021, 2, 1)
defined_period_history = qb.History(qb.Securities.Keys, start_time, end_time, Resolution.Daily)
timespan_history = qb.History(qb.Securities.Keys, timedelta(days=10), Resolution.Daily)
bar_count_history = qb.History(qb.Securities.Keys, 10, Resolution.Daily)

The History method returns an IEnumerable<Slice> for multiple-security requests.
