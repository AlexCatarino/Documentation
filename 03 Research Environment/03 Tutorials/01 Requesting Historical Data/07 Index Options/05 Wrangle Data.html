The DataFrame that contains historical data for Options contracts has a MultiIndex with levels for the expiry, strike, OptionRight, Symbol, and timestamp. For some of the wrangling operations, you need to drop the first three index levels. To drop the index levels, run:

price_history.index = price_history.index.droplevel([0,1,2])
open_interest_history.index = open_interest_history.index.droplevel([0,1,2])
timespan_price_history.index = timespan_price_history.index.droplevel([0,1,2])
timespan_open_interest_history.index = timespan_open_interest_history.index.droplevel([0,1,2])

Unstack the Dataframe

If you request historical data for multiple contracts, you can transform the DataFrame so that it is a time series of bid closing prices for all of the contracts that it contains. Select the column that you want to display for each contract and then call the unstack method to transform the DataFrame into the desired format. You can rename the columns of the transformed DataFrame to be the respective OptionsContract Symbols.

To display the closing bid prices of all of the contracts, run:

timespan_price_history = timespan_price_history['bidclose'].unstack(level=0)
columns = [Symbol.GetAlias(SecurityIdentifier.Parse(x), index.Symbol)
    for x in timespan_price_history.columns]
timespan_price_history.columns = columns
timespan_price_history

The Jupyter Notebook displays a DataFrame that contains the closing bid prices of the contracts and has the Symbol of each OptionsContract as the column indices.

To display the open interest of all of the contracts, run:

timespan_open_interest_history = timespan_open_interest_history['openinterest'].unstack(level=0)
columns = [Symbol.GetAlias(SecurityIdentifier.Parse(x), index.Symbol)
    for x in timespan_open_interest_history.columns]
timespan_open_interest_history.columns = columns
timespan_open_interest_history

The Jupyter Notebook displays a DataFrame that contains the open interest of the contracts and has the Symbol of each OptionsContract as the column indices.
