Follow these steps to subscribe to a Future security:

  1. Instantiate a QuantBook.
    2. var qb = new QuantBook();
    qb = QuantBook()
  2. Call the AddFuture method with a ticker and resolution.
    3. var future = qb.AddFuture(Futures.Indices.SP500EMini,
                dataNormalizationMode: DataNormalizationMode.BackwardsRatio,
                dataMappingMode: DataMappingMode.LastTradingDay,
                contractDepthOffset: 0
            );
    future = qb.AddFuture(Futures.Indices.SP500EMini,
                dataNormalizationMode = DataNormalizationMode.BackwardsRatio,
                dataMappingMode = DataMappingMode.LastTradingDay,
                contractDepthOffset = 0
			)

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  3. Save a reference to the future Symbol.
    4. var symbol = future.Symbol;
    symbol = future.Symbol
  4. (Optional) Call the SetFilter method with a specified expiration range value.
    5. future.SetFilter(0, 90);
    future.SetFilter(0, 90)

    If you don't use this method to pass a specified expiration range value, the Future chain will not be populated by futures contracts. This choice impacts one of the ways to get historical data we will see next.
