You can save data in the objectStore object in different formats.

Follow this steps to create data in different formats:

  1. Create example data to be stored 
    2. var sample = new Dictionary<string, int> { {"One", 1}, {"Two", 2}, {"Three", 3} };
  2. Convert the example data to string:
    3. var stringSample = string.Join(Environment.NewLine, sample);
  3. Convert the example data to byte:
    4. var bytesSample = Encoding.UTF8.GetBytes(stringSample);
  4. Convert the example data to JSON:

    5. Using SaveJson<T> method could store data as a JSON-encoded string object. The helper method will help serialize the data into JSON format automatically.

    var bytesSample = Encoding.UTF8.GetBytes(stringSample);

    sample.Select(kvp => new XElement(kvp.Key, kvp.Value)));

SaveBytes

Call the SaveBytes method could store the byte list object.

objectStore.SaveBytes("saveBytes", bytesSample);
object_store.SaveBytes("save_bytes", bytes_sample)

The following methods are helper methods of SaveBytes that is internally called by them.

The following method is helper method of SaveBytes that is internally called by it.

Save

Call the Save or SaveString method could store data as a string object.

objectStore.Save("saveString", stringSample);
object_store.Save("save_string", string_sample)

SaveJson

Call the SaveJson<T> method to store data as a JSON object.

objectStore<Dictionary<string, int>>.SaveJson("saveJson", sample);

SaveXml

Call the SaveXml<T> method to store data the XML object. 

objectStore.SaveXml("saveXml", xmlSample);
