The ObjectStore is accessible in the root of your algorithm. It has the following methods available for storing data.

Save

Using .Save() method could store data as a string object.

  1. Example data to be stored.
    2. var sample = new Dictionary<string, int> {["One"] = 1, ["Two"] = 2, ["Three"] = 3};
    sample = {"One": 1, "Two": 2, "Three": 3}
  2. Convert data into string.
    3. var stringSample = string.Join(Environment.NewLine, sample);
    string_sample = str(sample)
  3. Store into ObjectStore.
    4. qb.ObjectStore.Save("saveString", stringSample);
    qb.ObjectStore.Save("save_string", string_sample)
  4. Use ContainsKey to check if the key-value pair is in the ObjectStore.
    5. qb.ObjectStore.ContainsKey("saveString")
    qb.ObjectStore.ContainsKey("save_string")

SaveBytes

Using .SaveBytes() method could store data as a bytes object.

  1. Convert data into bytes.
    2. var bytesSample = Encoding.UTF8.GetBytes(stringSample);
    import json
bytes_sample = json.dumps(string_sample).encode('utf-8')
  2. Store into ObjectStore.
    3. qb.ObjectStore.SaveBytes("saveBytes", bytesSample);
    qb.ObjectStore.SaveBytes("save_bytes", bytes_sample)
  3. Use ContainsKey to check if the key-value pair is in the ObjectStore.
    4. qb.ObjectStore.ContainsKey("saveBytes")
    qb.ObjectStore.ContainsKey("save_bytes")

SaveJson

Using .SaveJson<T>() method could store data as a JSON-encoded string object. The helper method will help serialize the data into JSON format automatically.

This method is intended for C# projects.

  1. Store into ObjectStore.
    2. qb.ObjectStore.SaveJson("saveJson", sample);
  2. Use ContainsKey to check if the key-value pair is in the ObjectStore.
    3. qb.ObjectStore.ContainsKey("saveJson")

SaveXml

Using .SaveXml<T>() method could store data as a XML-encoded string object.

This method is intended for C# projects.

  1. Convert the data into XML format.
    2. XElement xmlSample = new XElement("sample",
    sample.Select(kvp => new XElement(kvp.Key, kvp.Value)));
Console.Write(xmlSample.ToString());

    3. qb.ObjectStore.SaveXml("saveXml", xmlSample);
  3. Use ContainsKey to check if the key-value pair is in the ObjectStore.
