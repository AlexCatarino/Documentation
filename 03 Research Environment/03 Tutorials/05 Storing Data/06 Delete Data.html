You can delete data from the ObjectStore with the Delete method.

It is good practice to check wether the ObjectStore has the key to the object you want to delete. If not present, the method will raise an exception.

Single Item

  1. Use ContainsKey to check if the key-value pair is in the ObjectStore.
    2. objectStore.ContainsKey("saveXml")
    object_store.ContainsKey("save_bytes")
  2. Delete data from ObjectStore by its key.
    3. objectStore.Delete("saveXml");
    object_store.Delete("save_bytes")
  3. (Optional) Use ContainsKey to check if the key-value pair is in the ObjectStore to confirm the deletion.

All Items

  1. Iterate the ObjectStore object, and call the Delete method for every key.
    2. foreach(var kvp in objectStore)
{
    objectStore.Delete(kvp.Key);
