We can use the key to call the stored data in ObjectStore. It has the following methods available for reading data.

Read

Using .Read() method could read data as from a stored string object.

  1. Read from ObjectStore.
    2. var stringData = qb.ObjectStore.Read("saveString");
    string_data = qb.ObjectStore.Read("save_string")
  2. Restore as the dictionary.
    3. var dict = stringData.Split(new[] {'\n'}, StringSplitOptions.RemoveEmptyEntries)
               .Select(part => part.Split('[')[1]).Select(part => part.Split(']')[0]).Select(part => part.Split(','))
               .ToDictionary(split => split[0], split => split[1]);
    string_data = eval(string_data)
  3. Call assert to check if the read and original data are the same. If no error message displayed, the two objects are the same.
    4. assert string_data == sample

ReadBytes

Using .ReadBytes() method could read data from a stored bytes object.

  1. Read from ObjectStore.
    2. var stringData = qb.ObjectStore.Read("saveString");
    string_data = qb.ObjectStore.Read("save_string")
  2. Convert to string.
  3. Call json.loads to restore the data string.
    4. var stringFromBytes = Encoding.UTF8.GetString(bytesData);
    byte_data = json.loads(bytearray(byte_data).decode('utf-8'))
  4. Restore as the dictionary.
    5. var dict = stringFromBytes.Split(new[] {'\n'}, StringSplitOptions.RemoveEmptyEntries)

               .ToDictionary(split => split[0], split => split[1]);

    6. assert byte_data == sample

ReadJson

Using .ReadJson<T>() method could read data from a stored JSON-encoded string object.

This method is intended for C# projects.

  1. Read data from ObjectStore.
    2. var jsonData = qb.ObjectStore.ReadJson<Dictionary<string, int>>("saveJson");

ReadXml

Using .ReadXml<T>() method could read data from a stored XML-encoded string object.

This method is intended for C# projects.

  1. Read data from ObjectStore.
    2. var xmlData = qb.ObjectStore.ReadXml<XElement>("saveXml");
  2. Reconstruct the dictionary.
    3. var dict = xmlData.Elements().ToDictionary(x => x.Name.LocalName, x => int.Parse(x.Value));
