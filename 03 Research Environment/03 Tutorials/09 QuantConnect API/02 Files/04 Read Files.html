Read A Specific File

Follow these steps to read details of a file by the QuantConnect Api.

  1. Call ReadProjectFile with the project ID and the file name.
    2. var files = api.ReadProjectFile(projectId, "newFile.cs");
    files = api.ReadProjectFile(project_id, "new_file.py")
  2. Call Files attribute, then obtain the first file.
    3. var file = files.Files[0];
    file = files.Files[0]
  3. Display the file's details.
    4. Console.WriteLine($@"Name: {file.Name}
Last Modified: {file.DateModified}
Content: {file.Code}")
    print(f"""Name: {file.Name}
Last Modified: {file.DateModified.strftime("%Y/%m/%d %H:%M:%S")}
Content: {file.Code}""")
