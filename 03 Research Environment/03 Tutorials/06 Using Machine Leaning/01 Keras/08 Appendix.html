You can use some helper methods to get all of the ObjectStore keys and clear the ObjectStore.

Get ObjectStore Keys

To get all of the ObjectStore keys, call:

def get_ObjectStore_keys():
    return [str(j).split(',')[0][1:] for _, j in enumerate(qb.ObjectStore.GetEnumerator())]

Clear ObjectStore

To clear all of the objects in the ObjectStore, call:

def clear_ObjectStore():
    for key in get_ObjectStore_keys():
        qb.ObjectStore.Delete(key)
