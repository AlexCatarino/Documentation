Jupyter Notebooks don't currently support packages to plot historical data, but we are working on adding the functionality. Until the functionality is added, use Python to plot historical Equity Options data.

We can call the in-built plot method of pandas.DataFrame to plot the indicators.

  1. Select the columns/features for plotting.
    2. bb_plot = bb_indicator[["upperband", "middleband", "lowerband", "price"]]
sma_of_rsi_plot = sma_of_rsi_indicator[["simplemovingaverage"]]
  2. Call plot to plot the indicator.
    3. bb_plot.plot(figsize=(12, 8), title=f"20-day 2-STD Bollinger Bands of {symbol}")
atr_indicator.plot(figsize=(12, 8), title=f"10-day ATR of {symbol}")
vwap_indicator.plot(figsize=(12, 8), title=f"10-day VWAP of {symbol}")
sma_of_rsi_plot.plot(figsize=(12, 8), title=f"10-day SMA of 10-day RSI of {symbol}")
  3. Show the plots.
    4. plt.show()
