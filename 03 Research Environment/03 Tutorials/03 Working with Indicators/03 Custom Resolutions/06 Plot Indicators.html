Jupyter Notebooks don't currently support packages to plot historical data, but we are working on adding the functionality. Until the functionality is added, use Python to plot historical Equity Options data.

We can call the in-built plot method of pandas.DataFrame to plot the indicators.

bb_df.plot(figsize=(12, 8), title=f"Bollinger Band Width of {symbol}")
atr_df.plot(figsize=(12, 8), title=f"Average True Range of {symbol}")
vwap_df.plot(figsize=(12, 8), title=f"Volume-weighted Average Price of {symbol}")
sma_of_rsi_df.plot(figsize=(12, 8), title=f"SMA of RSI of {symbol}")
