You must import the plotting packages and get some historical data to create histogram.

Follow these steps to create a histogram:

  1. Select the column you want to plot.

    2. For example, to select the closing prices, run:

    close_price = history['close'].unstack(level=0)
  2. Call the pct_change method on the pandas object.
    3. daily_ret = close_price.pct_change()
  3. Set up the figure with figsize setting.
    4. plt.figure(figsize=(15, 10))
  4. Call hist method with the daily return data, bin number, and labels.
    5. plt.hist(daily_ret, bins=20, label=daily_ret.columns)
  5. Set up the title, x-axis label, y-axis label and label legend of the plot.
    6. plt.title('Are normally distributed returns a hoax?')
plt.xlabel('Daily Return')
plt.ylabel('Count')
plt.legend()
  6. Show the plot.
    7. plt.show()

    The Jupyter Notebook displays a histogram.
