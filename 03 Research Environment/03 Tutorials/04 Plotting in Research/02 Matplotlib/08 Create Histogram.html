You must import the plotting packages and get some historical data to create histogram.

Follow these steps to create a histogram:

Prepare the Data

  1. Select the Symbol that you want to plot in the line chart.

    2. For example, to select the first Symbol, run:

    symbol = symbols[0]
  2. Slice the history DataFrame with symbol and then select the column that you want to plot.

    3. For example, to select the closing prices, run:

    data = history.loc[symbol]['close']
  3. Call the pct_change method on the pandas object.
    4. daily_ret = data.pct_change()[1:]

Create Normal Distribution Curve

We'd like to also plot the normal distribution values for comparison. To do so, we must first create the corresponding values of the probability desity curve (PDF).

  1. Get the mean and standard deviation of the daily return Series.
    2. mean = daily_ret.mean()
std = daily_ret.std()
  2. Call linspace with lower limit, upper limit and number data points for the x-axis values.
    3. x = np.linspace(-3*std, 3*std, 1000)
  3. Calculate the y-axis values for the normal distribution PDF.
    4. pdf = 1/(std * np.sqrt(2*np.pi)) * np.exp(-(x-mean)**2 / (2*std**2))

Plot the Data

  1. Set up the figure with figsize setting.
    2. plt.figure(figsize=(15, 10))
  2. Call plot method to plot the normal distribution curve.
    3. plt.plot(x, pdf, label="Normal Distribution")
  3. Call hist method with the daily return data and bin number.
    4. plt.hist(daily_ret, bins=20)
  4. Set up the title, x-axis label and y-axis label of the plot.
    5. plt.title(f'{symbol} Return Distribution')
plt.xlabel('Daily Return')
plt.ylabel('Count')
  5. Show the plot.
    6. plt.show()
