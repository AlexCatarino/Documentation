You must import the plotting packages and get some historical data to create pie charts.

We would use a pie chart to present our asset allocation over various bank stocks and bank sector ETF. In this example, we'll use inverse variance as our allocation strategy.

Follow these steps to create a pie chart:

  1. Select the column you want to plot.

    2. For example, to select the closing prices, run:

    close_price = history['close'].unstack(level=0)
  2. Call the pct_change method on the pandas.DataFrame object.
    3. daily_ret = close_price.pct_change()
  3. Call var to obtain the variance of daily return.
    4. var = daily_ret.var()
  4. Convert the variance into inverse variance for our allocation.
    5. inv_var = 1 / var
  5. Set up the figure with figsize setting.
    6. plt.figure(figsize=(15, 10))
  6. Call pie method with the pandas.Series, labels and percentage display format.
    7. plt.pie(inv_var, labels=inv_var.index, autopct='%1.1f%%')
  7. Set up the title of the plot.
    8. plt.title('Banking Stocks and Bank Sector ETF Allocation')
  8. Show the plot.
    9. plt.show()
