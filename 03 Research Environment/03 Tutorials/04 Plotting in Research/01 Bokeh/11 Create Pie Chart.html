You must import the plotting packages and get some historical data to create pie charts.

We would use a pie chart to present our asset allocation over various bank stocks and bank sector ETF. In this example, we'll use inverse variance as our allocation strategy.

Follow these steps to create a pie chart:

  1. Select the column you want to plot.

    2. For example, to select the closing prices, run:

    close_price = history['close'].unstack(level=0)
  2. Call the pct_change method on the pandas.DataFrame object.
    3. daily_ret = close_price.pct_change()
  3. Call var to obtain the variance of daily return.
    4. var = daily_ret.var()
  4. Convert the variance into normalized inverse variance for our allocation.
    5. inv_var = 1 / var
inv_var = inv_var/np.sum(inv_var)   # Normalization
inv_var *= np.pi*2    # For a full circle circumference in radian
  5. Convert the pd.Series into pd.DataFrame
    6. inv_var = pd.DataFrame(inv_var, columns=["inverse variance"])
  6. Reset index of the pandas.DataFrame.
    7. inv_var.reset_index(inplace=True)
  7. Configure the colors.
    8. inv_var['color'] = Category20c[len(inv_var.index)]
  8. Call figure to create a plot with title.
    9. plot = figure(title=f"Banking Stocks and Bank Sector ETF Allocation")
  9. Call wedge method to create a pie chart.
    10. plot.wedge(x=0, y=1, radius=0.6, start_angle=cumsum('inverse variance', include_zero=True), 
            end_angle=cumsum('inverse variance'), line_color="white", fill_color='color', 
            legend_field='symbol', source=inv_var)
  10. Call show to show the plot.
    11. show(plot)

    The Jupyter Notebook displays a pie chart.
