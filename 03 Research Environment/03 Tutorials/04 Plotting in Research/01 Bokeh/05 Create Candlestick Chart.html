You must import the plotting packages and get some historical data to create candlestick charts.

In this example, we'll create a candlestick chart that shows the open, high, low, and close prices of one of the banking securities. Follow these steps to create the candlestick chart:

  1. Select a Symbol.
    2. symbol = symbols[0]
  2. Slice the history DataFrame with the symbol.
    3. data = history.loc[symbol]
  3. Divide the price in positive or negtive intraday change.
    4. inc = data['close'] > data['open']
dec = data['open'] > data['close']
  4. Call figure to create a plot with title and axis labels.
    5. plot = figure(title=f"{symbol} Close Price", x_axis_label='Date', y_axis_label='Price')
  5. Call segment to plot the end points of the bars.
    6. plot.segment(data.index, data['high'], data.index, data['low'], color="black")
  6. Call vbar to plot the bars.
    7. width = 12*60*60*1000
plot.vbar(data.index[inc], width, data['open'][inc], data['close'][inc], fill_color="green", line_color="green")
plot.vbar(data.index[dec], width, data['open'][dec], data['close'][dec], fill_color="red", line_color="red")
  7. Call show to show the plot.
    8. show(plot)
