You must import the plotting packages and get some historical data to create line charts.

Follow these steps to create a line chart:

  1. Select the Symbol you want to plot.

    2. For example, to select the first Symbol, run:

    symbol = symbols[0]
  2. Slice the history DataFrame with symbol and then select the column that you want to plot.

    3. For example, to select the close price, run:

    data = history.loc[symbol]['close']
  3. Call figure to create a plot with title and axis labels.
    4. plot = figure(title=f"{symbol} Close Price", x_axis_label='Date', y_axis_label='Price')
  4. Call line to plot a line plot for each Symbol with x-axis value, y-axis value, legend lable, color and line width.
    5. plot.line(data.index, data, legend_label=symbol.Value, color="blue", line_width=2)
  5. Call show to show the plot.
    6. show(plot)

    The Jupyter Notebook displays a line plot.
