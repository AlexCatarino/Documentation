You must import the plotting packages and get some historical data to create line charts.

Follow these steps to create a line chart:

  1. Select the column you want to plot.

    2. For example, to select the closing prices, run:

    close_price = history['close'].unstack(level=0)
  2. Call figure to create a plot with title and axis labels.
    3. plot = figure(title=f"Bank Stocks and Bank Sector ETF Close Price", x_axis_label='Date', y_axis_label='Price')
  3. Call line to plot a line plot for each Symbol with x-axis value, y-axis value, legend lable and line width.
    4. for symbol in symbols:
    color = np.random.randint(low=0, high=255, size=3)
    plot.line(close_price.index, close_price[symbol], legend_label=symbol.Value, color=tuple(color), line_width=2)
  4. Call show to show the plot.
    5. show(plot)
