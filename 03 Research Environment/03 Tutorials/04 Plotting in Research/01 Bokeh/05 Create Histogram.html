You must import the plotting packages and get some historical data to create histogram.

Follow these steps to create a histogram:

  1. Select the Symbol that you want to plot in the line chart.

    2. For example, to select the first Symbol, run:

    symbol = symbols[0]
  2. Slice the history DataFrame with symbol and then select the column that you want to plot.

    3. For example, to select the closing prices, run:

    data = history.loc[symbol]['close']
  3. Call the pct_change method on the pandas object.
    4. daily_ret = data.pct_change()
  4. Call np.histogram method on the pandas object with density indicator and bin number to obtain the bar frequnecy and edges values.
    5. hist, edges = np.histogram(daily_ret, density=True, bins=20)
  5. Call figure to create a plot with title and axis labels.
    6. plot = figure(title=f"{symbol} Daily Return Distribution", x_axis_label='Return', y_axis_label='Frequency')
  6. Call quad to plot a histogram with the bars' top, bottom, left-side and right-side values, fill and edge colors, and transparency.
    7. plot.quad(top=hist, bottom=0, left=edges[:-1], right=edges[1:],
          fill_color="navy", line_color="white", alpha=0.5)
  7. Call show to show the plot.
    8. show(plot)

    The Jupyter Notebook displays a histogram.
