You must import the plotting packages and get some historical data to create histograms.

We use a histogram to visualize the distribution of return of a selected stock.
Follow these steps to create a histogram:

  1. Select a Symbol.
    2. symbol = symbols[0]
  2. Slice the history DataFrame with the symbol and then select the close column.
    3. data = history.loc[symbol]['close']
  3. Call the pct_change method on the pandas.Series object.
    4. daily_ret = data.pct_change()
  4. Drop any NaN values by dropna to avoid regression error.
    5. daily_ret.dropna(inplace=True)
  5. Call the DataFrame constructor with the data Series and then call the reset_index method.
    6. daily_ret = pd.DataFrame(daily_ret).reset_index()
  6. Call the histplot method with the data, the column names of the x- and y-axis in data, the plot title and number of bins.
    7. plot = sns.histplot(daily_ret, x='close', bins=20)
  7. Set up the x-axis label, y-axis label and title.
    8. plot.set(xlabel='Return', 
         ylabel='Frequency', 
         title=f'{symbol} Daily Return of Close Price Distribution')
  8. Show the plot.
    9. plt.show()
