You must import the plotting packages and get some historical data to create line charts.

Follow these steps to create a line chart:

  1. Select the Symbol that you want to plot.

    2. For example, to select the first Symbol, run:

    symbol = symbols[0]
  2. Slice the history DataFrame with symbol and then select the column that you want to plot.

    3. For example, to select the closing prices, run:

    data = history.loc[symbol]['close']
  3. Convert the pd.Series into pd.DataFrame
    4. data = pd.DataFrame(data)
  4. Call the reset_index method with the inplace argument enabled.
    5. data.reset_index(inplace=True)
  5. Call lineplot with pandas.DataFrame, x-axis column and y-axis column to plot the line chart
    6. plot = sns.lineplot(data=data,
                    x='time', 
                    y='close'
                    )
  6. Set up the y-axis label and title.
    7. plot.set(ylabel="price", title=f"{symbol} Price Over Time")
  7. Show the plot.
    8. plt.show()
