Correlations between stocks in the same sector are commonly applied to develop pairs trading strategies. We will visualize the correlations between a few stocks in the banking sector using a heat map.

Gathering Data

To begin, we retrieve daily return data of close price for the bank stocks.

  1. Instantiate a QuantBook.
    2. qb = QuantBook()
  2. Create a ticker list of the selected tickers.
    3. tickers = [
    "COF", # Capital One Financial Corporation
    "JPM", # J P Morgan Chase & Co
    "GS", # Goldman Sachs Group, Inc.
    "WFC" # Wells Fargo & Company
]
  3. Call the AddEquity method with their corresponding resolution while iterrating the ticker list.
    4. symbols = [qb.AddEquity(ticker, Resolution.Daily).Symbol for ticker in tickers]

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  4. Call the History method with list of Symbol objects, time argument(s), and resolution to request historical data for all of the Securities.
    5. history = qb.History(tickers, 
                    datetime(2020, 2, 1), 
                    datetime(2020, 7, 1), 
                    Resolution.Daily)
  5. Select close price data.
    6. close_price = history['close'].unstack(level=0)
  6. Call the pct_change() method on the pandas.DataFrame object.
    7. daily_ret = close_price.pct_change()
  7. Create the correlation matrix.

    8. Before we create the heat map, we must create a correlation matrix, which has all the correlations between all possible pairs of symbols. This can be easily done with pandas.DataFrame's built-in corr() method.

    corr_matrix = daily_ret.corr()

Plot the Data

Now, we want to visualize the correlation matrix with color indicating the correlation level of banks stocks.

  1. Call sns.heatmap method with the correlation matrix and annotation setting.
    2. plot = sns.heatmap(corr_matrix, annot=True)
  2. Set up the title of the plot.
    3. plot.set(title='Bank Stocks Correlation Coefficients')
  3. Show the plot.
    4. plt.show()

    Heat map displays the correlation strength of the property you selected over a period of time.
