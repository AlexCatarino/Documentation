You must import the plotting packages and get some historical data to create candlestick charts.

Follow these steps to create a candlestick chart:

  1. Select the Symbol that you want to plot in the candlestick chart.

    2. For example, to select the last Symbol, run:

    symbol = symbols[-1]
  2. Slice the history DataFrame with the Symbol that you selected.
    3. data = history.loc[symbol]
  3. Create a Candlestick.
    4. candlestick = go.Candlestick(x=data.index,
                             open=data['open'],
                             high=data['high'],
                             low=data['low'],
                             close=data['close'])
  4. Create a Layout.
    5. layout = go.Layout(title=go.layout.Title(text=f'{symbol.Value} OHLC'),
                   xaxis_title='Date',
                   yaxis_title='Price',
                   xaxis_rangeslider_visible=False)
  5. Create a Figure.
    6. fig = go.Figure(data=[candlestick], layout=layout)
  6. Show the Figure.
    7. fig.show()
