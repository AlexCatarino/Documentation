You must import the plotting packages and get some historical data to create line charts.

Follow these steps to create a line chart:

  1. Select the Symbol that you want to plot in the candlestick chart.

    2. For example, to select the last Symbol, run:

    symbol = symbols[-1]
  2. Slice the history DataFrame with symbol and then select the column that you want to plot.

    3. For example, to select the closing prices, run:

    data = history.loc[symbol]['close']
  3. Call the reset_index method with the inplace argument enabled.
    4. data.reset_index(inplace=True)
  4. Call the line method with data, the column names of the x- and y-axis in data, and the plot title.
    5. fig = px.line(data, x='time', y='close', title=f'{symbol} Close price')
  5. Show the Figure.
    6. fig.show()

    The Jupyter Notebook displays a line chart.
