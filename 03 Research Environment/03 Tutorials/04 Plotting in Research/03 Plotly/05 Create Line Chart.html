You must import the plotting packages and get some historical data to create line charts.

We will visualize the close price of SPY using a time-series line chart.

Gathering Data

To begin, we retrieve close price data for SPY.

  1. Instantiate a QuantBook.
    2. qb = QuantBook()
  2. Call the AddEquity method with SPY, and their corresponding resolution.
    3. spy = qb.AddEquity("SPY", Resolution.Daily)

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  3. Call the History method with a Symbol, time argument(s), and resolution to request historical data for the symbol.
    4. history = qb.History(spy.Symbol, 360, Resolution.Daily)
  4. Index the pandas.DataFrame with a Symbol to select the historical data of the security and then select a property column to get the historical values of the property.
    5. close_price = history.loc["SPY"]["close"]

Plot the Data

Here, we visualize the close price of SPY with a time-series line plot.

  1. Reset index of the pandas.DataFrame.
    2. close_price2 = close_price.reset_index()
  2. Call px.line with pandas.DataFrame, x-axis column, y-axis column and title to plot the line chart.
    3. fig = px.line(close_price2, x='time', y='close', title='SPY Close price')
  3. Show the plot.
    4. fig.show()

    Line charts display the value of the property you selected in a time series.
