You must import the plotting packages and get some historical data to create scatter plots.

Follow these steps to create a scatter chart:

  1. Select the 2 Symbols that you want to plot in the scatter chart.

    2. For example, to select the first 2 selected bank stocks Symbols, run:

    symbol1 = symbols[1]
symbol2 = symbols[2]
  2. Slice the history DataFrame with each Symbol and then select the column that you want to plot.

    3. For example, to select the closing prices, run:

    close_price1 = history.loc[symbol1]['close']
close_price2 = history.loc[symbol2]['close']
  3. Call the pct_change and dropna methods on each Series object.
    4. daily_return1 = close_price1.pct_change().dropna()
daily_return2 = close_price2.pct_change().dropna()

    These calls return Series that contain the daily returns of each banking company.

  4. Call the scatter method with the 2 return Series, the trendline option, and axis labels.
    5. fig = px.scatter(x=daily_return1, y=daily_return2, trendline='ols', 
                 labels={'x': symbol1.Value, 'y': symbol2.Value})
  5. Call the update_layout method with a title.
    6. fig.update_layout(title=f'{symbol1.Value} vs {symbol2.Value} Daily % Returns');
  6. Show the Figure.
    7. fig.show()

    The Jupyter Notebook displays a scatter plot.
