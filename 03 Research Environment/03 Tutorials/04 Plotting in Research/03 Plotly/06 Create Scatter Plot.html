GLD and BAR are two different Gold-tracking ETFs, and we would like to visualize their correlation. To do this, we will employ a scatter plot of their returns.

Gathering Data

To begin, we retrieve the daily return of close price for GLD and BAR.

  1. Instantiate a QuantBook.
    2. qb = QuantBook()
  2. Call the AddEquity method with GLD and BAR, and their corresponding resolution. Save a reference to their Symbol.
    3. gld = qb.AddEquity("GLD", Resolution.Daily).Symbol
bar = qb.AddEquity("BAR", Resolution.Daily).Symbol

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  3. Call the History method with a list of Symbol objects, time argument, and resolution to request historical data for all of the Securities.
    4. history = qb.History([gld, bar], 360, Resolution.Daily)
  4. Index the pandas.DataFrame with a Symbol to select the historical data of the security and then select a property column to get the historical values of the property.
    5. gld_close_price = history.loc[gld]['close']
bar_close_price = history.loc[bar]['close']
  5. Call the pct_change() method on each pandas.Series object.
    6. gld_daily_ret = gld_close_price.pct_change()
bar_daily_ret = bar_close_price.pct_change()
  6. Drop any NaN values by dropna() to avoid regression error.
    7. gld_daily_ret.dropna(inplace=True)
bar_daily_ret.dropna(inplace=True)

Plot the Data

Now, we want to visualize the returns relationship of GLD and BAR.

  1. Call px.scatter with the 2 return pandas.Series, trendline option and axis-labels.
    2. fig = px.scatter(x=gld_daily_ret, y=bar_daily_ret, trendline='ols', labels={'x': 'GLD', 'y': 'BAR'})
  2. Update the layout with title.
    3. fig.update_layout(title='GLD vs BAR Daily % Returns')
  3. Show the plot.
    4. fig.show()

    Scatter plot displays the historical data of the property you selected over a period of time for 2 Series against each other.
