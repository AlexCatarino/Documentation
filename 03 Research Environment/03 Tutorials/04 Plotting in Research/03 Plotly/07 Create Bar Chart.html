You must import the plotting packages and get some historical data to create bar charts.

We use a bar chart to compare the average daily percent returns of various banking stocks and bank sector ETF.

Follow these steps to create a bar chart:

  1. Select the column you want to plot.

    2. For example, to select the closing prices, run:

    close_price = history['close'].unstack(level=0)
  2. Call the pct_change method on the pandas.DataFrame object. Multiplied by 100 to get the %.
    3. daily_ret = close_price.pct_change() * 100
  3. Call apply method on the pandas.DataFrame using np.nanmean to obtain the average return.
    4. avg_daily_ret = daily_ret.apply(np.nanmean, axis=0)
  4. Convert the pd.Series into pd.DataFrame
    5. avg_daily_ret = pd.DataFrame(avg_daily_ret, columns=["avg_daily_ret"])
  5. Reset index of the pandas.DataFrame.
    6. avg_daily_ret.reset_index(inplace=True)
  6. Call bar method with the pandas.DataFrame, x-axis column and y-axis column.
    7. fig = px.bar(avg_daily_ret, x='symbol', y='avg_daily_ret')
  7. Update the layout with title.
    8. fig.update_layout(title='Banking Stocks Average Daily % Returns')
  8. Show the Figure.
    9. fig.show()
