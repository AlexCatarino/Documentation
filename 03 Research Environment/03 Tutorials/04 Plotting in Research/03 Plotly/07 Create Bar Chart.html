You must import the plotting packages and get some historical data to create bar charts.

Follow these steps to create a bar chart:

  1. Select the column of data that you want to plot and then call the unstack method.

    2. For example, to select the closing prices, run:

    close_prices = history['close'].unstack(level=0)
  2. Call the pct_change method and then multiply by 100.
    3. daily_returns = close_prices.pct_change() * 100
  3. Call the mean method.
    4. avg_daily_returns = daily_returns.mean()
  4. Convert the Series into a DataFrame. 
    5. avg_daily_returns = pd.DataFrame(avg_daily_returns, columns=["avg_daily_ret"])
  5. Call the reset_index method with the inplace argument enabled.
    6. avg_daily_returns.reset_index(inplace=True)
  6. Call the bar method with avg_daily_returns and the axis column names.
    7. fig = px.bar(avg_daily_returns, x='symbol', y='avg_daily_ret')
  7. Call the update_layout method with a title.
    8. fig.update_layout(title='Banking Stocks Average Daily % Returns');
  8. Show the Figure.
    9. fig.show()

    The Jupyter Notebook displays a bar plot of the average daily percent returns of each of the banking securities.
