You must import the plotting packages and get some historical data to create heat maps.

Correlations between stocks in the same sector are commonly applied to develop pairs trading strategies. We will visualize the correlations between a few stocks in the banking sector and SPY using a heat map.

Follow these steps to create a bar chart:

  1. Select close price data.
    2. close_price = history['close'].unstack(level=0)
  2. Call the pct_change method on the pandas.DataFrame object.
    3. daily_ret = close_price.pct_change()
  3. Call corr to create the correlation matrix.

    4. Before we create the heat map, we must create a correlation matrix, which has all the correlations between all possible pairs of symbols. This can be easily done with pandas.DataFrame's built-in corr() method.

    corr_matrix = daily_ret.corr()
  4. Call imshow method with the correlation matrix, x-axis and y-axis labels.
    5. fig = px.imshow(corr_matrix, x=tickers, y=tickers)
  5. Update the layout with title.
    6. fig.update_layout(title='Banking Stocks Correlation Heat Map')
  6. Show the Figure.
    7. fig.show()
