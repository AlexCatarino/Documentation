We will visualize the open, high, low, close (OHLC) data of SPY using a candlestick chart.

Gathering Data

To begin, we retrieve the historical data of close price for SPY.

  1. Instantiate a QuantBook.
    2. qb = QuantBook()
  2. Call the AddEquity method with SPY, and their corresponding resolution.
    3. spy = qb.AddEquity("SPY", Resolution.Daily)

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  3. Call the History method with a Symbol, time argument(s), and resolution to request historical data for the Symbol.
    4. history = qb.History(spy.Symbol, 360, Resolution.Daily)
  4. Index the pandas.DataFrame with a Symbol to select the historical data of the security.
    5. price = history.loc["SPY"]

Plot the Data

Now, we want to visualize the OHLC data of SPY.

  1. Create a Candlestick.
    2. candlestick = go.Candlestick(x=price.index,
                            open=price['open'],
                            high=price['high'],
                            low=price['low'],
                            close=price['close'])
  2. Create a Layout.
    3. layout = go.Layout(title=go.layout.Title(text='SPY OHLC'),
                    xaxis_title='Date',
                    yaxis_title='Price',
                    xaxis_rangeslider_visible=False)
  3. Create the Figure.
    4. fig = go.Figure(data=[candlestick], layout=layout)
  4. Show the plot.
    5. fig.show()

    Candlestick Chart displays the price data of the Symbol you selected over a period of time.
