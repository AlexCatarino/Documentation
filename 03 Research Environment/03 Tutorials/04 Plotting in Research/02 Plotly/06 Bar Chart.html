We use a bar chart to compare the average daily percent returns of various banking stocks.

Gathering Data

First, we get historical closing prices for four banking sector stocks, then compute the average daily return for each stock.

  1. Instantiate a QuantBook.
    2. qb = QuantBook()
  2. Create a ticker list of the selected tickers.
    3. tickers = [
    "COF", # Capital One Financial Corporation
    "JPM", # J P Morgan Chase & Co
    "GS", # Goldman Sachs Group, Inc.
    "WFC" # Wells Fargo & Company
]
  3. Call the AddEquity method with their corresponding resolution while iterrating the ticker list.
    4. symbols = [qb.AddEquity(ticker, Resolution.Daily).Symbol for ticker in tickers]

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  4. Call the History method with list of Symbol objects, time argument(s), and resolution to request historical data for all of the Securities.
    5. history = qb.History(tickers, 
                    datetime(2020, 2, 1), 
                    datetime(2020, 7, 1), 
                    Resolution.Daily)
  5. Select close price data.
    6. close_price = history['close'].unstack(level=0)
  6. Call the pct_change() method on the pandas.DataFrame object.
    7. daily_ret = close_price.pct_change()
  7. Call .apply() function on the pandas.DataFrame using np.nanmean to obtain the average return.
    8. avg_daily_ret = daily_ret.apply(np.nanmean, axis=0)
  8. Convert the pd.Series into pd.DataFrame
    9. avg_daily_ret = pd.DataFrame(avg_daily_ret, columns=["avg_daily_ret"])

Plot the Data

Now, we want to visualize the average daily return of the banks stocks.

  1. Reset index of the pandas.DataFrame.
    2. avg_daily_ret2 = avg_daily_ret.reset_index()
  2. Call px.bar method with the pandas.DataFrame, x-axis column and y-axis column.
    3. fig = px.bar(avg_daily_ret2, x='symbol', y='avg_daily_ret')
  3. # Update the layout with title.
    4. fig.update_layout(title='Banking Stocks Average Daily % Returns')
  4. Show the plot.
    5. fig.show()

    Bar chart displays the average daily return data of the property you selected over a period of time.
