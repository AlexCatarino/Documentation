We will visualize the distribution of daily return of SPY and QQQ using a histogram.

Gathering Data

To begin, we retrieve daily return data of close price for SPY and QQQ

  1. Instantiate a QuantBook.
    2. qb = QuantBook()
  2. Call the AddEquity method with SPY and QQQ, and their corresponding resolution.
    3. spy = qb.AddEquity("SPY", Resolution.Daily)
qqq = qb.AddEquity("QQQ", Resolution.Daily)

    If you do not pass a resolution argument, Resolution.Minute is used by default.

  3. Call the History method with qb.Securities.Keys, time argument(s), and resolution to request historical data for all of the Securities.
    4. history = qb.History(qb.Securities.Keys, 360, Resolution.Daily)
  4. Select close price data.
    5. close_price = history['close'].unstack(level=0)
  5. Call the pct_change() method on the pandas object.
    6. daily_ret = close_price.pct_change()

Plot the Data

Now, we want to visualize the returns distribution of SPY and QQQ.

  1. Set up the figure with figsize setting.
    2. plt.figure(figsize=(15, 10))
  2. Call plt.hist method with the daily return data, bin number, and labels.
    3. plt.hist(daily_ret, bins=20, label=["SPY", "QQQ"])
  3. Set up the title, x-axis label, y-axis label and label legend of the plot.
    4. plt.title('Are normally distributed returns a hoax?')
plt.xlabel('Daily Return')
plt.ylabel('Count')
plt.legend()
  4. Show the plot.
    5. plt.show()

    Histogram displays the distribution of the property you selected over a period of time.
