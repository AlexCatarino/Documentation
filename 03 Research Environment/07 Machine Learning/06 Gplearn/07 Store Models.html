Saving the Model

We dump the models using the joblib module and save them to ObjectStore file paths. This way, the models don't need to be retrained, saving time and computational resources.

  1. Set the key names of the models to be stored in objectStore.
    2. transformer_key = "transformer"
regressor_key = "regressor"
  2. Call GetFilePath with the keys' names to get the file paths.
    3. transformer_file = qb.ObjectStore.GetFilePath(transformer_key)
regressor_file = qb.ObjectStore.GetFilePath(regressor_key)
  3. Call dump with the models and file paths to save the models to the file path.
    4. joblib.dump(gp_transformer, transformer_file)
joblib.dump(gp_regressor, regressor_file)
  4. Call Save with the key's name to save the model into ObjectStore.
    5. qb.ObjectStore.Save(transformer_key)
qb.ObjectStore.Save(regressor_key)

Loading the Model

Let's retrieve the models from ObjectStore file paths and load by joblib.

  1. Call the ContainsKey method.
    2. qb.ObjectStore.ContainsKey(transformer_key)
qb.ObjectStore.ContainsKey(regressor_key)

    This method returns a boolean that represents if the model_key is in the ObjectStore. If the ObjectStore does not contain the model_key, save the model using the model_key before you proceed.

  2. Call GetFilePath with the keys' names to get the file paths.
    3. transformer_file = qb.ObjectStore.GetFilePath(transformer_key)
regressor_file = qb.ObjectStore.GetFilePath(regressor_key)
  3. Call load with the file paths to fetch the saved models.
    4. loaded_transformer = joblib.load(transformer_file)
loaded_regressor = joblib.load(regressor_file)
