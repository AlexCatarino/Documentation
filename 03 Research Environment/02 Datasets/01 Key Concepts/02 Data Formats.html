The History method returns a pandas DataFrame. Pandas is a python data manipulation and analysis framework. A pandas DataFrame is a 2 dimensional labeled data structure, much like a spreadsheet or SQL table. A label on a row is called an Index and labels on columns are simply called columns.

In the preceding example, the History method returns an IEnumerable<TradeBar> collection. Depending on how you call the History method, the return type can be different. For more information about the return types, see Key History Concepts .

In the example below, a history request is made for all securities that been subscribed to. The returned Slice collection contains attributes of Trade and Quote data available.The resulting data frame has columns for each data type returned; these data types correspond to components of the Trade data or the Quote data available. Notice that because SPY and BTCUSD do not have Quote data, they have NaN values for those columns.
