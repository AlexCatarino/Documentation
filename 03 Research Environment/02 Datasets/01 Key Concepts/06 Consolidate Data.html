History requests usually return data in one of the standard resolutions. To analyze data on custom time frames like 5-minute bars or 4-hour bars, consolidate the data.

Resample

The resample method converts the frequency of a time series DataFrame into a custom frequency. Consider an example where you make a history call for minute resolution data and want to create 5-minute resolution data.

qb = QuantBook()

spy = qb.AddEquity("SPY")

startDate = datetime(2018, 4, 1)
endDate = datetime(2018, 7, 15)

df = qb.History(spy.Symbol, startDate, endDate, Resolution.Minute)

The resample method only works on DataFrame objects that have a datetime index. The History method returns a DataFrame with a multi-index. The first index is a Symbol index for each security and the second index is a time index for the timestamps of each row of data. To make the DataFrame compatible with the resample method, call the reset_index method to drop the Symbol index.

# Drop level 0 index (Symbol index) from the DataFrame
df.reset_index(level = 0, drop = True, inplace=True)

The resample method returns a Resampler object, which needs to be downsampled using one of the pandas downsampling computations. For example, you can use the Resampler.ohlc downsampling method to aggregate price data.

When you resample a DataFrame with the ohlc downsampling method, it creates an OHLC row for each column in the DataFrame. To just calculate the OHLC of the close column, select the close column before you resample the DataFrame. A resample offset of 5T corresponds to a 5-minute resample. Other resampling offsets include 2D = 2 days, 5H = 5 hours, and 3S = 3 seconds. 

close_prices = df["close"]

offset = "5T" 
close_5min_ohlc = close_prices.resample(offset).ohlc()
