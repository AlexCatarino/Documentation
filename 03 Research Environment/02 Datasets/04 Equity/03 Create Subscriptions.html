Follow these steps to subscribe to an Equity security:

  1. Load the required assembly files and data types.
    2. #load "../Initialize.csx"
#load "../QuantConnect.csx"

using QuantConnect;
using QuantConnect.Data;
using QuantConnect.Algorithm;
using QuantConnect.Research;
using QuantConnect.Indicators;
  2. Instantiate a QuantBook.
    3. var qb = new QuantBook();
    qb = QuantBook()
  3. Call the AddEquity method with a ticker and resolution and then save a reference to the Equity Symbol.
    4. var spy = qb.AddEquity("SPY", Resolution.Daily).Symbol;
var tlt = qb.AddEquity("TLT", Resolution.Daily).Symbol;
    spy = qb.AddEquity("SPY", Resolution.Daily).Symbol
tlt = qb.AddEquity("TLT", Resolution.Daily).Symbol

    If you do not pass a resolution argument, Resolution.Minute is used by default.

    The default data normalization mode is DataNormalizationMode.Adjusted. To change it, pass a dataNormalizationMode argument.

    qb.AddEquity("SPY", Resolution.Daily, dataNormalizationMode: DataNormalizationMode.Raw);
    qb.AddEquity("SPY", Resolution.Daily, dataNormalizationMode=DataNormalizationMode.Raw)
