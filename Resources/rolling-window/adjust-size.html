To adjust the RollingWindow size, set the Size property.

closeWindow.Size = 3;
tradeBarWindow.Size = 3;
quoteBarWindow.Size = 3;
self.close_window.Size = 3
self.trade_bar_window.Size = 3
self.quote_bar_window.Size = 3

You remove the most recently added values when you decrease RollingWindow.Size. You add a series of default values, e.g. zero or nullNone, when you increase RollingWindow.Size.

To increase the RollingWindow size, and set the new item's value, set the value of the nth element.

// Increase the Size to 10, and set the value of the newest element to 3
closeWindow[10] = 3;
# // Increase the Size to 10, and set the value of the newest element to 3
self.close_window[10] = 3
