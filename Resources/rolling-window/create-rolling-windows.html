To create rolling windows, instantiate the RollingWindow object of a given type with a maximum size.

closePriceWindow = new RollingWindow<decimal>(4);
tradeBarWindow = new RollingWindow<TradeBar>(2);
quoteBarWindow = new RollingWindow<QuoteBar>(2);
self._close_price_window = RollingWindow(4)
self._trade_bar_window = RollingWindow(2)
self._quote_bar_window = RollingWindow(2)
self._tuple_window = RollingWindow(2)

You can define the rolling window type with RollingWindow[T], where T is the native or C# type of elements in the window.

self._close_price_window = RollingWindow[float](4)
self._trade_bar_window = RollingWindow[TradeBar](2)
self._quote_bar_window = RollingWindow[QuoteBar](2)

The typeless version, RollingWindow(int), allows for Python types. For example, tuple.
