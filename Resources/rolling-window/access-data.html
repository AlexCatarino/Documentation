RollingWindow objects operate on a first-in, first-out process to allow for reverse list access semantics with positive and negative indexing. Index 0 refers to the most recent item in the window and the largest index refers to the last item in the window. Index -1 refers to the latest item as well.

var currentClose = closePriceWindow[0];
var previousClose = closePriceWindow[1];
var oldestClose = closePriceWindow[closePriceWindow.Count-1];
oldestClose = closePriceWindow[-1];   // Negative indexing
current_close = self._close_price_window[0]
previous_close = self._close_price_window[1]
oldest_close = self._close_price_window[self._close_price_window.count-1]
oldest_close = self._close_price_window[-1]   # Negative indexing

To get the item that was most recently removed from the RollingWindow, use the MostRecentlyRemovedmost_recently_removed property.

var removedClose = closePriceWindow.MostRecentlyRemoved;
removed_close = self._close_price_window.most_recently_removed
