To build the Range bars with a different property than the Valuevalue of the IBaseData object, provide a selector argument. The selector should be a function that receives the IBaseData object and returns a decimal value.

self._consolidator = ClassicRangeConsolidator(1, selector=lambda data: data.high)
_consolidator = new ClassicRangeConsolidator(1m, data => (data as TradeBar).High);
