To send a file to your FTP server in your code files, run:

Notify.Ftp(hostname, username, password, filePath, fileContent, port);
self.notify.ftp(hostname, username, password, file_path, file_content, port)

To send a file to your SFTP server in your code files, run:

Notify.Sftp(hostname, username, password, filePath, fileContent, port);
self.notify.Sftp(hostname, username, password, file_path, file_content, port)

You can use password authentication over SFTP using SSH keys.

Notify.Sftp(hostname, username, privateKey, privateKeyPassphrase, filePath, fileContent, port);
self.notify.Sftp(hostname, username, private_key, private_key_passphrase, file_path, file_content, port)

The port parameter is optional.
