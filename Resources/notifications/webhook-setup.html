To send webhook notifications in your code files, run:

var data = "";
var headers = new Dictionary<string, string> { { "Authorization", "Basic XXXXXX" } };
Notify.Web(address: "http://api.myserver.com", data: data, headers: headers);
data = ""
headers = { 'Authorization': 'Basic XXXXX' }
self.notify.web(address="http://api.myserver.com", data=data, headers=headers)

The data argument can be any serializable typea string, float, or int object.

To send a webhook to a Discord server, generate a webhook URL (for instructions, see Intro to Webhooks in the Discord Help Center) and serialize the data in a dictionary with a content key.

var discordURL = "https://discord.com/api/webhooks/.../...";
Notify.Web(discordURL, JsonSerializer.Serialize(new { content = "Hello world!" }));
discord_url = "https://discord.com/api/webhooks/.../..."
self.notify.web(discord_url, json.dumps({'content': 'Hello world!'}))
