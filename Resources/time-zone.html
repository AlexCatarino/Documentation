LEAN supports international trading across multiple time zones and markets, so it needs a reference time zone for the algorithm to set the Time. The default time zone is Eastern Time (ET), which is UTC-4 in summer and UTC-5 in winter. To set a different time zone, call the SetTimeZone method. This method accepts either a string following the IANA Time Zone database convention or a NodaTime.DateTimeZone object. If you pass a string, the method converts it to a NodaTime.DateTimeZone object. The TimeZones class provides helper methods to create NodaTime.DateTimeZone objects.

SetTimeZone("Europe/London");
SetTimeZone(NodaTime.DateTimeZone.Utc);
SetTimeZone(TimeZones.Chicago);

self.SetTimeZone("Europe/London")
self.SetTimeZone(NodaTime.DateTimeZone.Utc)
self.SetTimeZone(TimeZones.Chicago)
