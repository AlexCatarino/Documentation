Follow these steps to create your TT user name, account name, routing sender, session password, app key, and app secret:

Sign in

Sign in to your TT account or your TT UAT account. If you don't have an account, see Account Types.

Add New User

  1. On the setup page, click the +New User button. If you have a UAT account, use the UAT setup page.
  2. In the New User dialog, select the radio button that corresponds to user's employment status to your company.
  3. Fill the new user form. You must observe the following:
    1. Provide a username.
    2. In the Status section, select "TT Pro" from Trade Mode drop-down menu.
    3. In Advanced Settings section, select "Can create TT Rest API Key" and "Can create TT.NET SDK Client Side key".
  4. Click the Create button.
  5. Select the new user in the table to open their settings. In the Status section, click the Send Invitation button.

Create App Keys

  1. Select the App Keys tab and click the New button.
  2. In the Create New Application Key dialog, provide a application key name, select "TT REST API" in the Application Key Type drop-down menu, and "Usage Plan - Free" in the Application Key Usage Plan drop-down menu.
  3. Click the Copy Secret to Clipboard button, and paste it to a text editor.

    4. The text contains the App Key and App Secret separated by a colon. For example, a79b17df-b249-45a1-9d12-d5bbd2964626:424cc666-42aa-4125-a48e-13f630afd5a1.

  4. Close the dialog.

Create Account

  1. On the left-side panel, click on Accounts to open the Accounts page.
  2. On the Accounts page, click the +New Account button.
  3. Fill the new account form. You must observe the following:
    1. Provide an account name or number.
    2. Select the parent account from Parent drop-down menu.
    3. Select the "Routing (external clearing)" from Type drop-down menu.
  4. Click the Create button.
  5. To associate the account to the user, click on the Users tab, and then click the Add button.
  6. In the Select Users dialog, select the user, and click the Select button.

Create FIX Sessions

  1. On the left-side panel, click on More, and then click on FIX Sessions
  2. On the FIX Sessions page, click the +New FIX Session button.
  3. Fill the new FIX session form:
    1. Select the "FIX Order Routing" from FIX Type drop-down menu.
    2. Provide a Remote Comp ID name. It is the routing sender.
    3. Choose a session password.
    4. In the Status section, ensure "Inactive" is not selected.
  4. Click the Create button.
  5. To associate the FIX session to the user, click on the Users tab, and then click the Add button.
  6. In the Select Users dialog, select the user, and click the Select button.
  7. Click the Save Changes button.

Verify the User Credentials

  1. On the left-side panel, click on Users to open the Users page.
  2. Select the user from the users table.
  3. Click on the Account, FIX Sessions, and App Keys tabs to verify the user has access to the added features.
