The following table describes the supported DateRules:

DateRules Class Example Description
On(year,month,day) self.DateRules.On(2013, 10, 7) Trigger an event on a specific date.
EveryDay(symbol) self.DateRules.EveryDay("SPY") Trigger an event every day a specific symbol is trading.
EveryDay() self.DateRules.EveryDay() Trigger an event every day.
Every(days) self.DateRules.Every(DayOfWeek.Monday, DayOfWeek.Friday) Trigger an event on specific days throughout the week.
MonthStart(daysOffset) self.DateRules.MonthStart() Trigger an event on the first day of each month plus an offset (default is 0).
MonthStart(symbol, daysOffset) self.DateRules.MonthStart("SPY") Trigger an event on the first tradable date of each month for a specific symbol plus an offset (default is 0).
MonthEnd(daysOffset) self.DateRules.MonthEnd() Trigger an event on the last day of each month minus an offset (default is 0).
MonthEnd(symbol, daysOffset) self.DateRules.MonthEnd("SPY") Trigger an event on the last tradable date of each month for a specific symbol minus an offset (default is 0).
WeekStart(daysOffset) self.DateRules.WeekStart() Trigger an event on the first day of each week plus an offset (default is 0).
WeekStart(symbol, daysOffset) self.DateRules.WeekStart("SPY") Trigger an event on the first tradable date of each week for a specific symbol plus an offset (default is 0).
WeekEnd(daysOffset) self.DateRules.WeekEnd() Trigger an event on the last day of each week minus an offset (default is 0).
WeekEnd(symbol, daysOffset) self.DateRules.WeekEnd("SPY") Trigger an event on the last tradable date of each week for a specific symbol minus an offset (default is 0).
