The following table describes the WritePolicy enumerator members:

QuantConnect.WritePolicy - Used by the Data.LeanDataWriter to determine which merge write policy should be applied
Member Value Description
Overwrite 0 Will overwrite any existing file or zip entry with the new content
Merge 1 Will inject and merge new content with the existings file content
Append 2 Will append new data to the end of the file or zip entry
