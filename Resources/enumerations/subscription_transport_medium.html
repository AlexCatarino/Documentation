The following table describes the SubscriptionTransportMedium class selector methods:

QuantConnect.SubscriptionTransportMedium - Specifies where a subscription's data comes from
Enum Value Description
0 LocalFile The subscription's data comes from disk
1 RemoteFile The subscription's data is downloaded from a remote source
2 Rest The subscription's data comes from a rest call that is polled and returns a single line/data point of information
3 Streaming The subscription's data is streamed
