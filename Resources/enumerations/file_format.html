The following table describes the FileFormat class selector methods:

QuantConnect.Data.FileFormat - Specifies the format of data in a subscription
Enum Value Description
0 Csv Comma separated values
1 Binary Binary file data
2 ZipEntryName Only the zip entry names are read in as symbols
3 UnfoldingCollection Reader returns a BaseDataCollection object. Lean will unfold the collection and consume it as individual data points
4 Index Data stored using an intermediate index source
5 FoldingCollection Data type inherits from BaseDataCollection. Reader method can return a non BaseDataCollection type which will be folded, based on unique time, into an instance of the data type.
