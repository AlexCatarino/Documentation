The following table describes the MarketDataType class selector methods:

QuantConnect.MarketDataType - Market data style: is the market data a summary (OHLC style) bar, or is it a time-price value.
Enum Value Description
0 Base Base market data type
1 TradeBar TradeBar market data type (OHLC summary bar)
2 Tick Tick market data type (price-time pair)
3 Auxiliary Data associated with an instrument
4 QuoteBar QuoteBar market data type [Bid(OHLC), Ask(OHLC) and Mid(OHLC) summary bar]
5 OptionChain Option chain data
6 FuturesChain Futures chain data
