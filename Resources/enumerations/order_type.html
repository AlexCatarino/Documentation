The following table describes the OrderType class selector methods:

QuantConnect.Orders.OrderType - Type of the order: market, limit or stop
Enum Value Description
0 Market Market Order Type
1 Limit Limit Order Type
2 StopMarket Stop Market Order Type - Fill at market price when break target price
3 StopLimit Stop limit order type - trigger fill once pass the stop price; but limit fill to limit price.
4 MarketOnOpen Market on open type - executed on exchange open
5 MarketOnClose Market on close type - executed on exchange close
6 OptionExercise Option Exercise Order Type
7 LimitIfTouched Limit if Touched Order Type - a limit order to be placed after first reaching a trigger value.
