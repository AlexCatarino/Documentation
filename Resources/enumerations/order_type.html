The following table describes the OrderType enumerator members:

Member Value Description
Market 0 Market Order Type
Limit 1 Limit Order Type
StopMarket 2 Stop Market Order Type - Fill at market price when break target price
StopLimit 3 Stop limit order type - trigger fill once pass the stop price; but limit fill to limit price.
MarketOnOpen 4 Market on open type - executed on exchange open
MarketOnClose 5 Market on close type - executed on exchange close
OptionExercise 6 Option Exercise Order Type
LimitIfTouched 7 Limit if Touched Order Type - a limit order to be placed after first reaching a trigger value.
