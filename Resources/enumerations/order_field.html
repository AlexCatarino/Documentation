The following table describes the OrderField class selector methods:

QuantConnect.Orders.OrderField - Specifies an order field that does not apply to all order types
Enum Value Description
0 LimitPrice The limit price for a LimitOrder or StopLimitOrder
1 StopPrice The stop price for a StopMarketOrder or a StopLimitOrder
2 TriggerPrice The trigger price for a LimitIfTouchedOrder
