The following table describes the AlgorithmStatus class selector methods:

QuantConnect.AlgorithmStatus - States of a live deployment.
Enum Value Description
0 DeployError Error compiling algorithm at start
1 InQueue Waiting for a server
2 Running Running algorithm
3 Stopped Stopped algorithm or exited with runtime errors
4 Liquidated Liquidated algorithm
5 Deleted Algorithm has been deleted
6 Completed Algorithm completed running
7 RuntimeError Runtime Error Stoped Algorithm
8 Invalid Error in the algorithm id (not used).
9 LoggingIn The algorithm is logging into the brokerage
10 Initializing The algorithm is initializing
11 History History status update
