The following table describes the AlgorithmStatus enumerator members:

QuantConnect.AlgorithmStatus - States of a live deployment.
Member Value Description
DeployError 0 Error compiling algorithm at start
InQueue 1 Waiting for a server
Running 2 Running algorithm
Stopped 3 Stopped algorithm or exited with runtime errors
Liquidated 4 Liquidated algorithm
Deleted 5 Algorithm has been deleted
Completed 6 Algorithm completed running
RuntimeError 7 Runtime Error Stoped Algorithm
Invalid 8 Error in the algorithm id (not used).
LoggingIn 9 The algorithm is logging into the brokerage
Initializing 10 The algorithm is initializing
History 11 History status update
