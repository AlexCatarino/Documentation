There are limitations on the number of live trading nodes that your organization can have and the number of brokerage connections that you can have at one time.

Maximum Number of Nodes

You need a live trading node for each simultaneous algorithm that you deploy. The tier of your organization determines the number of live trading nodes to which your organization can subscribe. The following number of live trading nodes are available for each tier:

Tier Node Quota
Free 0
Quant Researcher 2
Team 10
Trading Firm Unlimited
Institution Unlimited

Brokerage Connections

You need a brokerage connection or account for each simultaneous algorithm that you deploy. If you want to deploy multiple algorithms using a single brokerage, create subaccounts in your brokerage account so that each algorithm has its own set of brokerage connection credentials. If you log in to your brokerage account while an algorithm is connected to it, the algorithm may stop executing.
