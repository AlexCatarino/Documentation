The VWAP Execution Model seeks for the average fill price of your position to match or better the volume weighted average price for the trading day. This is a best effort algorithm and no guarantee can be made that it will reach the VWAP.

To use the pre-made Execution Model in your algorithm you should set it in Initialize():

SetExecution( new VolumeWeightedAveragePriceExecutionModel() );
self.SetExecution( VolumeWeightedAveragePriceExecutionModel() )
