The NullPortfolioConstructionModel can be used to skip the execution phase of the algorithm i.e. "do nothing". This is useful when you're trying analyze the Alpha Model in isolation. All Alpha Streams algorithms can use Null Portfolio Construction and Null Execution Models. 

SetPortfolioConstructionModel( new NullPortfolioConstructionModel() ); 
self.SetPortfolioConstructionModel( NullPortfolioConstructionModel() )
