Alpha模型主要是预测市场趋势，并向算法发出信号，表明投资的最佳时刻。这些信号或$[Insight，T：QuantConnect.Algorithm.Alphas.Insight]对象包含市场预测的方向，幅度和置信度。这些Insights对象应该在资产集选择模型提供的已定义资产集上生成，并且仅在资产集更改时发出。

您可以使用 AddAlpha(alpha)self.AddAlpha(alpha)方法来设置Alpha模型，这一步应在算法初始化 Initialize()def Initialize()方法中实现。 

public override void Initialize()
{
      // 用最简单的Alpha模型（恒定向上信号）初始化框架
      AddAlpha( new ConstantAlphaModel(InsightType.Price, InsightDirection.Up, TimeSpan.FromMinutes(20), 0.025, null) );
}
