算法框架允许您添加多个Alphas类到算法中，并生成这些类的Insights，然后将组合的Insights流传递给投资组合构建模型。只需使用AddAlpha()方法添加每个Insight。下面是将两个Alpha模型组合在一个算法中的示例： 

// 定义一个结合rsi和ema cross的Alpha模型
AddAlpha( new RsiAlphaModel() );
AddAlpha( new EmaCrossAlphaModel() );
# 定义一个结合rsi和ema cross的Alpha模型
self.AddAlpha( RsiAlphaModel() )
self.AddAlpha( EmaCrossAlphaModel() )

您可以将任意多的Alpha模型添加到算法中。每个Alpha模型都有一个独有的的名称，生成的Insights会根据创建它的源Alpha模型自动命名。
