有时算法的表现依赖于多个Insight对象同时交易-例如配对交易和跨式期权交易。这些Insights应该组合在一起。Insight组向执行模型发出信号，表明需要将这些Insights合并为单个单元进行操作，以最大化Alpha。

当您从Alpha模型返回Insight组时，只需使用Insight.Group()辅助方法将Insights标记为一组。 

 // 将Insights组合在一起的辅助方法
return Insight.Group(insight1, insight2, insight3);
# 将Insights组合在一起的辅助方法
return Insight.Group( [ insight1, insight2, insight3 ] )
