Update方法返回一个包含Insight对象的数组。一个Insight对象是对某资产的单个预测。这些可以被认为是可操作的交易信号，表明了不久的将来资产价格的方向，幅度和置信度。Insight类具有以下重要属性： 

class Insight {
    // 证券代号
    Symbol Symbol;

    // 类型 （价格或波动率）
    InsightType Type;

    // 预测方向（下跌、持平、上涨）
    InsightDirection Direction;

    // 有效期
    TimeSpan Period;

    // 幅度（可选）
    double? Magnitude;

    // 置信度（可选）
    double? Confidence { get; private set; }
}
class Insight:
    self.Symbol # 证券代号
    self.Type # 类型 （价格或波动率）
    self.Direction # 预测方向（下跌、持平、上涨）
    self.Period # 有效期(TimeSpan)
    self.Magnitude # 幅度（可选）(double)
    self.Confidence # 置信度（可选）(double)}

$[Insight，T：QuantConnect.Algorithm.Framework.Alphas.Insight]构造函数采用以下参数：

# Insight对象构造函数参数：
# Insight(symbol, timedelta, type, direction, magnitude=None, confidence=None, sourceModel=None)
 Insight("IBM", timedelta(minutes=20), InsightType.Price, InsightDirection.Up, 0.0025, 1.00)
// Insight对象构造函数参数：
// new Insight(symbol, period, type, confidence=null, magnitude=null, source=null);
var insight = new Insight("IBM", TimeSpan.FromMinutes(20), InsightType.Price, InsightDirection.Up);

我们提供了一种辅助方法，可以更轻松地创建Insights。此辅助方法可以在Update方法中用于为价格类型的Alpha模型创建Insight对象：

insight = Insight.Price("IBM", timedelta(minutes = 20), InsightDirection.Up)
var insight = Insight.Price("IBM", TimeSpan.FromMinutes(20), InsightDirection.Up);
