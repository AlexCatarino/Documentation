Term Definition
Universe Selection Module Framework module for selecting assets for your algorithm.
Alpha Creation Module Framework component for that generates trade signals; on the assets in your universe for your algorithm.
Insight A trade signal generated by your algorithm Alpha Module. An Insight signals the Direction, Magnitude, Confidence and Period of your signal.
Portfolio Construction Module Framework component that constructs a target portfolio from Insight signals created by the Alpha Module.
Portfolio Target A target quantity of holdings (e.g. shares, contracts) the algorithm desires to reach. This target is created by the Portfolio Construction Module.
Execution Module Framework component responsible for fulfilling the Portfolio Targets set by the Portfolio Construction Module. Execute trades to reach the desired portfolio make up.
Risk Module Framework component that manages ongoing market and portfolio risk; ensuring the algorithm remains in target parameters.
