Term Definition
Universe Selection Module Framework module for selecting assets for your algorithm. e.g. Manual Universe Selection Module.
Alpha Creation Module Framework component for that generates trade signals; on the assets in your universe for your algorithm. e.g. RSI Alpha Model.
Insight A trade signal generated by your algorithm Alpha Module. An Insight provides the Direction, Magnitude, Confidence and Period of your signal. e.g. "Apple Will Move +0.12% within 3 Minutes".
Portfolio Construction Module Framework component that constructs a target portfolio (array of Portfolio Targets) from Insights created by the Alpha Module. e.g. Equal Weighting Portfolio Model.
Portfolio Target A target quantity of holdings (e.g. shares, contracts) the algorithm desires to reach. This target is created by the Portfolio Construction Module. 
Execution Module Framework component responsible for fulfilling the Portfolio Targets set by the Portfolio Construction Module. Execute trades to reach the desired portfolio make up. e.g. VWAP Execution Model.
Risk Module Framework component that manages ongoing market and portfolio risk; ensuring the algorithm remains in target parameters. e.g. Trailing Stop Risk Model.
