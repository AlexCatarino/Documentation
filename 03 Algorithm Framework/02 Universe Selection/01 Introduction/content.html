The Universe Section Model creates Universe objects which select the assets for your strategy. We have identified four types of universes that cover most people's requirements. We have built helper classes for these universes and will expand on how to use them in the discussion below. Generally, creating Universe Selection Models from scratch isn't simple so we recommend you use one of the helper universes we've provided.

To set a Universe Selection Model set your UniverseSelectionself.UniverseSelection property. This should be done from your algorithm Initialize()def Initialize() method: 

public override void Initialize()
{
      UniverseSettings.Resolution = Resolution.Minute;
      UniverseSelection = new ManualUniverseSelectionModel(QuantConnect.Symbol.Create("SPY", SecurityType.Equity, Market.USA));
}
def Initialize(self):    # Set requested data resolution
    self.UniverseSettings.Resolution = Resolution.Minute
    symbols = [ Symbol.Create("SPY", SecurityType.Equity, Market.USA) ]
    self.UniverseSelection = ManualUniverseSelectionModel(symbols)

As with all LEAN algorithms; the default resolution of the assets added to the universe is configured by the UniverseSettings.Resolutionself.UniverseSettings.Resolution property.
