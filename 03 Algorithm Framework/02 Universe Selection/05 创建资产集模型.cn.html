资产集模型必须实现一个$[IUniverseSelectionModel,T:QuantConnect.Algorithm.Selection.IUniverseSelectionModel]接口。它有一个方法，$[CreateUniverses(QCAlgorithm algorithm),M:QuantConnect.Algorithm.Selection.UniverseSelectionModel.CreateUniverses]。 算法对象被传递到方法中，以便您访问QuantConnect API, 它应返回一个Universe对象的数组。 

// 算法框架模型，用于定义算法使用的Universe（资产集）
interface IUniverseSelectionModel
{
   // 创建此算法的Universe，在IAlgorithm.Initialize之后调用一次
   IEnumerable<Universe> CreateUniverses(QCAlgorithmFramework algorithm);
}

一般来说，你应该能够扩展上述一个Universe（资产集），所以如果你发现自己需要做一些不符合上述类别的事情，请告诉我们，我们将创建一个新的基础类型的资产集模型。
