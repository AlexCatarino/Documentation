Requesting Data

To add US Energy Information Administration data to your algorithm use the AddData() system to request the data. As with all datasets, you should save a reference to your symbol for easy use later in your algorithm. For detailed documentation on using custom data see Importing Custom Data. 

// Weekly net input of total petroleum products
var petroleumInput = AddData<USEnergy>(USEnergy.Petroleum.UnitedStates.WeeklyNetImportsOfTotalPetroleumProducts).Symbol;

# Add Weekly net input of total petroleum products data.
petroleumInput = self.AddData(USEnergy, USEnergy.Petroleum.UnitedStates.WeeklyNetImportsOfTotalPetroleumProducts).Symbol

Accessing Data

Data can be accessed via the Slice events. Slice delivers the unique events to your algorithm as they happen. We recommend saving the symbol object when you add the data for easy access to slice later. Data is available in Weekly resolution. You can see an example of the slice accessor in the code below. 

using QuantConnect.Data.Custom.USEnergy;
namespace QuantConnect.Algorithm.CSharp {
    public class USEnergyAlgorithm : QCAlgorithm {
        public override void Initialize() {
            SetStartDate(2019, 1, 1);
            
            // Weekly net input of total petroleum products
            AddData<USEnergy>(USEnergy.Petroleum.UnitedStates.WeeklyNetImportsOfTotalPetroleumProducts);
        }
        
        public override void OnData(Slice data) {
            // Accessing via Slice event
            var petroleumInput = data.Get<USEnergy>();
            
            foreach (var petroleumInput in petroleumInput.Values)
            {
                Log($"Value: {petroleumInput.Value}");
            }
        }
    }
}
from QuantConnect.Data.Custom.USEnergy import *

    class USEnergyAlgorithm(QCAlgorithm):
        def Initialize(self):
            self.SetStartDate(2019, 1, 1)

            # Weekly net input of total petroleum products
            self.AddData(USEnergy, USEnergy.Petroleum.UnitedStates.WeeklyNetImportsOfTotalPetroleumProducts)

        def OnData(self, data):
            # Accessing via Slice event
            petroleumInput = data.Get(USEnergy)
            
            for petroleumInput in petroleumInput.Values:
                self.Log(f"Value: {petroleumInput.Value}")

All custom data has the properties Time, Symbol, and Value.
