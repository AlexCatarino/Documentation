You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets Weekly net input of total petroleum products historical data using the History API. 

// Request 60 days of Weekly net input of total petroleum products history with the petroleumInput Symbol object
var history = History<USEnergy>(petroleumInput, 60, Resolution.Daily);
# Request 60 days of Weekly net input of total petroleum products history with the petroleumInput Symbol object
history = self.History(USEnergy, petroleumInput, 60, Resolution.Daily)
