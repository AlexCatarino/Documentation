Requesting Data

To add Trading Economics data to your algorithm select the dataset you'd like to use from the tables below and use the AddData() system to request the data. As with all datasets, you should save a reference to your symbol for easy use later in your algorithm. 

current = AddData(TradingEconomics.Calendar.Australia.CurrentAccount);
self.current = self.AddData(TradingEconomicsCalendar, TradingEconomics.Calendar.Australia.CurrentAccount)
