You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets u.s. federal reserve interest rate historical data using the History API. 

var interestRate = AddData<TradingEconomicsCalendar>(TradingEconomics.Calendar.UnitedStates.InterestRate).Symbol;

// Request 60 days of u.s. federal reserve interest rate history with the interestRate Symbol
var interestRateHistory = History<TradingEconomicsCalendar>(interestRate, 60, Resolution.Daily);

interestRate = self.AddData(TradingEconomicsCalendar, TradingEconomics.Calendar.UnitedStates.InterestRate).Symbol

# Request 60 days of u.s. federal reserve interest rate history with the interestRate Symbol
interestRateHistory = self.History(TradingEconomicsCalendar, interestRate, 60, Resolution.Daily)
