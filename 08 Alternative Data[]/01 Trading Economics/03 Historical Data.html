You can request for historical custom data in your algorithm. To learn more about historical data requests, please visit the Historical Data documentation page. The following example gets Trading Economics Calendar historical data using the History API. 

// Add Trading Economics calendar data
var rates = AddData<TradingEconomicsCalendar>(TradingEconomics.Calendar.UnitedStates.InterestRate).Symbol;
// Request 60 days of interest rate history with the TradingEconomicsCalendar custom data Symbol.
var history = History<TradingEconomicsCalendar>(rates, 60, Resolution.Daily);

# Add Trading Economics calendar data

history = self.History(TradingEconomicsCalendar, rates, 60, Resolution.Daily)
