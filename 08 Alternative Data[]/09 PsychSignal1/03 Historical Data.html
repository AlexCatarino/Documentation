You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets AAPL sentiment historical data using the History API. 

// Request 60 days of AAPL sentiment history with the aaplSentiment Symbol object
var history = History<PsychSignalSentiment>(aaplSentiment, 60, Resolution.Daily);
# Request 60 days of AAPL sentiment history with the aaplSentiment Symbol object
