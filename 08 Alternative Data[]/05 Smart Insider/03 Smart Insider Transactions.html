Description of what has taken place. Possible values are:

  • Authorisation - The boards authority to repurchase
  • Intention - Notification to the board that shares will be repurchased
  • Transaction - Repurchase transactions that have been actioned

BuybackVia

string

Describes how transaction was executed. Possible values are:

  • Authorisation - Took place via the open market place
  • Intention Via a companywide tender offer to all shareholders
  • Transaction - Under a specific agreement between the issuer and shareholder

transaction.BuybackVia
BuybackBy

string

Describes which entity carried out the transaction. Possible values are:

  • Issuer - Issuer carried out the transaction
  • Subsidiary - A subsidiary of the issuer makes the transaction
  • Broker - Brokers are commonly used to repurchase shares under mandate to avoid insider information rules and to allow repurchases to carry on through close periods
  • Employee Benefit Trust - To cater for shares which will need to be transferred to employees as part of remunerative plans
  • Independent 3rd Party - An undisclosed 3rd party, althrough this is likely to be a broker

transaction.BuybackBy
HoldingType

string

Describes what will be done with those shares following repurchase. Possible values are:

  • In Treasury - Held in treasury until they are sold back to the market
  • For Cancellation - Immediately canceled
  • In Trust Held in trust, generally to cover employee remunerative plans

transaction.HoldingType
