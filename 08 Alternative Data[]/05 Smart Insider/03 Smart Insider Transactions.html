Smart Insider Transactions
EventType

SmartInsiderEventType

Description of what has taken place in an execution. Possible values are:

  • Authorization - The boards authority to repurchase
  • Intention - Notification to the board that shares will be repurchased
  • Transaction - Repurchase transactions that have been actioned
  • UpwardsRevision - Unknown
  • DownwardsRevision - Unknown
  • RevisedDetails - Unknown

Execution

SmartInsiderExecution

Describes how transaction was executed. Possible values are:

  • Market - Took place via the open market place
  • TenderOffer - Via a companywide tender offer to all shareholders
  • OffMarket - Under a specific agreement between the issuer and shareholder
transaction.Execution
ExecutionEntity

SmartInsiderExecutionEntity

Describes which entity carried out the transaction. Possible values are:

  • Issuer - Issuer carried out the transaction
  • Subsidiary - A subsidiary of the issuer makes the transaction
  • Broker - Brokers are commonly used to repurchase shares under mandate to avoid insider information rules and to allow repurchases to carry on through close periods
  • EmployerBenefitTrust - Unknown
  • EmployeeBenefitTrust - To cater for shares which will need to be transferred to employees as part of remunerative plans
  • ThirdParty - An undisclosed 3rd party, althrough this is likely to be a broker
transaction.ExecutionEntity
ExecutionHolding

SmartInsiderExecutionHolding

Describes what will be done with those shares following repurchase. Possible values are:

  • Treasury - Held in treasury until they are sold back to the market
  • Cancellation - Immediately canceled
  • Trust - Held in trust, generally to cover employee remunerative plans
  • SatisfyEmployeeTax - Unknown
  • NotReported - Unknown
  • Error - Unknown
