QuantConnect hosts Tiingo News data for US Equities. The data provides the article title, tickers mentioned and a long form description of the news story for each news item. There are approximately 8,000-12,000 articles per day recorded since January, 1st 2014.

Each $[TiingoNews, T:QuantConnect.Data.Custom.Tiingo.TiingoNews] object has five key properties you can use in your analysis: Title, Description, Tags, Source, and Symbols.

class TiingoNews {
      string Title;                       // Title of the article
      string Description;                 // Sentence summary of article
      List<string> Tags;                  // List of tags for article
      string Source;                      // Name of publication
      List<string> Symbols;               // List of mentioned companies.
}

class TiingoNews:
      self.Title;                       # Title of the article
      self.Description;                 # Sentence summary of article
      self.Tags;                        # List of tags for article
      self.Source;                      # Name of publication
      self.Symbols;                     # Symbol list of mentioned companies.
}
