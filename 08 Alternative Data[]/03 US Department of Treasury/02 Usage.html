U.S. Treasury yield curve data is not a linked data source. This means that there is no equity or Symbol that the data is related to, and can be added via a ticker. This data source is special in that you can use any ticker to add the data, as demonstrated below.

Requesting Data

Select Language: 
// Add U.S. Treasury yield curve data. We can use any ticker for this data source
AddData<USTreasuryYieldCurveRate>("FOOBAR");
# Add U.S. Treasury yield curve data. We can use any ticker for this data source
self.AddData(USTreasuryYieldCurveRate, "FOOBAR")

Accessing Data

Data can be accessed the normal way via the Slice events. Slice events deliver unique articles to your algorithm as they happen.
