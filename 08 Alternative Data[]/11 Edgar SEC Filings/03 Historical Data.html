You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets AAPL 8-K report historical data using the History API. 

// Request 60 days of AAPL 8-K report history with the report8k Symbol object
var history = History<SECReport8K>(report8k, 60, Resolution.Daily);
# Request 60 days of AAPL 8-K report history with the report8k Symbol object
history = self.History(SECReport8K, report8k, 60, Resolution.Daily)
