You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets U.S. Treasury yield curve rate historical data using the History API. 

// Request 60 days of U.S. Treasury yield curve rate history with the yieldCurve Symbol object
var history = History<USTreasuryYieldCurveRate>(yieldCurve, 60, Resolution.Daily);
# Request 60 days of U.S. Treasury yield curve rate history with the yieldCurve Symbol object
history = self.History(USTreasuryYieldCurveRate, yieldCurve, 60, Resolution.Daily)
