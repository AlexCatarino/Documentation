Requesting Data

To add U.S. Department of Treasury data to your algorithm use the AddData() system to request the data. As with all datasets, you should save a reference to your symbol for easy use later in your algorithm. For detailed documentation on using custom data see Importing Custom Data. 

// U.S. Treasury yield curve rate
var yieldCurve = AddData<USTreasuryYieldCurveRate>("USTYCR").Symbol;

# Add U.S. Treasury yield curve rate data.
yieldCurve = self.AddData(USTreasuryYieldCurveRate, "USTYCR").Symbol

Accessing Data

Data can be accessed via the Slice events. Slice delivers the unique events to your algorithm as they happen. We recommend saving the symbol object when you add the data for easy access to slice later. Data is available in the following resolutions: Daily. You can see an example of the slice accessor in the code below. 

using QuantConnect.Data.Custom.USTreasury;
namespace QuantConnect.Algorithm.CSharp {
    public class USTreasuryAlgorithm : QCAlgorithm {
        public override void Initialize() {
            SetStartDate(2019, 1, 1);
            
            // U.S. Treasury yield curve rate
            AddData<USTreasuryYieldCurveRate>("USTYCR");
        }
        
        public override void OnData(Slice data) {
            // Accessing via Slice event
            var yieldCurve = data.Get<USTreasuryYieldCurveRate>();
            
            foreach (var rates in yieldCurve.Values)
            {
                Log($"One month yield curve: {rates.OneMonth}, Two month yield curve: {rates.TwoMonth}, Three month yield curve: {rates.ThreeMonth}, Six month yield curve: {rates.SixMonth}, One year yield curve: {rates.OneYear}, Two year yield curve: {rates.TwoYear}, Three year yield curve: {rates.ThreeYear}, Five year yield curve: {rates.FiveYear}, Seven year yield curve: {rates.SevenYear}, Ten year yield curve: {rates.TenYear}, Twenty year yield curve: {rates.TwentyYear}, Thirty year yield curve: {rates.ThirtyYear}");
            }
        }
    }
}
