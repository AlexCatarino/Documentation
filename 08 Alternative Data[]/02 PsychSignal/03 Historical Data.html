You can request for historical custom data in your algorithm. To learn more about historical data requests, please visit the Historical Data documentation page. The following example gets PsychSignal historical data using the History API. 

// Request historical data
var ibm = AddEquity("IBM", Resolution.Minute).Symbol;
// Add news data for the underlying IBM asset
var psy = AddData<PsychSignalSentiment>(ibm).Symbol;
// Request history with the PsychSignal IBM Custom Data Symbol.
var history = History<PsychSignalSentiment>(psy, TimeSpan.FromDays(5));

# Request underlying equity data.
ibm = self.AddEquity("IBM", Resolution.Minute).Symbol
# Add sentiment data for the underlying IBM asset
psy = self.AddData(PsychSignalSentiment, ibm).Symbol
# Request history with the PsychSignal IBM Custom Data Symbol
history = self.History(PsychSignalSentiment, psy, timedelta(days=5))
