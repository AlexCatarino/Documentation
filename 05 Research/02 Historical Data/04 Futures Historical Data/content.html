To request options and future data in the research environment you need to add the underlying asset to the QuantBook as demonstrated above by the symbols. This returns a Security object. 

goog = qb.AddOption("GOOG") # add the GOOG options
es = qb.AddFuture("ES") # add S&P 500 futures

Once you have the Security object for the underlying asset you must set the filter you'd like to apply on the universe of available contracts. Options and Futures have a contracts for every strike-expiration date combination.

In Options this filter is set with the SetFilter(minStrike, maxStrike, minExpiry, maxExpiry) method. The strike is an integer number and the expiry is time span. If not specified, the default filter is SetFilter(-1, 1, timedelta(0), timedelta(35)).

For Futures you only need to set the contract expiration date range SetFilter(minExpiry, maxExpiry). The default filter is set to SetFilter(timedelta(0), timedelta(35)). 

# filter option contracts
goog.SetFilter(-2, 2, timedelta(0), timedelta(180))

# filter future contracts
es.SetFilter(timedelta(0), timedelta(180))
