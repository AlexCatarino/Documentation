You must add the assets you'd like to analyse to your QuantBook to request historical data for them. Adding assets uses the same API as QCAlgorithm. For more information on these methods see the Initializing Algorithms section. This is the case for all asset classes. 

qb = QuantBook()
spy = qb.AddEquity("SPY")   # add equity data
eur = qb.AddForex("EURUSD") # add forex data
btc = qb.AddCrypto("BTCUSD")# add crypto data
goog = qb.AddOption("GOOG") # add goog options
es  = qb.AddFuture("ES")    # add S&P 500 futures
