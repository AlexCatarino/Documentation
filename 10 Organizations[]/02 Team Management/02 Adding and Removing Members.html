You can add a new member by selecting the Add Member button. You can add an additional member using either their name or the email address associated with their account. Each member in your organization will have access to the resources available. However, you can also grant various permissions to your members. The organization manager has full permissions for managing resources, managing members and designating permissions to other members. There a few different categories of permissions you can allow members.

Node Permissions

  • Permission to create new nodes, specific to each node type
  • Permission to stop running nodes, specific to each node type
  • Permission to delete existing nodes, specific to each node type

Team Permissions

  • Permission to add new members
  • Permission to remove existing members
  • Permission to change the permissions of other members

Storage Permissions

  • Permission to write to the Object Store
  • Permission to delete data in the Object Store
  • Permission to subscibe to more space in the Object Store

Alpha Permissions

Token Permissions

  • Permission to create a new token
  • Permission to delete an existing token
  • Permission to read tokens

Exchange Permissions

  • Permission to read from the Alpha Streams exchange
