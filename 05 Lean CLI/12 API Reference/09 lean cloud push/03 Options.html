The lean cloud push command supports the following options:

Option Description
--project <path> Path to the local project to push (all local projects if not specified).
--organization-id <value> ID of the organization in which to create the project. If you omit this option, the project is created in your preferred organization. If you already created the project in the cloud, this option is ignored.
--verbose Enable debug logging.
--help Display the help text of the lean cloud push command and exit.
