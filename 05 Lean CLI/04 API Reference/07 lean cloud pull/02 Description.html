Pulls projects from QuantConnect to your local directory, while preserving the directory structure of your projects on QuantConnect. Both files as well as project parameters are pulled from the cloud. By default all cloud projects are pulled, unless a --project is specified, in which case a single project is pulled.

Before pulling a cloud project the CLI will check if the local directory with the same path already exists. If it does and the local directory is not linked to the cloud project (because of an earlier lean cloud pull or lean cloud push) the CLI will skip pulling the cloud project and log a descriptive warning message.

Local files which do not exist in the cloud will not be deleted by this command.
