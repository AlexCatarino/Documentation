The lean data download forex command supports the following options:

Option Description
--ticker <value> The ticker of the data.
--market <value> The market of the data. Must be either fxcm or oanda.
--resolution <value> The resolution of the data. Must be one of tick, second, minute, hour or daily.
--start <yyyyMMdd> The inclusive start date of the data in yyyyMMdd format (ignored for daily and hourly data).
--end <yyyyMMdd> The inclusive end date of the data in yyyyMMdd format (ignored for daily and hourly data).
--overwrite Overwrite existing local data.
--verbose Enable debug logging.
--help Display the help text of the lean data download forex command and exit.
