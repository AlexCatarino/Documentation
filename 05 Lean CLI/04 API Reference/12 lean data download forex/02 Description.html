Downloads Forex data from the QuantConnect Data Library. The configured options should match with the Forex data that you previously added to your QuantConnect account.

--start and --end are only required when downloading tick, second or minute data. In that case, these values are the inclusive boundaries of the data that is downloaded. The command will not download data for non-tradable dates, which are read from the market hours database in your data directory.

In case the local data already exists a warning is logged and the local data is not overwritten. Use the --overwrite flag to override this behavior.
