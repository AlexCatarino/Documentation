The lean research command supports the following options:

Option Description
--port The port to run Jupyter Lab on (defaults to 8888)
--update Pull the selected research environment version before starting it
--version The version of the research environment version to run (defaults to the latest installed version)
--lean-config <path> The Lean configuration file that should be used (defaults to the nearest lean.json file).
--verbose Enable debug logging.
--help Display the help text of the lean research command and exit.
