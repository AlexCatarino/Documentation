The lean backtest command supports the following options:

Option Description
--output Directory to store results in (defaults to PROJECT/backtests/TIMESTAMP)
--debug Enable a certain debugging method (see --help for more information)
--update Pull the selected LEAN engine version before running the backtest
--version The LEAN engine version to run (defaults to the latest installed version)
--lean-config <path> The Lean configuration file that should be used (defaults to the nearest lean.json file).
--verbose Enable debug logging.
--help Display the help text of the lean backtest command and exit.
