The lean optimize command supports the following options:

Option Description
--output Directory to store results in (defaults to PROJECT/optimizations/TIMESTAMP)
--optimizer-config The optimizer configuration file that should be used
--update Pull the selected LEAN engine version before running the optimizer
--version The LEAN engine version to run (defaults to the latest installed version)
--lean-config <path> The Lean configuration file that should be used (defaults to the nearest lean.json file).
--verbose Enable debug logging.
--help Display the help text of the lean optimize command and exit.
