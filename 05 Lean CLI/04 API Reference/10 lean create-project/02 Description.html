Creates a new project with some basic starter code. If the language is set to python this will generate a main.py file, a Python-based research notebook and a project configuration file. If the language is set to csharp this will generate a Main.cs file, a C#-based research notebook and a project configuration file.

If no --language is given, the default language saved in the global configuration is used. You can update the default language by running lean config set default-language <python/csharp>.

If the given project name contains slashes the name will be parsed as a path and the project will be created in a subdirectory. Any subdirectories that don't exist yet will be created automatically.
