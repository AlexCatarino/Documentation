The lean data download command supports the following options:

Option Description
--product <value> The type of product to download in non-interactive mode.
--organization <value> The name or id of the organization to purchase and download data with.
--data-type <value> The type of data to download, must be one of Trade, Quote, OpenInterest, and Margins.
--market <value> The market of the data to download.
--ticker <value> The ticker of the data to download.
--resolution <value> The resolution of the data to download, must be one of Tick, Second, Minute, Hour, and Daily.
--option-style <value> The option style of the data to download, must be one of American and European.
--start <yyyyMMdd> The inclusive start date of the data to download (only for tick, second, and minute resolutions).
--end <yyyyMMdd> The inclusive end date of the data to download (only for tick, second, and minute resolutions).
--report-type <value> The type of SEC reports to download, must be one of 10K, 10Q, and 8K.
--overwrite Overwrite existing local data.
--verbose Enable debug logging.
--help Display the help text of the lean data download command and exit.
