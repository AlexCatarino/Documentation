If you trade India Equities, you need the India Equity Security Master and some data from the India Equities dataset. The India Equities dataset is available is several resolutions. The resolution you need depends on the India Equity subscriptions you create in your algorithm. The following table describes the file format of the available resolutions in the India Equities dataset:

Resolution File Format
Minute One file per security per day
Hour One file per security
Daily One file per security
