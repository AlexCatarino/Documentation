If you trade US Equities, you need the US Equity Security Master and some data from the US Equities dataset. The US Equities dataset is available is several resolutions. The resolution you need depends on the US Equity subscriptions you create in your algorithm. The following table describes the file format of the available resolutions in the US Equities dataset:

Resolution File Format
Tick One file per security per day
Second One file per security per day
Minute One file per security per day
Hour One file per security
Daily One file per security

If you add universes to your algorithm, the following table shows the additional datasets you need:

Universe Type Required Dataset
Coarse US Coarse Universe
Dollar Volume US Coarse Universe
ETF Constituents US ETF Constituents
