Follow these steps to download the US Equities dataset history in bulk:

  1. Log in to the Algorithm Lab.
  2. Connect to the organization you want to use to download the dataset.
  3. If you want the download to include hourly and/or daily data, on the Pricing page, subscribe to the US Equity Hourly & Daily History by AlgoSeek dataset.
  4. If you want the download to include minute, second, and/or tick data, contact us and request a subscription to the US Equity Minute, Second, Tick History by AlgoSeek dataset.
  5. On the CLI tab of the dataset listing, use the CLI Command Generator to generate your download command.

    The Ticker, Start Date, and End Date fields are irrelevant for bulk downloads.

  6. Copy the command from the CLI Command Generator.
  7. Open a terminal in your organization workspace and then run the command from the CLI Command Generator.
