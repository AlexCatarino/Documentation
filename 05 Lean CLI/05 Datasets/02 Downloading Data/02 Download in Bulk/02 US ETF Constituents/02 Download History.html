Follow these steps to download the US ETF Constituents dataset history in bulk:

  • Log in to the Algorithm Lab.
  • Connect to the organization you want to use to download the dataset.
  • On the Pricing page, subscribe to the US ETF Constituents History by QuantConnect dataset.
  • On the CLI tab of the dataset listing, use the CLI Command Generator to generate your download command.

    The Ticker, Start Date, and End Date fields are irrelevant for bulk downloads.

  • Copy the command from the CLI Command Generator.
  • Open a terminal in your organization workspace and then run the command from the CLI Command Generator.
