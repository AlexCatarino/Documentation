After you bulk download the US Coarse Universe dataset, new daily updates are available at 7 AM Eastern Time (ET) after each trading day. To gain access to the data updates, contact us. You need billing permissions to change the organization's subscriptions.

After you subscribe to datasets updates, to update your local copy of the US Coarse Universe dataset, open a terminal in your organization workspace and run the following command:

$ lean data download --dataset "US Coarse Universe" --data-type "Bulk"

Alternatively, instead of directly calling the lean data download command, you can place a Python script in the data directory of your workspace and run it to update your data files. The following example script downloads the latest data when it's available:

import os
from datetime import datetime
from pytz import timezone

END_DATE = datetime.now(timezone("US/Eastern")).strftime("%Y%m%d")
latest_date = sorted([f for f in os.listdir(f"equity/usa/fundamental/coarse")])[-1]
if latest_date >= END_DATE:
    print(f"Your local data is already up to date.")
else:
    print("Updating data...")
    os.system(f'lean data download --dataset "US Coarse Universe" --data-type "Bulk"')

We structure universe data files so there is one file per day. To update your local dataset, the preceding script checks the date of the most recent US Coarse Universe data you have. If there is new data available, it downloads the new data files.
