To deploy a local live algorithm that uses IEX Cloud as the data provider, open a terminal in your organization workspace and then run lean live deploy <projectName> --data-provider IEX --iex-cloud-api-key <apiKey> --iex-price-plan <pricePlan> --brokerage <brokerageName> <requiredBrokerageOptions>.

$ lean live deploy "My Project" --data-provider IEX --iex-cloud-api-key apiKey --iex-price-plan Grow --brokerage "Paper Trading"

The --iex-price-plan option must be Launch, Grow, or Enterprise. Depending on the brokerage you select, you may need to provide some required brokerage options. If you provide any of the preceding options, your Lean configuration file saves them so that you only need to run lean live deploy <projectName> --brokerage <brokerageName> to deploy another live algorithm with the same options.
