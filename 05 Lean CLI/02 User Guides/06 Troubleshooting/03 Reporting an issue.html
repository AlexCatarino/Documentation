If with the information on this page and the error message shown by the CLI you're still unable to solve your issues you are welcome to reach out to our engineers by opening an issue in the QuantConnect/lean-cli repository on GitHub. Before doing so, please run the command that's giving issues with the --verbose flag and copy and paste the output into the issue (feel free to mask sensitive information). The --verbose flag enables debug messages to be printed, which makes it easier for us to help solve your issues.
