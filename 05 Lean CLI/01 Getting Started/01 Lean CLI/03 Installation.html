Run pip install lean in a terminal to install the latest version of the CLI.

After installing the CLI, open a terminal in an empty directory and run lean init. This command scaffolds a standard directory structure containing all the files you need to get going. We recommend to run all Lean CLI commands in the same directory lean init was ran in. 

lean init
