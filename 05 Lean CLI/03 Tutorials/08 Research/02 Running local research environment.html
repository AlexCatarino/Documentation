Run lean research "Project Name" to launch a local Jupyter Lab environment for the ./Project Name directory. This will automatically run the latest downloaded version of the quantconnect/research Docker image and opens the browser once the environment is available.

You can use the --version option if you want to run a specific version of the quantconnect/research image. The value of the option should be the tag that should be used, so lean research "Project Name" --version 10992 will run the 10992 tag.

By default Jupyter Lab is exposed on port 8888 locally. You can specify a custom port with the --port option, which may be useful if you want to run two Jupyter Lab environments alongside each other.
