Run lean research "Project Name" to launch a local Jupyter Lab environment for the ./Project Name directory. This command automatically runs the quantconnect/research Docker image and opens the browser once the environment is available.

You can use the --version option to run a specific version of the quantconnect/research image. The value of the option is the tag that is used, so lean research "Project Name" --version 10992 runs the 10992 tag.

By default Jupyter Lab is exposed on port 8888 locally. You can specify a custom port with the --port option, which is required to run two Jupyter Lab environments alongside each other.
