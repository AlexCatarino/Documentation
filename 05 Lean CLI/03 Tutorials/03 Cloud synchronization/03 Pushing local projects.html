Run lean cloud push to push all your local projects in the current directory to the cloud. This command pushes all locally modified files to the cloud and creates cloud files and projects when necessary. You can use the --project "Project Name" option to push a single project, where Project Name must be the path to the local project that should be pushed.

Before uploading any files to the cloud the CLI performs some checks to make sure it doesn't perform unwanted destructive actions. If the CLI determines a problem with uploading a project's files, it logs a warning and skips the project.
