Run lean cloud push to push all your local projects in the current directory to the cloud. This will push all locally modified files to the cloud and will create cloud files and projects when necessary. You can use the --project "Project Name" option if you want to push single project, where Project Name should be the path to the local project that should be pushed.

Before uploading any files to the cloud the CLI performs some checks to make sure it doesn't perform unwanted destructive actions. If the CLI determines a problem with uploading a project's files, it will log a warning and skip the project.
