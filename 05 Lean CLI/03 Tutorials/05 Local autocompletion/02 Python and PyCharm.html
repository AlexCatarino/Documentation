Follow these steps to set up local autocompletion for Python in PyCharm:

  1. Open the directory you ran lean init in with PyCharm and wait for the project to load.
  2. In the bottom-right of the editor there should be a button saying "Python 3.x", click on it and select "Add Interpreter".
  3. Configure the Python interpreter to the one you used to run pip install lean.
  4. Save the Python interpreter configuration, wait for PyCharm to index all packages and autocompletion starts working.
