Follow these steps to set up local autocompletion for Python in Rider:

  1. Make sure the .NET Framework 4.6.2 Developer Pack or a recent version of Mono is installed.
  2. Open the LeanCLI.csproj file in the directory you ran lean init in with Rider and wait for the project to load. Rider will automatically download the required dependencies and index them to provide autocompletion.
