Follow these steps to set up local autocompletion for Python in VS Code:

  1. Make sure the .NET Framework 4.6.2 Developer Pack or a recent version of Mono is installed.
  2. Install the C# extension in VS Code.
  3. Open the directory you ran lean init in with VS Code and wait for the project to load.
  4. Click on the "Restore" button if a pop-up shows in the bottom-right telling you there are dependencies to restore.
