Run lean create-project --language python "Project Name" to create a new Python project named "Project Name". This will create the ./Project Name directory and create a simple main.py file, a Python-based research notebook and a project configuration file. If you want to create a C# project you can replace --language python with --language csharp, which will create a simple Main.cs file and a C#-based research notebook instead. The specified project name may contain slashes, in which case the CLI will automatically create any non-existent subdirectories.

You can provide a project name containing forward slashes if you want to create a project in a subdirectory. In case any of the given subdirectories does not exist yet the CLI will create them for you.
