When running optimizations you need to have parameters to optimize. Project parameters are stored in the project configuration stored in the config.json file in the project directory. To update a project's parameters, open this file, update the key/value object of the parameters property and save the file. Make sure all the keys and the values of the parameters property are strings, other data types are not supported.

To retrieve a parameter in your algorithm you can use the GetParameter(string name) method on QCAlgorithm, which returns the value of a parameter as a string. See ParametrizedAlgorithm.py and ParametrizedAlgorithm.cs for examples which use this method.
