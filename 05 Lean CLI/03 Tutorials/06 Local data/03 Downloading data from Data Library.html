After adding data to your QuantConnect account you can download it using the CLI. There is a different command for each security type.

Downloading Forex data

Run lean data download forex --ticker <ticker> --market <market> --resolution <resolution> to download Forex data. The ticker, market and resolution values should match with the data you added to your QuantConnect account. If you are downloading minute, second or tick data you also need to specify a --start and a --end option specifying the inclusive start and end dates of the data in yyyyMMdd format.

Downloading CFD data

Run lean data download forex --ticker <ticker> --resolution <resolution> to download CFD data. The ticker and resolution values should match with the data you added to your QuantConnect account. If you are downloading minute, second or tick data you also need to specify a --start and a --end option specifying the inclusive start and end dates of the data in yyyyMMdd format.
