After adding data to your QuantConnect account you can download it using the CLI. There is a different command for each security type.

Follow these steps to download Forex data:

  1. Ensure you are logged in using lean login.
  2. Open a terminal in the directory you ran lean init in.
  3. Run lean data download forex --ticker <ticker> --market <market> --resolution <resolution> --start <yyyyMMdd> --end <yyyyMMdd> to download Forex data. 
    $ lean data download forex --ticker eurusd --market oanda --resolution minute --start 20020505 --end 20210322
[1/5903] Downloading forex/oanda/minute/eurusd/20020505_quote.zip
[2/5903] Downloading forex/oanda/minute/eurusd/20020506_quote.zip
[3/5903] Downloading forex/oanda/minute/eurusd/20020507_quote.zip
    The option values must match with the data you added to your QuantConnect account. If you are downloading hourly or daily data you can omit the --start and the --end options.

Follow these steps to download CFD data:

  1. Ensure you are logged in using lean login.
  2. Open a terminal in the directory you ran lean init in.
  3. Run lean data download cfd --ticker <ticker> --resolution <resolution> --start <yyyyMMdd> --end <yyyyMMdd> to download CFD data. 
    $ lean data download cfd --ticker au200aud --resolution minute --start 20040317 --end 20210322
