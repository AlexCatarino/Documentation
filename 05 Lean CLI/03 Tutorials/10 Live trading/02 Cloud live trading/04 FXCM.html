Follow these steps after selecting FXCM in step 4 of the tutorial in the introduction:

  1. Ask the FXCM staff to enable API access for your account. You can do so by sending the following email to pcs@fxcm.com titled "Please enable Java API for my account": 
    Hello FXCM staff,

Please enable Java API for all accounts which are associated with this email address.

Also, please respond to this email address once Java API has been enabled, letting me know that the change was done successfully.

Thank you very much in advance
  2. Enter your username and account password. 
    $ lean cloud live "My Project" --push --open
Username: john
Account password: ****************
  3. Enter which environment you want to use. You can either choose demo or real. 
    $ lean cloud live "My Project" --push --open
Environment (demo, real): demo
  4. Continue with step 6 of the tutorial in the introduction.
