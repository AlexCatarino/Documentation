Make sure the following environment exists in your lean.json file before you start local live trading on Bitfinex: 

{
    "environments": {
        "live-bitfinex": {
            "live-mode": true,

            "live-mode-brokerage": "BitfinexBrokerage",
            "data-queue-handler": "BitfinexBrokerage",
            "setup-handler": "QuantConnect.Lean.Engine.Setup.BrokerageSetupHandler",
            "result-handler": "QuantConnect.Lean.Engine.Results.LiveTradingResultHandler",
            "data-feed-handler": "QuantConnect.Lean.Engine.DataFeeds.LiveTradingDataFeed",
            "real-time-handler": "QuantConnect.Lean.Engine.RealTime.LiveTradingRealTimeHandler",
            "transaction-handler": "QuantConnect.Lean.Engine.TransactionHandlers.BrokerageTransactionHandler",
            "history-provider": "BrokerageHistoryProvider"
        }
    }
}

Make sure the following properties are configured above the environments object in your lean.json file before you start local live trading on Bitfinex:

  • bitfinex-api-secret: This property must contain your API secret. You can create a new API key on bitfinex.com/api#createkey.
  • bitfinex-api-key: This property must contain the API key belonging to the given secret.

After setting up the required configuration you can continue with step 4 of the tutorial in the introduction and use live-bitfinex as environment name in the steps that follow.
