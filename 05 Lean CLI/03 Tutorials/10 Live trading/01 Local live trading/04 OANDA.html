To start local live trading on OANDA, make sure the following environment exists: 

{
    "environments": {
        "live-oanda": {
            "live-mode": true,

            "live-mode-brokerage": "OandaBrokerage",
            "data-queue-handler": "OandaBrokerage",
            "setup-handler": "QuantConnect.Lean.Engine.Setup.BrokerageSetupHandler",
            "result-handler": "QuantConnect.Lean.Engine.Results.LiveTradingResultHandler",
            "data-feed-handler": "QuantConnect.Lean.Engine.DataFeeds.LiveTradingDataFeed",
            "real-time-handler": "QuantConnect.Lean.Engine.RealTime.LiveTradingRealTimeHandler",
            "transaction-handler": "QuantConnect.Lean.Engine.TransactionHandlers.BrokerageTransactionHandler",
            "history-provider": "BrokerageHistoryProvider"
        }
    }
}

The following properties should be configured before you start local live trading on OANDA:

  • oanda-environment: This property must be set to Trade to use the OANDA's fxTrade environment, or to Practice to use OANDA's fxTrade Practice environment.
  • oanda-access-token: This property must contain your API access token. You can generate one on the My Account > My Services > Manage API Access page.
  • oanda-account-id: This property must contain your account id.

After setting up all the above, you can start local live trading on OANDA by running lean live <project name> live-oanda.
