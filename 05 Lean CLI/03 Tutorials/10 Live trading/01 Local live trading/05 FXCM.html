To start local live trading on FXCM, make sure the following environment exists: 

{
    "environments": {
        "live-fxcm": {
            "live-mode": true,

            "live-mode-brokerage": "FxcmBrokerage",
            "data-queue-handler": "FxcmBrokerage",
            "setup-handler": "QuantConnect.Lean.Engine.Setup.BrokerageSetupHandler",
            "result-handler": "QuantConnect.Lean.Engine.Results.LiveTradingResultHandler",
            "data-feed-handler": "QuantConnect.Lean.Engine.DataFeeds.LiveTradingDataFeed",
            "real-time-handler": "QuantConnect.Lean.Engine.RealTime.LiveTradingRealTimeHandler",
            "transaction-handler": "QuantConnect.Lean.Engine.TransactionHandlers.BrokerageTransactionHandler",
            "history-provider": "BrokerageHistoryProvider"
        }
    }
}

The following properties must be configured before you start local live trading on FXCM:

  • fxcm-server: This property must contain the full URL of the server (example: http://www.fxcorporate.com/Hosts.jsp).
  • fxcm-terminal: This property must contain the name of the connection, either Demo or Real.
  • fxcm-user-name: This property must contain your username.
  • fxcm-password: This property must contain your password.
  • fxcm-account-id: This property must contain your account id.

After setting up all the above, you can start local live trading on FXCM by running lean live <project name> live-fxcm.
