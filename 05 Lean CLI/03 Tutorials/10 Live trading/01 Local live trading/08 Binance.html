To start local live trading on Binance, make sure the following environment exists: 

{
    "environments": {
        "live-binance": {
            "live-mode": true,

            "live-mode-brokerage": "BinanceBrokerage",
            "data-queue-handler": "BinanceBrokerage",
            "setup-handler": "QuantConnect.Lean.Engine.Setup.BrokerageSetupHandler",
            "result-handler": "QuantConnect.Lean.Engine.Results.LiveTradingResultHandler",
            "data-feed-handler": "QuantConnect.Lean.Engine.DataFeeds.LiveTradingDataFeed",
            "real-time-handler": "QuantConnect.Lean.Engine.RealTime.LiveTradingRealTimeHandler",
            "transaction-handler": "QuantConnect.Lean.Engine.TransactionHandlers.BrokerageTransactionHandler",
            "history-provider": "BrokerageHistoryProvider"
        }
    }
}

The following properties should be configured before you start local live trading on Binance:

  • binance-api-secret: This property must contain your API secret. You can create a new API key on binance.com/en/my/settings/api-management.
  • binance-api-key: This property must contain the API key belonging to the given secret.

After setting up all the above, you can start local live trading on Binance by running lean live <project name> live-binance.
