To start local live trading on Tradier, make sure the following environment exists: 

{
    "environments": {
        "live-tradier": {
            "live-mode": true,

            "live-mode-brokerage": "TradierBrokerage",
            "data-queue-handler": "TradierBrokerage",

            "setup-handler": "QuantConnect.Lean.Engine.Setup.BrokerageSetupHandler",
            "result-handler": "QuantConnect.Lean.Engine.Results.LiveTradingResultHandler",
            "data-feed-handler": "QuantConnect.Lean.Engine.DataFeeds.LiveTradingDataFeed",
            "real-time-handler": "QuantConnect.Lean.Engine.RealTime.LiveTradingRealTimeHandler",
            "transaction-handler": "QuantConnect.Lean.Engine.TransactionHandlers.BrokerageTransactionHandler"
        }
    }
}

The following properties should be configured before you start local live trading on Tradier:

  • tradier-use-sandbox: This property must be true if Tradier's developer sandbox should be used, or false if it shouldn't.
  • tradier-account-id: This property must contain your account id.
  • tradier-access-token: This property must contain your API access token.

After setting up all the above, you can start local live trading on Tradier by running lean live <project name> live-tradier.
