Instead of live trading using actual money, you can also start paper trading locally. Although some brokerages have built-in support for paper trading, LEAN also offers a paper brokerage. This brokerage allows you to run live, real-time data through your algorithm while using fictional capital. To start local paper trading, make sure the following environment exists: 

{
    "environments": {
        "live-paper": {
            "live-mode": true,

            "live-mode-brokerage": "PaperBrokerage",

            "setup-handler": "QuantConnect.Lean.Engine.Setup.BrokerageSetupHandler",
            "result-handler": "QuantConnect.Lean.Engine.Results.LiveTradingResultHandler",
            "data-feed-handler": "QuantConnect.Lean.Engine.DataFeeds.LiveTradingDataFeed",
            "data-queue-handler": "<data queue handler>",
            "real-time-handler": "QuantConnect.Lean.Engine.RealTime.LiveTradingRealTimeHandler",
            "transaction-handler": "QuantConnect.Lean.Engine.TransactionHandlers.BacktestingTransactionHandler"
        }
    }
}

The <data queue handler> must have the same value as the data-queue-handler of one of the other brokerages. You must also configure all the properties of the brokerage from which you copy the data queue handler. For example, you can set data-queue-handler to BinanceBrokerage and configure Binance's required properties to use the live data feed from Binance.

After setting up the environment, you can start local paper trading by running lean live <project name> live-paper.
