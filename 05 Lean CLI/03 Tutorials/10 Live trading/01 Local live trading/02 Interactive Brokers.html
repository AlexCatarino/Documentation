Make sure the following environment exists in your lean.json file before you start local live trading on Interactive Brokers: 

{
    "environments": {
        "live-interactive": {
            "live-mode": true,

            "live-mode-brokerage": "InteractiveBrokersBrokerage",
            "setup-handler": "QuantConnect.Lean.Engine.Setup.BrokerageSetupHandler",
            "result-handler": "QuantConnect.Lean.Engine.Results.LiveTradingResultHandler",
            "data-feed-handler": "QuantConnect.Lean.Engine.DataFeeds.LiveTradingDataFeed",
            "data-queue-handler": "QuantConnect.Brokerages.InteractiveBrokers.InteractiveBrokersBrokerage",
            "real-time-handler": "QuantConnect.Lean.Engine.RealTime.LiveTradingRealTimeHandler",
            "transaction-handler": "QuantConnect.Lean.Engine.TransactionHandlers.BrokerageTransactionHandler",
            "history-provider": "BrokerageHistoryProvider"
        }
    }
}

Make sure the following properties are configured above the environments object in your lean.json file before you start local live trading on Interactive Brokers:

  • ib-account: This property must contain the name of the account (example: DU1234567).
  • ib-user-name: This property must contain the username of the given account (example: trader123).
  • ib-password: This property must contain the password of the given account.
  • ib-agent-description: This property describes the type of trader that you are. It must be Individual or I if you are an individual, A if you are an agency, or W if you are an agent or other member.
  • ib-trading-mode: This property must be paper if the specified account is a paper account, or live if the specified account is a live account.
  • ib-enable-delayed-streaming-data: This property configures the behavior when your algorithm attempts to subscribe to market data for which you don't have a market data subscription on Interactive Brokers. When set to true, your algorithm continues running using delayed market data. When set to false, live trading will stop and LEAN will shut down.

After setting up the required configuration you can continue with step 4 of the tutorial in the introduction and use live-interactive as environment name in the steps that follow.
