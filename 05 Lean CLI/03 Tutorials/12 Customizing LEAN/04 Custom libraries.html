Follow these steps to add custom C# libraries to your own version of LEAN:

  1. Follow step 1 of Building custom images.
  2. Open Algorithm.CSharp/QuantConnect.Algorithm.CSharp.csproj in an editor.
  3. In the ItemGroup containing the PackageReference tags, add PackageReference tags for the libraries you want to add. For example, if you want to add XPlot.Plotly 4.0.2 you would add the following PackageReference: 
    <ItemGroup>
    <PackageReference Include="XPlot.Plotly" Version="4.0.2" />
</ItemGroup>
  4. Save your changes to Algorithm.CSharp/QuantConnect.Algorithm.CSharp.csproj.
  5. Continue with step 3 of Building custom images.

Follow these steps to add custom Python libraries to your own version of LEAN:

  1. Follow step 1 of Building custom images.
  2. Open Dockerfile in an editor.
  3. Add the necessary statements to the Dockerfile to install your libraries. In most cases this means adding a RUN pip install <library> statement in between the FROM and the COPY statements.
  4. Save your changes to Dockerfile.
  5. Continue with step 3 of Building custom images.
