Follow these steps to start local debugging for C# in VS Code:

  1. Follow the How to set up local autocomplete for C# in VS Code tutorial.
  2. Install version 15.8 of the Mono Debug extension. You can do this by first installing the latest version and then clicking on the arrow button next to the Uninstall button, which opens a context menu containing the "Install Another Version" option.
  3. If you are using macOS or Linux, open VS Code's settings and set "Omnisharp: Use Global Mono" to "always". Restart the editor after saving the changes.
  4. Run the lean backtest command with the --debug mono option and wait until the CLI tells you to attach to the debugger. 
    $ lean backtest "My Project" --debug mono
Docker container starting, attach to Mono debugger at localhost:55556 to begin
  5. In VS Code, open the Run tab and run the configuration called "Debug with Lean CLI" (this configuration is created when you create a new project with the CLI).
