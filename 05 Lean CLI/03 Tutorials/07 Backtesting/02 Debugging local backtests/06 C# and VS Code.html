Follow these steps to start local debugging for C# in VS Code:

  1. Install version 15.8 of the Mono Debug extension. You can do this by first installing the latest version and then clicking on the arrow button next to the Uninstall button, which opens a context menu containing the "Install Another Version" option.
  2. Run the lean backtest command with the --debug mono option and wait until the CLI tells you to attach to the debugger. 
    $ lean backtest "My Project" --debug mono
Docker container starting, attach to Mono debugger at localhost:55556 to begin
  3. In VS Code, open the Run tab and run the configuration called "Debug with Lean CLI" (this configuration is created when you create a new project with the CLI).
