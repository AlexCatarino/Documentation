Follow these steps to start local debugging for C# in Rider:

  1. Run the lean backtest command with the --debug mono option and wait until the CLI tells you to attach to the debugger. 
    $ lean backtest "My Project" --debug mono
Docker container starting, attach to Mono debugger at localhost:55555 to begin
  2. In Rider, start debugging using the "Debug with Lean CLI" run configuration (this configuration is created when you create a new project with the CLI).
