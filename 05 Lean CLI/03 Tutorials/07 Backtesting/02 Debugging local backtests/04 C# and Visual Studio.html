Follow these steps to start local debugging for C# in Visual Studio:

  1. Follow the How to set up local autocomplete for C# in Visual Studio tutorial.
  2. Install the VSMonoDebugger extension.
  3. In Visual Studio, go to "Extensions > Mono > Settings" and enter the following settings:
    • Remote Host IP: 127.0.0.1
    • Remote Host Port: 55556
    • Mono Debug Port: 55556
  4. Run the lean backtest command with the --debug mono option and wait until the CLI tells you to attach to the debugger. 
    $ lean backtest "My Project" --debug mono
Docker container starting, attach to Mono debugger at localhost:55556 to begin
  5. In Visual Studio, attach to the debugger using "Extensions > Mono > Attach to mono debugger".
