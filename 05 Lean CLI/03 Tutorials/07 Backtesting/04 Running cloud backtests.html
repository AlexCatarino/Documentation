Run lean cloud backtest "Project Name" to run a cloud backtest for a certain cloud project. This will start a cloud backtest, let you know about its progress and show you the summarized statistics and a link to the full results once the backtest is finished. You can use the --open flag if you want to automatically open the full results in the browser after the backtest has finished.

If you have previously pulled the cloud project to your local drive using lean cloud pull you can also use the --push flag to automatically push modified files to the cloud before running the backtest. This prevents you from having to run manually lean cloud push every time you want to test (small) changes using a cloud backtest.
