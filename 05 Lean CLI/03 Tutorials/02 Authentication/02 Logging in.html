1. Navigate to your QuantConnect Account page and click "Request Email With Token and Your User-Id for API Requests" under the Security section. After clicking on this link your user-id and API token is sent to your email address.
  2. Retrieve your user id and API token from the email you registered on QuantConnect.
  3. Type lean login to login to the CLI, and enter your user-id and token when prompted. This command opens an interactive wizard asking you for your user id and API token.
