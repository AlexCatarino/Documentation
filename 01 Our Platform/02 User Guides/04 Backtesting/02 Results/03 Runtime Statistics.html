Runtime statistics of your backtest performance are displayed in the banner at the top of the backtest results page. You can create your own custom runtime statistics, but the following statistics are included by default:

  • PSR: Probabilistic Sharpe Ratio
  • Unrealized: Unrealized profit
  • Fees: Fees incurred by filling orders
  • Net Profit: Net profit = current equity - starting equity - unrealized profit + |fees|
  • Return: Return = (current equity - starting equity) / starting equity
  • Equity: Value of cashbook and portfolio holdings
  • Holdings: Value of portfolio holdings
  • Volume: Dollar-value of all trades
