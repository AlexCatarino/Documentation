运行时统计信息是在实时算法顶部显示的可自定义的横幅统计信息。

您可以使用SetRuntimeStatistic(string name, string value);方法来添加自己的运行时统计信息。在上图中，用户显示BTC的当前价格：SetRuntimeStatistic("BTC", data["BTC"].Close); 

// 用户显示BTC的当前价格
SetRuntimeStatistic("BTC", data["BTC"].Close);

# 用户显示BTC的当前价格
btc = self.Securities["BTC"].Symbol
self.SetRuntimeStatistic("BTC", data[btc].Close)
