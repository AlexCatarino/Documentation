The optimization results page displays a Server Statistics section to show the status of the nodes running the optimization job.

The Server Statistics section displays the following information:

  • CPU: CPU usage of each node
  • RAM: RAM usage of each node
  • HOST: Node type and quantity
  • Uptime: The length of time that the optimization job has been running

View the Server Statistics section to see if your algorithm is demanding a lot of CPU power or RAM. If your algorithm is demanding a lot of resources, consider using more powerful nodes on the next optimization job or improving the efficiency of your algorithm.
