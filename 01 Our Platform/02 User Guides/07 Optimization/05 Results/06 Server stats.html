What are the server stats:
-A widget to show CPU and RAM usage of the hardware nodes you selected in the wizard as the optimization progresses.
-It also displays how long the optimization process has been running and the number of hardware nodes that are being used to optimize

Why would I use the server stats:
-To see your CPU and RAM usage
-If the algorithm produces to much data (700MB), it's not capable of uploading the results, so the optimization looks like it never finished
