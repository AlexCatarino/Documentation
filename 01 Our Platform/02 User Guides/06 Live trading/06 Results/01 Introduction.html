What are the live trading results:
-A page to monitor and see the results of your live trading algorithm

Why is there a result page:
-To inform you of how your algorithm is doing
-To see if you need to stop the algorithm
-To see if you need to adjust the algorithm holdings
-To see if orders are being placed/filled properly
-To see the state of the algorithm
-To download information regarding orders and logs
-To see the code that's running
-To see the server statistics
    - CPU usage, ram, host, up-time
