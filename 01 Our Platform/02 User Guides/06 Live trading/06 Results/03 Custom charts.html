What are custom charts:
-Custom line and scatter plots that you can add to your algorithms to be displayed in the live trading results page

What custom charts do we support:
-Line and scatter plots
-Can customize color & shapes

What are custom charts used for:
- You can chart variable values over time
- Useful if you hit your logging limits

Limitations:
- we limit the number of points a chart can have to 4000
- because intensive charting generates hundreds of megabytes (200MB) of data
- far too much to stream online or display in a web browser
- you see the error Exceeded maximum points per chart, data skipped
- Refer to the Charting docs (link) for more info [since this will take a full page]
