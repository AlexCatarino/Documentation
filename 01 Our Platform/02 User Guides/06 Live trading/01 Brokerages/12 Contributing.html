What is contributing:
-Adding a new brokerage to the QC platform
-Using the Brokerage Development Guide
-You'll make a PR to the LEAN repo

<h4>Requirements
-Open API (not private)
    - Incase we need to debug
-Needs to be supported by the brokerage
-Popular (used by many people)
-Stable

<h4>Wish list
-A list of brokerages we or the community want developed
-To request new brokerages that fulfill the requirements, contact support@quantconnect.com

<h4>Compensation (Why contribute)
-QCC
-If you want to implement:
    - Please contact us first (may already be under development)
