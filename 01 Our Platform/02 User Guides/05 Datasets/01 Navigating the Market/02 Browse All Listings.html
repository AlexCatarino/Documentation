The homepage of the Dataset Market displays all the datasets we have available. Each dataset on the homepage displays the name, description, coverage, start date, and price of the dataset. Coverage is the number of assets, securities, contracts, currency pairs, or articles that the dataset includes. You can select a subset of the datasets by using the following functions search features:

  • Filter for datasets in a specific category
  • Filter for datasets that have either "Cloud only" or "cloud and download" delivery options
  • Search for datasets by their name or description
