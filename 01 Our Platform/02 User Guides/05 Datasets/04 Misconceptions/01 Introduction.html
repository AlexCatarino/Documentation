What are misconceptions:
- Some data issues are reported that aren't actually data issues, they are from a misunderstanding of how the data is collected, timestamped, formatted, and normalized.

Why are there misconceptions:
- They are caused by assumptions that the data
   - should be the same across different platforms
   - should have the same timezones
   - should be timestamped a certain way
   - should have consistent normalization across data sources
