The Alpha listing page displays charts to show the algorithm's returns per trade, per day, per month, per year, and the cumulative returns over the backtest and live trading periods.

Returns Per Trade

The Returns Per trade plot displays a histogram that shows the distribution of returns per trade. The red dotted line represents the average return per trade.

Daily Returns

The Daily Returns plot displays the daily returns during the backtest and live trading periods. Blue bars represent profitable daily returns during the backtest, orange bars represent profitable daily returns during live trading, and grey bars represent unprofitable daily returns.

Monthly Returns

The Monthly Returns plot displays the return of each month. We convert the original equity curve series into a monthly series and calculate the returns of each month. Green cells represent months with a positive return and red cells represent months with a negative return. Months that have a greater magnitude of returns are represented with darker cells. Yellow cells represent months with a relatively small gain or loss. White rectangles represent months that are not included in the backtest or live trading periods. The values in the cells are percentages.

Annual Returns

The Annual Returns plot displays the return of each year. We calculate the total return within each year and represent each year with a blue bar. The red dotted line represents the average of the annual returns.

Cumulative Returns

The Cumulative Returns plot displays the cumulative returns of the Alpha. The blue line represents the Alpha and the gray line represents the benchmark.
