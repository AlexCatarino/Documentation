The Alpha listing page displays all of the information in the backtest report and additional information.

Returns

Daily Returns

This chart displays the returns of each day. Orange bars represent profitable daily returns during live trading.

Monthly Returns

This chart displays the returns of each month.

Cumulative Returns

This chart displays the cumulative returns. The orange line represents the live trading period.

Rolling Statistics

Rolling Portfolio Beta

This chart displays rolling portfolio beta over trailing 6 and 12 month periods. The orange line represents the 6 month period during live trading and the yellow line represents the 12 month period during live trading.

Rolling Sharpe Ratio

This chart displays the rolling portfolio Sharpe ratio over a trailing 6 month period. The orange line represents the live trading period.

Leverage

This chart displays the utilization of leverage over time. The orange bars represent the live trading period.

Exposure

This chart displays the long-short exposure by asset class over time. Each asset class and trading period is given its own unique color of bars in the plot.
