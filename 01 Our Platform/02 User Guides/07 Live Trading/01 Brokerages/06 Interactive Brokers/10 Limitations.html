Note the following limitations for the InteractiveBrokersBrokerageModel.

Plans

You need an IBKR PRO plan. The IB API does not support the IBKR LITE plan.

Connections

You can only have 1 connection at a time to your account. To run more than 1 algorithm, open an IB subaccount for each additional algorithm.

Feeds

You need a subscription for each IB data feed that you use in live algorithms.

ETFs

You may not be able to trade all assets with the InteractiveBrokersBrokerageModel. For example, if you live in the EU, you canâ€™t trade US ETFs. Check with your local regulators to know which assets you are allowed to trade. You may need to adjust settings in your brokerage account to live trade some assets.

API Outages

The InteractiveBrokersBrokerageModel uses the IB API to place live trades. Sometimes the API may be down. Check the IB status page to see if the API is currently working.

SMS 2FA

We don't support Two-Factor Authentication (2FA) via SMS. Use the IB Key protocol in IBKR Mobile instead.

System Resets

If your IB account has 2FA enabled, you'll have to re-authenticate once a week. The reset schedule is available on the IB Status page. If you miss the timeout period, your algorithm quits executing.
