When you deploy a live algorithm with the IB brokerage, you can use the QC data feed, the IB data feed, or both. If you use both data feeds, Lean gives priority to the QuantConnect data feed. If our data feed doesn't have a stream for the securities you request, Lean starts using the IB data feed. This process makes it possible to use our data feed for Equity universe selection and then place Options trades on the securities in the universe.
