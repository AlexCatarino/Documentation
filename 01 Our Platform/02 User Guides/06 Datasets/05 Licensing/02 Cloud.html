If you have a Cloud license for a dataset, you can access the dataset in the Algorithm Lab for research, backtests, and live trading. The cost of the license is added to your monthly bill, which you can pay with your organization's credit card or QuantConnect Credit balance. With one Cloud license for a dataset, all of the members in your organization can access the dataset in the cloud. If you have more than 3 Alphas listed in the Alpha Streams market, you can run backtests in the cloud without paying the Cloud license fee. You can add and remove cloud licenses at any time in the Algorithm Lab.
