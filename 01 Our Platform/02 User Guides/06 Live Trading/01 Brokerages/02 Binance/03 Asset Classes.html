The BinanceBrokerageModel supports trading Cryptocurrencies.

AddCrypto("BTCUSDT", Resolution.Minute, Market.Binance);
self.AddCrypto("BTCUSDT", Resolution.Minute, Market.Binance)

If you call SetBrokerageModel with the correct BrokerageName, then you don't need to pass a Market argument to AddCrypto because the market will be automatically mapped.

Subscribe to GitHub Issue #6002 to be notified when the BinanceBrokerageModel supports trading Futures.
