The Binance brokerage supports the following order types:

  • MarketOrder
  • LimitOrder
  • StopLimitOrder

You can't update orders, but you can cancel an existing order and then create a new order with the desired properties.

var ticket = LimitOrder(_symbol, 1, data[_symbol].Price-10);
ticket.Cancel();
ticket = LimitOrder(_symbol, 1, data[_symbol].Price-100);
ticket = self.LimitOrder(self.symbol, 1, data[self.symbol].Price-10)
ticket.Cancel()
ticket = self.LimitOrder(self.symbol, 1, data[self.symbol].Price-100)
