The BinanceBrokerageModel supports cash accounts.

SetBrokerageModel(BrokerageName.Binance, AccountType.Cash);
self.SetBrokerageModel(BrokerageName.Binance, AccountType.Cash)

The BinanceBrokerageModel supports paper trading through the Binance Spot Test Network. You can create API credentials for the Spot Test Network without a Binance account.
