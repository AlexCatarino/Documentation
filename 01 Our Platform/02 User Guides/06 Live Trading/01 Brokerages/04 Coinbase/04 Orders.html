Order Types

The Coinbase brokerage supports the following order types:

  • MarketOrder
  • LimitOrder
  • StopMarketOrder (supported after 3/23/20193)
  • StopLimitOrder

    





Updates


The Coinbase brokerage model does not support order updates. However, you can cancel an existing order and then create a new order with the desired arguments.




    

    





Time In Force


The Coinbase brokerage model supports the GoodTilCanceled TimeInForce.
