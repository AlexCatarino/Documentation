The Coinbase brokerage supports the following order types:

  • MarketOrder
  • LimitOrder
  • StopMarketOrder (supported after 3/23/20193)
  • StopLimitOrder

    





You can't update orders, but you can cancel an existing order and then create a new order with the desired properties.




    

    





The Coinbase brokerage model supports the GoodTilCanceled time in force.
