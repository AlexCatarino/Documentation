Note the following limitations for the IB brokerage model.

Plans

You need an IBKR PRO plan. The IB API does not support the IBKR LITE plan.

Connections

You can only have 1 connection at a time to your account. To run more than 1 algorithm, open an IB subaccount for each additional algorithm.

Feeds

You need a subscription for each IB data feed that you want to use. Refer to the Data Feeds documentation for more information.

ETFs

API Outages

The IB brokerage model uses the IB API to place live trades. Sometimes the API may be down. Check the IB status page to see if the API is currently working.
