Note the following limitations on the number of live trading nodes your organization can have and the number of brokerage connections you can have.

Maximum Number of Nodes

You need a live trading node for each simultaneous algorithm that you deploy. The tier of your organization determines the number of live trading nodes to which your organization can subscribe. The following number of live trading nodes are available for each tier:

  • Free: 0
  • Quant Researcher: 2
  • Team: 10
  • Trading Firm: No limit
  • Institution: No limit

Brokerage Connections

You need a brokerage connection or account for each simultaneous algorithm that you deploy. If you want to deploy multiple algorithms using a single brokerage, create subaccounts in your brokerage account so that each algorithm has its own set of brokerage connection credentials. If you log in to your brokerage account while an algorithm that's running is connected to the account, the algorithm may stop executing.
