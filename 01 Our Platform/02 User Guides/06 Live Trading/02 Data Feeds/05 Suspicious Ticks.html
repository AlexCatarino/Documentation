Tick data is raw and unfiltered, so it can contain a lot of noise. If a tick is not tradable, we flag it as suspicious and don't include it in our TradeBars and QuoteBars. If you use tick data, you should avoid using suspicious ticks in your algorithms as informative data points. We recommend only using tick data if you understand the risks and are able to perform your own tick filtering. Ticks are flagged as suspicious in the following situations:

  • The tick occurs below the best bid or above the best ask
  • The tick occurs far from the current market price
  • The tick occurs on a dark pool
  • The tick is rolled back
  • The tick is reported late
