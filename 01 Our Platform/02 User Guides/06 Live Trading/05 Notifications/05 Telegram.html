Enter a Telegram user or group ID into the deployment wizard to set up Telegram notifications. If you are not using our Telegram notification bot, you need to also provide the token of your bot when you set up Telegram notifications. These notifications are rate-limited to 20 requests per 30 minutes. If you manually send a Telegram notification, you must provide a message.
