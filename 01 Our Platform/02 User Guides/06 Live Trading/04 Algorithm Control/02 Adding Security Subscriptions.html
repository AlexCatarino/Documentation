The live performance page enables you to manually create security subscriptions for your algorithm instead of calling AddsecurityType in your code files. The IDE provides fields for each of the arguments that the AddsecurityType method accepts, including the following:

  • Symbol
  • Security type
  • Resolution
  • Leverage
  • Market
  • Fill forward
  • Extended market hours

Add manual security subscriptions to your algorithm so that you can plan manual trades through the web IDE without having to edit and redeploy the algorithm. However, note that you can't manually remove security subscriptions from the web IDE during live execution.
