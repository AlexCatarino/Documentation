The live performance page enables you to manually create security subscriptions for your algorithm instead of calling the AddsecurityType methods in your code files. The live performance page provides fields for each of the arguments that the AddsecurityType methods accept, including the following:

  • Security type
  • Symbol
  • Resolution
  • Market
  • Fill forward
  • Leverage
  • Extended market hours

Add manual security subscriptions to your algorithm so that you can place manual trades through the web IDE without having to edit and redeploy the algorithm. However, note that you can't manually remove security subscriptions from the web IDE during live execution.
