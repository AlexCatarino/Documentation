What is sharing:
     - Generating a link to a backtest (and the project files) to give to others
     - Can just give the link to others, attach to a forum post, or embed into a website
     - You can shares once a backtest is complete. You can share earlier, but the link won't have the full backtest results
     - Links can be opened in any browser
Why share:
     - Share to the forum if you need guidance or are reporting an issue
     - Attach to a data issue if it highlights a data issue
     - Sharing is a good way to get feedback on your algorithm to improve efficiency or design from others
     - Share the results of an algorithm with others (calculated for you automatically)
     - It's possible to share a backtest with others (to present the results) without sharing the actual code that produced the results

-Click a button and 2 links is generated for you
   - One opens the backtest results in the browser
   - The other is used to embed into a website
