-What is it: 
        - process of creating a fresh copy of a project and saving it within the same organization
        - project files are duplicated
        - backtest results and the content stored in the ObjectStore is not retained
-Why clone:
        - Helpful for making changes to a file before merging the changes back to the original
        - Helpful for quickly deleting ObjectStore storage (clone then delete the original project)
        - Gives you a new copy of the project to adjust for further research without adjusting the original
