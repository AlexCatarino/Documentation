What are the file limits:
     -Code files can contain up to 64,000 characters. You'll see the error message.
     -Notebook files can be upto 128KB. After that, they won't save.

Why are there file limits:
     - To encourage users to split up their files into modular components
     - If you need more than 64k characters in a code file => either:
         - separate the content into multiple files
         - if just data, import the data from a custom data source or read the data from the ObjectStore
     - If you need more than 128KB in a notebook file => either:
         - Move the code in the notebook to a code file and then import it into the notebook for usage
         - if just data, import the data from a custom data source or read the data from the ObjectStore
