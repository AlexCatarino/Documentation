What is debugging: Debugging is the process of diagnosing and resolving errors in your code
Why use debugging:
      - Required to have an error-free project
      - Developers spend a large portion of their time debugging code that they have written
Methods to debugging:
   - Log/Debug/Error/Quit statements
   - Built-in debugger
         -Debuggers work by stepping through your code and with breakpoints and variable observation
         -You can control the steps, place breakpoints, what variables to watch
    - Options to use local IDE debugging through CLI
    - Charting is an alternative option
