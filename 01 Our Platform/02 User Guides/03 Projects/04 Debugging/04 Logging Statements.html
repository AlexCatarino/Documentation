Logging statements are statements that you can add to your algorithm in order to add information to the log file, add information to the Console, or even quit execution. Use these statements to debug your backtests and live algorithms. Consider adding them in the code block of an if statement to signify an error has been caught.

Log

Log statements are added to the log file while your algorithm continues executing. Logging dataset information is not permitted. Use Log statements to debug your backtests and live trading algorithms. Log length is capped by organization tier. If your organization hits the daily limit, contact us. Furthermore, if multiple messages are sent with the same content, only the first message is added to the log file. Add a timestamp to your log messages to bypass this rate-limiting.

Log("My log message");
self.Log("My log message")

Debug

Debug statements are the same as log statements, but Debug statements are displayed in green text in the Console. Use these statements when you want to give more attention to a message in the Console. Debug messages are capped at 200 characters in length. If multiple debug statements are sent within one second, your messages are rate-limited to avoid crashing your browser.

Debug("My debug message");
self.Debug("My debug message")

Error

Error statements are the same as log statements, but Error statements are displayed in red text in the Console. Use these statements when you want to give the most attention to a message in the Console. Error statements are rate-limited like debug statements.

Error("My error message");
self.Error("My error message")

Quit

Quit statements cause your project to stop running and may log some data to the log file and Console. These statements are displayed in green text in the Console. The Quit statement does not work in the Initialize method. When you call this method, it proceeds to the end of the method definition. If you want to quit execution immediately, return after you call Quit.

Quit("My quit message");
self.Quit("My quit message")
