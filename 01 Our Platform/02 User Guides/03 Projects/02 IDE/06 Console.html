-Console window as at the bottom of Algorithm Lab
-Helpful for viewing algorithm state
   - Debug messages
   - Errors
   - Log file

- List of commands
    clear : Clear the console.
    help : Show console help.
    pid : Return the project id.
    uid : Return your user id.
    oid : Return the active organization id.
    jedi : Turns the IDE in to Jedi Mode.
    moo : Make Console Moo.
    lean : Display selected Lean Version Data.

       - Lean version data? To check if GitHub source code has an issue in it(?)

h4: timezone
EST time zone is used for console timestamps
-Changing timezone of algorithm with SetTimeZone doesn't change console window timestamps
-**Real-time** timestamps on far left, but your backtest & live trading timestamps are in whatever timezone you specify
