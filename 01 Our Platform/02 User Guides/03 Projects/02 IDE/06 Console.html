The Console is a panel at the bottom of the Algorithm Lab that you can use to execute commands or view build reports, logs, and errors.

The following table describes all of the commands the Console accepts:

Command Description
clear Clear the console.
help Show console help.
pid Return the project id.
uid Return your user id.
oid Return the active organization id.
jedi Turns the IDE into Jedi Mode.
moo Make the Console Moo.
lean Display the selected Lean Version Data.

The timestamps in the left column of the Console are always in EST time. Changing the timezone of your algorithm with the SetTimeZone method does not change the timezone of the timestamps in the left column of the Console. If you issue a Log or Debug message in your algorithm and provide the algorithm time in the message, the timestamp in the message will be in the algorithm timezone.
