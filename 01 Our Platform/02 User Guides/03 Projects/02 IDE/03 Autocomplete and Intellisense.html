What is it:
    - GUI tool that shows options for auto-complete and presents the members that are accessible from the current object
    - A search algorithm attempts to find the correct statement you're looking for, given the context.
    - You can use it to autocomplete method names and object attributes.
    -  It doesn't work with user-defined members/variables. Also doesn't work with class names (SimpleMovingAverage)
Why use it:
    - Speed up your development
    - If you don't want to use it in our cloud IDE, local autocomplete is also available for select IDEs
          -Refer to https://www.quantconnect.com/docs/v2/lean-cli/tutorials/local-autocomplete


- Type CRTL-Space and a box appears with suggestions on code completion. You can click or press Tab to auto-complete the statement
- The box that appears includes
     - Member type
     - Member description
     - If method => show description of the method and list the parameters it accepts
