-Web IDE with autocomplete/intellisense support
     - Definition: Tools to help speed up your development by showing options for auto-complete, and presenting the members that are accessible from the object. A search algorithm attempts to find the correct statement you're looking for, given the context.
    - Accessiblility: You can use it to autocomplete method names and object attributes. It doesn't work with user-defined members/variables. Also doesn't work with class names (SimpleMovingAverage)
     - Type CRTL-Space and a box appears with suggestions on code completion. You can click or press Tab to auto-complete the statement
     - The box that appears includes
             - Member type
             - Member description
             - If method => show description of the method and list the parameters it accepts
     - Include gif demo like seen here: https://code.visualstudio.com/docs/editor/intellisense

-Local autocomplete is also available for select IDEs
