-What is it:
    - Window in the IDE (show image) that list of all properties and methods of QCAlgorithm class
    - Properties include type and description
    - methods include description, argument types/names, and return type.
    - There are 2 symbols:
        - Wrench: Property
        - Cube: Method

-Why use it:
    - Find which methods/properties you have available to use (and what args they expect) without referencing the documentation
    - Faster development
