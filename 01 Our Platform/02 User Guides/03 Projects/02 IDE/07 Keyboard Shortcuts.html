Keyboard shortcuts are combinations of keys that you can issue to manipulate the IDE. They can speed up your workflow because they remove the need for you to reach for your mouse. The following table displays some of the helpful keyboard shortcuts:

Shortcut Description
Ctrl + Space Initiate full auto-complete
Ctrl + / Toggle comments on selected code
Ctrl + Up/Down Increase/decrease font size
Ctrl + F Find
Ctrl + H Replace
Ctrl + S Save the document and compile the project
Ctrl + D Remove line
Ctrl + Z Undo
Ctrl + Y (or Ctrl + Shift + Z) Redo
Alt + Up/Down Move line up/down
Alt + Shift + Up/Down Copy lines up/down
Alt + 0 (zero) Collapse all code blocks
Alt + Shift + 0 (zero) Expand all code blocks

For a full list of keyboard shortcuts, refer to the ACE Editor wiki.
