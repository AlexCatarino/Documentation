-C# and Python

      - The engine is written in C# so C# is the native language.
      - Python is supported using pythonnet.

Pros and cons of each language:
- C# is faster. It's easier to contribute to lean if you have features written in C# modules. 
     - C# is strongly type
- Python has less syntax, more third-party libraries, the algo trading community uses mostly Python, may be easier to learn
     - Python is dynamic(?) type
     - Python is the native language for the notebooks, easier to use
