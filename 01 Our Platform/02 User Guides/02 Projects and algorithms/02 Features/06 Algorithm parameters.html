The algorithm parameter functionality enables you to assign values to your algorithm parameters outside of your code files. To get the parameter value into your algorithm, you use the $[GetParameter(), M:QuantConnect.Algorithm.QCAlgorithm.GetParameter] method. Alternatively, in C# projects, you can apply the Parameter attribute to define public class members with algorithm parameter values. By adding algorithm parameters to your code files, you unlock the ability to use the parameter optimizer.
