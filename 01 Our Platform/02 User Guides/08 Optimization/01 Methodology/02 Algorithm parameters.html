What are parameters:
- Values for variables to use in your algorithm
- They are stored outside of your algorithm code but the values of the parameters are injected into your code when it runs
- Parameters can be any text string, but the optimizer only works with numerical parameters
- "Optimization parameters are variables detected from your code which are iterated through over multiple parallel backtests" // IDE description

Why use parameters:
- Parameters are required in order to perform optimization. The optimizer adjusts the value of the parameters automatically.
- The optimizer will adjust the value of these parameters from a minimum value to a maximum value in whatever step size you define

How(?):
- Parameters can be added in the IDE (provide the name of the parameter and you can add a default value [optional])
- Max of 2 parameters can be optimized
- Use the GetParameter method to fetch the value of the parameter and inject it into your algorithm
    - GetParameter returns a string so you may need to cast it
