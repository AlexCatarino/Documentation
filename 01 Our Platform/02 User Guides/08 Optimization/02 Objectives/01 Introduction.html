What is an optimization target:
- A backtest metric (or "property") that you'd like to optimize

Why do I need an optimization target:
- The target is used to quantify the optimizer parameter adjustment decisions
- It's the objective function the optimizer is working to optimize

What targets are available:
-Sharpe
-Compounding annual return
-PSR (link to forum post that defines the PSR)
-Drawdown
-^ Each of these targets can be maximized or minimized. Drawdown is the only one you'd want use minimize for
