What is an optimization strategy:
- The optimization strategy decides each batch of parameters for the optimizer.
- One backtest is run for each batch of parameter values

Why do I need an optimization strategy:
- Because it defines how the optimizer selects which parameters to adjust and how to adjust their values
- Optimization can't be performed without selecting an optimization strategy

What optimization strategies are available:
- Grid search is the most complete, but the most expensive option which looks through all combinations of parameters.
    - "Brute force"

How can I contribute an optimization strategy(?):
- To add a new optimization strategy please make a pull-request to LEAN.
