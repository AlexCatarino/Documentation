To access fundamental data in your algorithms, in addition to coarse universe selection, you need fine universe selection. The fine universe selection method you define accepts FineFundamental objects and returns a list of Symbol objects. The FineFundmental objects are updated once per day. FineFundmental objects contain the following data attributes for each Equity asset:

  • CompanyReference
  • SecurityReference
  • FinancialStatements
  • EarningReports
  • OperationRatios
  • EarningRatios
  • ValuationRatios
  • CompanyProfile
  • AssetClassification
  • Symbol - Symbol representation for the underlying Security
  • Price - Yesterday’s closing price
  • IsFillForward - True if this piece of data is filled forward

To include FineFundamental objects in your algorithm, you need the US Fundamental Data from Morningstar. This dataset covers 5,000 US Equities, starting in January 1998, and is delivered on a daily frequency. It’s created by using a combination of string matching, Regular Expressions, and Machine Learning to gather the fundamental data published by companies.
