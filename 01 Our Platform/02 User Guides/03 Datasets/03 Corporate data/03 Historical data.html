Historical corporate data can be accessed in the research environment using the GetFundamental method. In the following snippet, the daily PE ratio for Apple is gathered from January 1, 2020 to today. For some quarterly or yearly reported fundamentals like FinancialStatements and OperationRatios, you need to specify a period. For example, EarningReports.BasicAverageShares.TwelveMonths returns the one year value of the basic average shares, while EarningReports.BasicAverageShares.ThreeMonths provides the three month value. However, since the following snippet uses the PE ratio, a period doesn’t need to be selected.

qb = QuantBook()
aapl = qb.AddEquity("AAPL").Symbol
pe_history = qb.GetFundamental(aapl, 
                               "ValuationRatios.PERatio", 
                               datetime(2020, 1, 1), 
                               datetime.now())

It’s not currently possible to retrieve historical corporate data in the backtesting and live trading environments. However, we are working on adding this functionality. Subscribe to our progress via GitHub Issue #4890.
