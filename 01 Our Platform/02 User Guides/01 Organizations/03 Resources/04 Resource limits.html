The type of organization a hardware resource is assigned to determines the functionality of the resource. For each organization type, the following resource limitations are imposed:

  • the amount of logging that can be done per backtest
  • the amount of logging that can be done per day
  • the number of backtest nodes an organization can have
  • the number of research nodes an organization can have
  • the number of live trading nodes an organization can have
