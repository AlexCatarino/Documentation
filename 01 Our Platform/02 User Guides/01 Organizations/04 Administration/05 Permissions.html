Each member of your organization has access to the resources within the organization. However, if you are the manager of a Trading Firm or Institution organization, you can grant additional permissions to your members. There are several categories of permissions you can grant to members.

Node Permissions

The following table shows the supported node permissions:

Permission Description
Create Permission to add new nodes
Stop Permission to stop running nodes
Delete Permission to delete existing nodes

Team Permissions

The following table shows the supported team permissions:

Permission Description
Add Permission to add new members
Remove Permission to remove existing members
Edit Permission to change the permissions of other members

Storage Permissions

The following table shows the supported team permissions:

Permission Description
Create Permission to write to the Object Store
Delete Permission to delete data in the Object Store
Billing Permission to subscribe to more space in the Object Store

Alpha Token Permissions

The following table shows the supported Alpha token permissions:

Permission Description
Create Permission to create a new token
Delete Permission to delete an existing token
Read Permission to read tokens

Alpha Exchange Permissions

The following table shows the supported Alpha exchange permissions:

Permission Description
Read Permission to read from the Alpha Streams exchange
