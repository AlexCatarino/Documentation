The Quant Researcher tier builds on the features included in the Free tier. Organizations on the Quant Researcher tier have access to the QuantConnect API and the Alpha Streams Market. These organizations can subscribe to the CLI add-on to run LEAN locally and can subscribe to the Alpha Analytics API add-on to perform bulk research, licensing, and correlation analysis on the Alphas in the Alpha Market. When members in these organizations need assistance from a QuantConnect engineer, support seats are available to request private support.

In Quant Researcher organizations, members can use second and tick resolution data from the Datasets Market. There is no limit on the number of projects these organizations can hold. They can produce up to 100KB of logs/backtest, 1MB of logs/day. 10 million orders/backtest, and run up to two concurrent backtests by using two backtesting nodes. After a successful backtest, members in Quant Researcher organizations can use parameter optimization tools to improve the performance of their backtest. When you're ready to deploy your strategies live, Quant Researcher organizations can subscribe to up to 2 live trading nodes to unlock live trading with real or fictional money.
