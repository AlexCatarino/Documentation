Institution organizations have all of the features included in Trading Firm organizations and more features are included. This organization has no limit on the amount of backtest logs it can produce. Institution organizations gain access to the Bloomberg ESMX Brokerage to live trade Equities, Futures, and Options via the Bloomberg ESMX. These organizations can request custom libraries and frameworks to use in the QuantConnect web IDE. Additionally, Institution organizations can receive instant messaging support from a QuantConnect engineer.
