What is the API:
- A way to interact with QuantConnect.com without using the website GUI

Why use the API:
- Use instead of navigating the GUI
- A lot of the functionality listed here can be accomplished by navigating the GUI, but the API can be used to perform deeper analysis on backtest results (ReadBacktest) and live algorithms (ReadLiveAlgorithm)


<h3>Project management
What is project management with the API:
- The API gives you the ability to:
   - create projects
   - get details about a single project
   - list details of all projects
   - add a file to a project
   - update the name of a file
   - update the contents of a file
   - read all files in a project
   - read a single file in a project
   - delete a file in a project
   - delete a project


<h3>Backtest management
What is backtest management with the API:
- The API gives you the ability to:
   - Create a new compile job request
   - Read a compile packet job result
   - Create a new backtest request
   - Read out a backtest (results)
   - Update a backtest name
   - List all the backtests for a project
   - Delete a backtest
   - Read out the report of a backtest in the project id specified

Why retrieve backtest information:
-The backtest results from the API method include more statistics than just that shown in the backtest results page in the GUI

<h3>Live algorithm management
What is live algorithm management with the API:
- The API gives you the ability to:
    - Create a live algorithm
    - Get a list of your live running algorithms
    - Read out a live algorithm (results)
    - Liquidate a live algorithm
    - Stop a live algorithm
    - Gets the logs of a specific live algorithm

<h3>Dataset management
What is dataset management with the API:
- The API gives you the ability to:
    - Get the link to the downloadable data
    - Get valid data entries for a given filepath from data/list
    - Gets data prices from data/prices
    - Purchase and download data from QuantConnect


<h3>Organization management
What is resource management with the API:
- The API gives you the ability to:
    - Create a new node in the organization
    - Read the nodes associated with the organization
    - Update an organizations node with a new name
    - Delete a node from an organization
    - Stop a running node in a organization
    - Read the organization account status
    - Get a list of organizations tied to this account
    - Fetch organization data from web API
