What is tslearn:
- tslearn is a Python package that provides machine learning tools for the analysis of time series. This package builds on (and hence depends on) scikit-learn, numpy and scipy libraries.
- Supported models: classification, clustering and regression (https://github.com/tslearn-team/tslearn#available-features)
- Link to docs: https://tslearn.readthedocs.io/en/stable/
