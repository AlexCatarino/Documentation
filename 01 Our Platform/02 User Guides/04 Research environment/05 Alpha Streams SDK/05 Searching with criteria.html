What:
-There are methods in the SDK to search for alphas (SearchAlphas) and authors (SearchAuthors) by providing search criteria
-Criteria to search alphas by:
    - Symbols traded
    - Asset classes traded
    - Author ID
    - Tags (include / exclude)
    - Minimum/maximum insight accuracy, fees, parameter count, shape, uniqueness, DTW, return correlation
-Criteria to search authors by:
    - Location
    - Languages
    - Bio
    - Min/max number of alphas listed, sign up time, last login, forum discussions, forum comments, number of projects

Why:
-Enables you to quickly search for alphas and authors that meet your criteria and investment objectives
