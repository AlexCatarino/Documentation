What is the SDK:
-Software Development Kit (SDK)
-A tool to enable you to perform automated bulk alpha research, licensing, and correlation analysis
-implementations for both Python and C#, connected to the same REST API backend

Why use the SDK:
-can collect the insights from the alphas you have licensed from the Alpha Market and process them however you want
-Without this bundle, you need one brokerage account for every alpha
-Streamlines your alpha research


<h3>Generating lists
What:
-The SDK has a method to list of IDs for all of the alphas listed in the Alpha Market (GetAlphaList)
-The SDK has a method to list all of the tags given to alphas Alpha Market, along with the number of alphas that have the associated tag (GetAlphaTags)

Why use these:
-The alpha ID is needed to get more information on the alpha for further analysis
-Tags are useful because if you want to search for an alpha with a specific tag, you need to know what tags are currently used in the Alpha Market.
-The number of alphas that have a tag is useful information because if there is a low number of alphas with a certain tag, the alphas with that tag may have lower correlation with the other alphas in the market


<h3>Inspecting alphas
What:
-There are methods in the SDK to gather information on a specific alpha:
   - Alpha stats (GetAlphaById)
          - Authors, Asset classes traded, Insight accuracy, number of backtests run, description,
          - market depth, number of development hours, uniqueness (correlation with the rest of the market)
          - Sharpe ratio, status, tags, number of parameters, DTW distance, backtest vs live correlation, capacity
   - Insights (GetAlphaInsights)
   - Runtime errors (GetAlphaErrors; Including time, the error, and the stack trace)
   - Equity curve (GetAlphaEquityCurve; In a DataFrame)
   - Orders and OrderEvents (GetAlphaOrders; Including order type, quantity, status, direction, symbol, etc)
             - Can use DownloadOrders to save to file
   - Holdings (GetHoldings; Current portfolio weights for the alpha)
-Alpha IDs can be gathered from GetAlphaList or in the Alpha Market URLs.

Why:
-To get a close look at an individual alpha in the market
-To calculate correlation of returns between individual alphas or an alpha and your current investment portfolio
-To copy the portfolio weights of a particular alpha model
-To see how an alpha has performed live relative to its backtest (dissimilar performance may be a red flag / luck)


<h3>Inspecting authors
What:
-There is a method in the SDK to get information on a specific Alpha Streams author (GetAuthorById)
     - Alphas in the market, Avg # of hours of development per alpha, number of backtests, bio, number of forum discussions
     - Number of forum comments, language, last time online, location (city, state, country), Number of projects
     - Sign up time, social media links

Why:
-Provides more information to analyze the Alpha Market with
-Provides the ability to see social media profiles of the quants you're investing in
-To see how experienced the quant it and how active they are in QC community and on the QC platform
-To see the bio of the authors your investing in


<h3>Searching with criteria
What:
-There are methods in the SDK to search for alphas (SearchAlphas) and authors (SearchAuthors) by providing search criteria
-Criteria to search alphas by:
    - Symbols traded
    - Asset classes traded
    - Author ID
    - Tags (include / exclude)
    - Minimum/maximum insight accuracy, fees, parameter count, shape, uniqueness, DTW, return correlation
-Criteria to search authors by:
    - Location
    - Languages
    - Bio
    - Min/max number of alphas listed, sign up time, last login, forum discussions, forum comments, number of projects

Why:
-Enables you to quickly search for alphas and authors that meet your criteria and investment objectives


<h3>Managing bids
What:
- There are methods in the SDK to create (CreateBid), fetch (GetBidById), and remove (RemoveBid) bids in the Alpha Market.
- When fetching bid information, you'll receive the expiration time of the bid, allocation lots, license period, and maximum price

Why:
-Provides automated services to manage your Alpha Market bids
-You could remove a bid from an alpha if the recent performance of the alpha is poor
