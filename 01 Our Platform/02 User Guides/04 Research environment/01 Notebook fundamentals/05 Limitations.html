What are the limitations:
-Research nodes aren't as powerful as backtesting nodes
-You can't currently import C# objects from the .cs project files into your notebook, but you can for Python
-Universe selection not currently supported
    - https://github.com/QuantConnect/Lean/issues/1614
-Collaboration is not currently supported
-File save limit: 128KB

Why:
-Research nodes aren't as powerful because you will work with static data in Research.
    -In Research, you can control the amount of "snap shot" (the amount of data you're analyzing). In backtesting, you have a lot of data to process.
    -Research is a picture, backtesting is a movie, live trading is real life (emotion)
-Collaboration is not supported because it's not supported by Jupyter
-The reason for the file size limit on notebooks is because whenever there is a change to a file, the entire file is sent to our servers to save.
    - The file limit ensures a fast and successful upload/download communication.
    - For that reason, we encourage users to break up their files into manageable sizes.
