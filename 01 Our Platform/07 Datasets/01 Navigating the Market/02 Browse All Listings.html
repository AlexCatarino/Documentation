The Dataset Market displays all our supported datasets. To view the page, in the top navigation bar, click Data.

Each dataset displays the name, description, coverage, start date, and price of the dataset. Coverage is the number of assets, securities, contracts, currency pairs, or articles that are included in the dataset. You can search the market using the following metrics:

  • Name
  • Description
  • Category
  • Delivery options (cloud/download)

To view the listing page of a dataset, click the dataset.
