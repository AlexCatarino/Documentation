You must have an active subscription to the security in your live algorithm to place a manual order for the security.

Follow these steps to place manual orders:

  1. On the Live page, click the Holdings tab.
  2. If you don't see the security you want to place a manual order for in the Algorithm Holdings table, check the Show All Portfolio checkbox.
  3. Click the security you want to place a manual order for.
  4. If you want to liquidate the position, click Liquidate. Otherwise, follow these steps:
    1. Click Create Order.
    2. Enter an order quantity and order type.
  5. Click Submit Order.
  6. Click Close.
