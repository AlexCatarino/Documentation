Follow these steps to create an algorithm using the Atreyu brokerage:

  1. Create a new project.
  2. In the Initialize method in main.pyMain.cs, set the brokerage model to the TradierBrokerageModel and declare your AccountType.
    3. SetBrokerageModel(BrokerageName.TradierBrokerage, AccountType.Margin);
        
    self.SetBrokerageModel(BrokerageName.TradierBrokerage, AccountType.Margin)
  3. (Optional) In the Initialize method in main.pyMain.cs, set the DefaultOrderProperties.
    4. DefaultOrderProperties.TimeInForce = TimeInForce.Day;
    self.DefaultOrderProperties.TimeInForce = TimeInForce.Day
  4. If you want to send an order with properties that are not the default properties, pass an OrderProperties argument to the order method.
    5. var orderProperties = new OrderProperties { TimeInForce = TimeInForce.GoodTilCanceled };
    LimitOrder(_symbol, quantity, limitPrice, orderProperties: orderProperties);
    order_properties = OrderProperties()
    order_properties.TimeInForce = TimeInForce.GoodTilCanceled
    ticket = self.LimitOrder(self.symbol, quantity, limit_price, orderProperties = order_properties)
  5. If you want to update the quantity of an active order, Cancel the original order and then submit a new order with the desired quantity.
    6. var ticket = LimitOrder(_symbol, quantity, limitPrice, orderProperties: orderProperties);
ticket.Cancel();
    LimitOrder(_symbol, newQuantity, limitPrice, orderProperties: orderProperties);
    ticket = self.LimitOrder(self.symbol, quantity, limit_price, orderProperties = order_properties)
ticket.Cancel()
self.LimitOrder(self.symbol, new_quantity, limit_price, orderProperties = order_properties)
  6. If you want to update the non-quantity fields of an active order, Update the OrderTicket with an UpdateOrderFields object.
    7. var ticket = LimitOrder(_symbol, quantity, limitPrice, orderProperties: orderProperties);
    var orderFields = new UpdateOrderFields { 
        LimitPrice = newLimitPrice,
        Tag = newTag
    };
    ticket.Update(orderFields);
    ticket = self.LimitOrder(self.symbol, quantity, limit_price, orderProperties = order_properties)
    update_fields = UpdateOrderFields()
update_fields.LimitPrice = new_limit_price
update_fields.Tag = new_tag
ticket.Update(update_fields)
