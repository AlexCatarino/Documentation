Follow these steps to create an algorithm using the Bitfinex brokerage:

  1. Create a new project.
  2. In the Initialize method in main.pyMain.cs, set the brokerage model to the BitfinexBrokerageModel and declare your AccountType.
    3. SetBrokerageModel(BrokerageName.Bitfinex, AccountType.Cash);
    self.SetBrokerageModel(BrokerageName.Bitfinex, AccountType.Cash)
  3. In the Initialize method in main.pyMain.cs, create a security subscription and save a reference to the Symbol.
    4. _symbol = AddCrypto("BTCUSD", Resolution.Minute).Symbol;
    self.symbol = self.AddCrypto("BTCUSD", Resolution.Minute).Symbol
  4. In the Initialize method in main.pyMain.cs, deposit some of the quote currency into your CashBook.
    5. SetCash(100000);
    self.SetCash(100000)
  5. (Optional) In the Initialize method in main.pyMain.cs, set the DefaultOrderProperties.
    6. DefaultOrderProperties = new BitfinexOrderProperties
{
    TimeInForce = TimeInForce.GoodTilCanceled,
    Hidden = false,
    PostOnly = false
};
    self.DefaultOrderProperties = BitfinexOrderProperties()
self.DefaultOrderProperties.TimeInForce = TimeInForce.GoodTilCanceled
self.DefaultOrderProperties.Hidden = False
self.DefaultOrderProperties.PostOnly = False
  6. When you want to send an order with properties that are not the default properties, pass a BitfinexOrderProperties argument to the order method.
    7. var orderProperties = new BitfinexOrderProperties { PostOnly = true, Hidden = true };
    LimitOrder(_symbol, quantity, limitPrice, orderProperties: orderProperties);
        
    orderProperties = BitfinexOrderProperties()
orderProperties.PostOnly = True
    orderProperties.Hidden = True
self.LimitOrder(self.symbol, quantity, limit_price, orderProperties = orderProperties)
  7. When you want to update an order, call Update on the OrderTicket with an UpdateOrderFields object.
    8. var ticket = LimitOrder(_symbol, quantity, limitPrice);
var response = ticket.Update(new UpdateOrderFields() { 
  LimitPrice = newLimitPrice
});
if (response.IsSuccess) { 
     Debug("Order updated successfully");
