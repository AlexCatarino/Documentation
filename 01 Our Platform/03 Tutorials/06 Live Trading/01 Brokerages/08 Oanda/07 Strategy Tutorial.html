Follow these steps to create an algorithm using the Oanda brokerage:

  1. Create a new project.
  2. In the Initialize method in main.pyMain.cs, set the brokerage model to the OandaBrokerageModel and declare your AccountType.
    3. SetBrokerageModel(BrokerageName.OandaBrokerage, AccountType.Margin);
    self.SetBrokerageModel(BrokerageName.OandaBrokerage, AccountType.Margin)
  3. In the Initialize method in main.pyMain.cs, create a security subscription and save a reference to the Symbol.
    4. _symbol = AddForex("EURUSD", Resolution.Minute).Symbol;
    self.symbol = self.AddForex("EURUSD", Resolution.Minute).Symbol
  4. When you want to update an order, Update the OrderTicket with an UpdateOrderFields object.
    5. var ticket = LimitOrder(_symbol, quantity, limitPrice);
var orderFields = new UpdateOrderFields { 
        Quantity = newQuantity,
        LimitPrice = newLimitPrice,
        Tag = newTag
    };
    ticket.Update(orderFields);

    ticket = self.LimitOrder(self.symbol, quantity, limit_price)
