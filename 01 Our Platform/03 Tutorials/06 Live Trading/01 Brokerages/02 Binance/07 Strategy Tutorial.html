Follow these steps to create an algorithm using the Binance brokerage:

  1. Create a new project.
  2. In the Initialize method in main.pyMain.cs, set the brokerage model to the BinanceBrokerageModel and declare your AccountType.
    3. SetBrokerageModel(BrokerageName.Binance, AccountType.Cash);
    self.SetBrokerageModel(BrokerageName.Binance, AccountType.Cash)
  3. In the Initialize method in main.pyMain.cs, create a security subscription and save a reference to the Symbol.
    4. _symbol = AddCrypto("BTCUSDT", Resolution.Minute).Symbol;
    self.symbol = self.AddCrypto("BTCUSDT", Resolution.Minute).Symbol
  4. If you are backtesting the strategy before deploying it live, in the Initialize method in main.pyMain.cs, deposit some of the quote currency into your CashBook.
    5. SetCash("USDT", 100000);
    self.SetCash('USDT', 100000)
  5. (Optional) In the Initialize method in main.pyMain.cs, set the DefaultOrderProperties.
    6. DefaultOrderProperties = new BinanceOrderProperties
{
    TimeInForce = TimeInForce.GoodTilCanceled,
    PostOnly = false
};
    self.DefaultOrderProperties = BinanceOrderProperties()
self.DefaultOrderProperties.TimeInForce = TimeInForce.GoodTilCanceled
self.DefaultOrderProperties.PostOnly = False
  6. When you want to send an order with properties that are not the default properties, pass a BinanceOrderProperties argument to the order method.
    7. var orderProperties = new BinanceOrderProperties { 
        TimeInForce = TimeInForce.Day,
        PostOnly = true 
    };
    LimitOrder(_symbol, quantity, limitPrice, orderProperties: orderProperties);

    order_properties = BinanceOrderProperties()
    order_properties.TimeInForce = TimeInForce.Day
    order_properties.PostOnly = True
    self.LimitOrder(self.symbol, quantity, limit_price, orderProperties = order_properties)
  7. When you want to update an order, Cancel the original order and then submit a new order with the desired arguments.
    8. var ticket = LimitOrder(_symbol, quantity, limitPrice, orderProperties: orderProperties);
ticket.Cancel();
    LimitOrder(_symbol, newQuantity, newLimitPrice, orderProperties: newOrderProperties);
    ticket = self.LimitOrder(self.symbol, quantity, limit_price, orderProperties = order_properties)
ticket.Cancel()
self.LimitOrder(self.symbol, new_quantity, new_limit_price, orderProperties = new_order_properties)
