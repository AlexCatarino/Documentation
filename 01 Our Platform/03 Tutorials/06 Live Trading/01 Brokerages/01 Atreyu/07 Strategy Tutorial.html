Follow these steps to create an algorithm using the Atreyu brokerage:

  1. Create a new project.
  2. In the Initialize method in main.pyMain.cs, set the brokerage model to the AtreyuBrokerageModel and declare your AccountType.
    3. SetBrokerageModel(BrokerageName.Atreyu, AccountType.Margin);
        
    self.SetBrokerageModel(BrokerageName.Atreyu, AccountType.Margin)
  3. (Optional) In the Initialize method in main.pyMain.cs, set the DefaultOrderProperties.
    4. DefaultOrderProperties = new AtreyuOrderProperties
{
    Exchange = Exchange.BATS,
    TimeInForce = TimeInForce.Day
};
        
    self.DefaultOrderProperties = AtreyuOrderProperties()
self.DefaultOrderProperties.Exchange = Exchange.BATS
self.DefaultOrderProperties.TimeInForce = TimeInForce.Day
  4. When you want to send an order to a different exchange, pass an AtreyuOrderProperties argument to the order method.
    5. MarketOrder(symbol, quantity, orderProperties: new AtreyuOrderProperties { Exchange = Exchange.ARCA });
        
    orderProperties = AtreyuOrderProperties()
orderProperties.Exchange = Exchange.ARCA
self.MarketOrder(self.symbol, quantity, orderProperties = orderProperties)
  5. When you want to check if a there are enough shares available to fill your short order, call the Shortable method.
    6. var quantity = CalculateOrderQuantity(symbol, portfolioWeight);
var canShort = Shortable(symbol, quantity);
        
    quantity = self.CalculateOrderQuantity(self.symbol, portfolio_weight)
can_short = self.Shortable(self.symbol, quantity)
