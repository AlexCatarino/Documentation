Follow these steps to place manual orders in your live algorithm:

  1. Open the live performance page of the algorithm you want to place manual orders.
  2. On the live performance page, click the Holdings tab.
  3. If the algorithm doesn't have a subscription to the security you want to trade, follow these steps:
    1. Click Add Security.
    2. Enter the symbol, security type, resolution, leverage, and market.
    3. If you want the data for the security to be filled-forward, check the Fill Forward checkbox.
    4. If you want to subscribe to extended market hours for the security, check the Extended Market Hours checkbox.
    5. Click Add Security.

      6. "Success Command Enqueue" displays.

  4. If you don't see the security you want to place a manual order for in the Algorithm Holdings table, select the Show All Portfolio checkbox.
  5. Click the security you want to place a manual order for.
  6. If you want to liquidate the position, click Liquidate. Otherwise, follow these steps:
    1. Click Create Order.
    2. Enter an order quantity and order type.
  7. Click Submit Order.
  8. Click Close.
