Follow these steps to change the code of your live algorithm:

  1. Stop the algorithm.
  2. Edit the code files.
  3. Deploy the algorithm.

    4. If the brokerage you select in the deployment reveals the Initialize Cash State or Initialize Holdings State sections, add the cash and holdings of the algorithm you stopped.

When you stop and redeploy a live algorithm, your project's live equity curve will be retained between deployments. If you want to erase the equity curve history, clone the project and then redeploy the cloned version of the project.
