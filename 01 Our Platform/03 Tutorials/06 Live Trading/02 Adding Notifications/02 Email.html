Follow these steps to set up email notifications:

  1. On the Deploy Live page, enable at least one of the notification types.

    2. The following table shows the supported notification types:

    Notification Type Description
    Order Events Notifications for when OrderEvents occur
    Insights Notifications for when Insights are emitted
  2. Click Email.
  3. Enter an email address.
  4. Enter a subject.
  5. Click Add.

    6. If you want to add more email notifications, click Add Notification and then continue from step 2.
