Follow these steps to set up webhook notifications:

  1. On the Deploy Live page, enable at least one of the notification events.

    2. The following table shows the supported notification events:

    Environment Description
    Order Events Be notified when order events occur
    Insights Be notified when insights are emitted
  2. Click Webhook.
  3. Enter a URL.
  4. If you want to add header information, click Add Header and then enter a key and value.

    5. Repeat this step to add multiple header keys and values.

  5. Click Add.

    6. If you want to add more webhook notifications, click Add Notification and then continue from step 2.
