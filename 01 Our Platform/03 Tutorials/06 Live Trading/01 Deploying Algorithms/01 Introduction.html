-Deploying multiple algorithms
1. Select Brokerage
2. Configure Brokerage Settings
3. Select Node
4. Initialize Cash State (doesn’t apply to all brokerages)
5. Initialize Holdings State (doesn’t apply to all brokerages)
6. Setup Notifications
     - Email
     - Webhook
     - SMS
     - Telegram
           - Provide the chat ID and supply a token for the custom bot.
7. Specify Automatic Restarts
8. Click Deploy
9. Verify
       - If you know your brokerage positions before you deploy, you can verify they have been loaded properly by using the QC GUI. Check your equity at the top, check the cashbook holdings, and position holdings
