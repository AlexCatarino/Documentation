Follow these steps after selecting Interactive Brokers as your brokerage in the tutorial in the introduction:

  1. Log in to Interactive Brokers in your browser.
  2. Click the person icon in the top-right corner and then click User Settings.
  3. In the Security section, click the gear icon next to Secure Login System.
  4. On the Secure Login System page, deselect all options or only select IB Key Security via IBKR Mobile.
  5. On the algorithm deployment page, enter your Interactive Brokers user name, ID, and password.
  6. If you have subscriptions to Interactive Brokers data feeds and want to use them, click the Data Provider dropdown and then click Interactive Brokers.
  7. Continue from step 5 in the tutorial in the introduction.
