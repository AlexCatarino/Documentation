Follow these steps to define parameter values for the optimization:

  1. Under the Parameters & Constraints tab of the optimization wizard, enter the name of the parameter to optimize.

    2. The parameter name must match the parameter names in the project panel.

  2. Enter the minimum and maximum parameter values.
  3. Click the gear icon next to the parameter and then enter a step size.
  4. If you want to add a second parameter to optimize, click Add Parameter and then continue from step 1.

    5. A maximum of 2 parameters can be optimized.
