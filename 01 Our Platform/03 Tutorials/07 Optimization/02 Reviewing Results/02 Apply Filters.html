Follow these steps to apply filters to the Backtests table:

  1. On the right edge of the Backtests table, click Filters.
  2. Click the column name you want the filter to be applied to.
  3. If the column you selected in step 2 is numerical, click the operation dropdown and select one of the following operations to apply:
    • Greater than or equals: The column values must be greater than or equal to the filter value for the backtest to be presented in the Backtests table.
    • Less than or equals: The column values must be less than or equal to the filter value for the backtest to be presented in the Backtests table.
    • In range: The column values must be within a range of values for the backtest to be presented in the Backtests table.
  4. Fill the fields below the operation dropdown you selected in the previous step.
