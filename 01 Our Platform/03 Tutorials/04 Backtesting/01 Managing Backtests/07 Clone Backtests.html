When you clone a backtest a new project is created with the backtest code files. To clone a backtest, follow these steps:

  1. In the top-right corner of the IDE, click Results.
  2. (Optional) If the backtest you want to rename isn't visible, use the pagination tools at the bottom of the screen to see older backtests.
  3. Hover over the backtest you want to clone and then click the clone icon that appears.
