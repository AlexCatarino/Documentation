If you have the Automatic Builds setting enabled, your projects will automatically build after each keystroke. Open the project that you want to build and then click Build to manually build the project. When you manually build a project, "Building" displays.

If the build process is unresponsive, try refreshing the page and building again. If the build process continues to be unresponsive, check our Status page or contact us. If the compiler finds errors during the build process, the IDE highlights the lines of code that caused the errors.
