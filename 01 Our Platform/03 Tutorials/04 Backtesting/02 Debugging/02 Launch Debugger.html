Follow these steps to launch the debugger:

  1. Open the project you want to debug.
  2. In your project's code files, click a line number to add a breakpoint.

    3. You need at least one breakpoint. You can only place breakpoints on non-empty lines.

  3. On the right edge of the IDE, click Debug.
  4. If you want to set watch variables, follow these steps:
    1. In the Debug panel, click the plus icon.
    2. Enter the name of the variable to watch and then click OK.
    3. If you want to add more watch variables, continue from step 1.
    4. If you want to update your watch variables, follow these steps:
      1. In the Debug panel, click the variable name you want to update.
      2. Enter the new variable name and then click OK.
    5. If you want to remove a watch variable, click X next to the watch variable you want to remove.
  5. Click Backtest to start debugging.
  6. Control the debugger with the following buttons in the Debug panel.
    • Continue: Execute until the next breakpoint.
    • Stop: Stop debugging and delete the backtest.
    • Step: Execute the next line of code.
