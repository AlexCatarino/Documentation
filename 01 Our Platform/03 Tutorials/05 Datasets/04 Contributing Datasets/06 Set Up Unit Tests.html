You must create a demonstration algorithm for your dataset before you set up unit tests.

Follow these steps to set up unit tests for your dataset:

  1. In the Lean.DataSource.vendorNameDatasetName/tests/MyCustomDataTypeTests.cs file, define the CreateNewInstance method to return an instance of your DataSource class.
  2. Run the following commands from your command line and then ensure all of the tests pass:
    3. $ dotnet build tests/Tests.csproj
$ dotnet test tests/bin/Debug/net5.0/Tests.dll
  3. Rename MyCustomDataTypeTests.cs to vendorNameDatasetNameTests.cs (for example, XYZAirlineTicketSalesTests.cs).
