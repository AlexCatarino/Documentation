You must create a demonstration algorithm for your dataset before you set up unit tests.

In the Lean.DataSource.vendorNameDatasetName/vendorNameDatasetNameTests.cs file, define the CreateNewInstance method to return an instance of your DataSource class and then execute the following commands to run the unit tests:

$ dotnet build tests/Tests.csproj
$ dotnet test tests/bin/Debug/net5.0/Tests.dll

All of the unit tests must pass before you start testing the processed data.
