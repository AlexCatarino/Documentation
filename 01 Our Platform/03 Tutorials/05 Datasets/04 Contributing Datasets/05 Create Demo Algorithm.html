You must process your dataset before you can create a demonstration algorithm that uses the dataset.

Follow these steps to create a demonstration algorithm for your dataset:

  1. Open the Lean.DataSource.vendorNameDatasetName/Demonstration.cs file.
  2. In the Demonstration.cs file, create a simple algorithm that demonstrates how to subscribe to your dataset and place trades.
  3. Rename the CustomDataAlgorithm class to vendorNameDatasetNameAlgorithm (for example, XYZAirlineTicketSalesAlgorithm).
  4. Rename the Demonstration.cs file to vendorNameDatasetNameAlgorithm.cs (for example, XYZAirlineTicketSalesAlgorithm.cs).
