Follow these steps to submit a new alpha to the Alpha Streams market:

  1. On the Strategies > List to Marketplace page, click Submit New Alpha.
  2. Click the Select a Project dropdown and then click the project you want to submit to the marketplace.
  3. Click the Select a Backtest dropdown and then click the name of the backtest you want to submit to the marketplace.

    4. The screening tool analyzes your backtest results to determine if it's suitable for the Alpha Streams market.

  4. If the Alpha Streams analysis tool determines your algorithm is not suitable for the marketplace, improve your algorithm and then continue from step 1.
  5. Enter a name for the alpha.
  6. Enter a description for the alpha.
  7. If you would like to set a custom reserve price, deselect the Accept default pricing check box and then enter a reserve price.
  8. Select an image for the alpha.
  9. Select the Yes, I'm live trading this strategy or the No, I'm not live trading this strategy check box and then click Next.
  10. Click Submit Alpha.
