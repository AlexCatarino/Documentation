Apply Filters

Follow these steps to apply filters to the table:

  1. On the right edge of the table, click Filters.
  2. Click the column name you want the filter to be applied to.
  3. If the column you selected in step 2 is alphabetical, enter the characters that must be included in the column for the alpha to be presented in the table.
  4. If the column you selected in step 2 is numerical, click the operation dropdown and select one of the following operations to apply:
    • Greater than or equals: The column values must be greater than or equal to the filter value for the alpha to be presented in the table.
    • Less than or equals: The column values must be less than or equal to the filter value for the alpha to be presented in the table.
    • In range: The column values must be within a range of values for the alpha to be presented in the table.
  5. If the column you selected in step 2 is numerical, fill the field(s) below the operation dropdown you selected in the previous step.

Sort Columns

Click one of the column names in the table to sort the table by that column.

Toggle Columns

Follow these steps to hide and show columns in the table:

  1. On the right edge of the table, click Columns.
  2. Select the columns you want to include in the table and deselect the columns you want to exclude.
