Follow these steps to change the LEAN engine branch that's used to run your backtests and live trading algorithms:

  1. Open the project in which you want to change the LEAN engine branch.
  2. In the project panel, click LEAN Engine (languageName): branchName (hash).
  3. Click the branch that's currently used and then select the new branch.
  4. If you want to always use the master branch, select the Always use Master Branch checkbox.
  5. Click Save.

    6. "Project Lean version saved" displays.

Changing the LEAN engine branch only affects the current project. If you create a new project, the new project will use the master branch by default.
