To add an algorithm parameter, follow these steps:

  1. Open the project.
  2. In the project panel, click Add New Parameter.
  3. Enter the parameter name and then click OK.

    4. If the parameter name already exists in your project, the default value of the parameter will be erased.

  4. (Optional) Add a default value for the parameter by following these steps:
    1. Hover over the algorithm parameter and then click the pencil icon that appears.
    2. Enter a default value for the parameter and then click OK
  5. Add the GetParameter method to your code files to inject the parameter value into your algorithm.
