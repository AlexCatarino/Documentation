The process to delete a project depends on if the project is open or not.

The Project Is Open

To delete the project, follow these steps:

  1. In the project panel, click Delete.
  2. Click OK.

The Project Is Not Open

To delete the project, follow these steps:

  1. On the Project page, click Open Project.
  2. If the project is in a directory, click the directory.

    3. Continue to traverse down the directory levels until the project file appears.

  3. Hover over the project file and then click the trash can that appears.
