The steps to delete a project depend on if the project is open or not.

The Project Is Open

Follow these steps to delete the project:

  1. In the project panel, click Delete.
  2. Click OK.

    3. "Project deleted" displays.

The Project Is Not Open

To delete the project, follow these steps:

  1. Log in to the Algorithm Lab.
  2. In the left navigation bar, click Projects.
  3. On the Projects page, click Open Project.
  4. If the project is in a directory, click down the directory levels until the project file displays.
  5. Hover over the project file and then click the trash can that appears.

    6. "Project deleted" displays.
