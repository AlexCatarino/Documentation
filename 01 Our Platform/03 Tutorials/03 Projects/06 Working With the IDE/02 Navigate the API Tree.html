Toggle

On the right side of the IDE, click API to open and close the API Tree.

Search

In the search bar at the top of the API tree panel, enter the name of a QCAlgorithm base class member to search the API tree.

Expand and Collapse Member Details

In the API tree panel, click on a member name in the tree to expand and collapse member details. If the member you select returns a class (for example, self.DateRulesDateRules), then the members of the class are added to the API tree.
