Toggle

To open and close the API Tree, on the right side of the IDE, click API.

Search

To search the API tree, in the search bar at the top of the API tree panel, enter the name of a class member. You can only search for members of the QCAlgorithm base class.

Expand and Collapse Member Details

To expand and collapse member details in the API tree, click on a member name in the tree. If the member you select returns a class (for example, self.DateRulesDateRules), then the members of the class will be added to the API tree.
