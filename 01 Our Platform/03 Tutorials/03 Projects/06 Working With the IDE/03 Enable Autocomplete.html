You need the Code Completion setting enabled on the Your Account page to use autocomplete.

Follow these steps to use autocomplete:

  1. Open the project in which you want to use autocomplete.
  2. Type the first few characters of a class member that you want to autocomplete (for example, self.SetSet or SimpleMovingAverage.Upda).
  3. Press CTRL+Space.

    4. If there are class members that match the characters you provided, a list of class members displays.

  4. Select the class member that you want to autocomplete.

    5. The rest of the class member name is automatically written in the code file.
