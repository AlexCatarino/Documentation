You need the Code Completion setting enabled on the Your Account page to use autocomplete.

Follow these steps to use autocomplete:

  1. Open the project you want to use autocomplete in.
  2. Type the first few characters of a class member you want to autocomplete (for example, self.SetSet or SimpleMovingAverage.Upda).
  3. Press CTRL+Space.

    4. If there are class members that match the characters you provided, a list of class members displays.

  4. Select the class member you want to autocomplete.
