To use autocomplete, on the Your Account page, the Code Completion setting must be enabled. If the setting is enabled, follow these steps:

  1. Type the first few characters of a QCAlgorithm class member you want to autocomplete (for example, self.SetSet).

    2. Only the members of the QCAlgorithm base class are supported.

  2. Press CTRL+Space.

    3. If there are class members that match the characters you provided, a list of QCAlgorithm class members will appear.

  3. Select the class member you want to autocomplete.
