The process to delete a library depends on if the library project is open or not.

The Library Project Is Open

To delete the library, follow these steps:

  1. In the project panel, click Delete.
  2. Click OK.

The Library Project Is Not Open

To delete the library, follow these steps:

  1. On the Project page, click Open Project.
  2. If the library is in a directory, click the directory.

    3. Continue to traverse down the directory levels until the library file appears.

  3. Hover over the library file and then click the trash can that appears.
