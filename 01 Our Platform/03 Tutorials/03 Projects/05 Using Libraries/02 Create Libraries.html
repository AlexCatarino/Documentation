Follow these steps to create a library:

  1. Create a new project.
  2. In the project panel, click Add New Library.
  3. Click Create a New Library.
  4. Write your library in the project files.
  5. If the root of your project files contains a main.py file, either:
    • Rename main.py.
    • Rename your project files so they are in a directory.

    When you import the library into a project, the library files are added to your project's root directory. If more than one file in the root directory of your project is named main.py, then your project won't build.
