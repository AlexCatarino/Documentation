Follow these steps to add notebook files to a project:

  1. Open the project.
  2. In the project panel, click Add New Notebook.
  3. Enter a notebook name and then click OK.

    4. "File created successfully" displays and the new file opens, opening the Research Environment.

You can only open one research notebook per project. If you have a notebook open, to open a different notebook in the same project, stop the research node that the first notebook is using and then open the new notebook.
