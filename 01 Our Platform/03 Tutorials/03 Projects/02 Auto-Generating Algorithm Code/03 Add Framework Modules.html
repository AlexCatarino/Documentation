You need the Strategy Builder enabled to use it to add framework modules.

Follow these steps to add modules with the Strategy Builder:

  1. Open a new project.
  2. On the Strategy Builder page, click Add Module.

    3. The modules catalog displays.

  3. (Optional) At the bottom of the page, click the pagination buttons to view other modules.
  4. (Optional) Click a module in the table to view more information about the module.
  5. Click Add next to the modules you want to add.
  6. Click Exit Strategy Builder.
  7. Click OK.

    8. The web IDE displays the project with the module codes added.
