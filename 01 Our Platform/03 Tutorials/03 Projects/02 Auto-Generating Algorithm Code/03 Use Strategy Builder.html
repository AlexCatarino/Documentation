You need the Strategy Builder enabled to use it to auto-generate your algorithm code.

Follow these steps to use the Strategy Builder:

  1. Open a new project.
  2. On the Strategy Builder page, click Add Module.

    3. The modules catalog displays.

  3. (Optional) At the bottom of the page, click the pagination buttons to view other modules.
  4. (Optional) Click a module in the table to view more information about the module.
  5. If you want to apply filters to the module catalog, follow these steps:
    1. Click Filters.
    2. Select the column name to which you want to apply the filter:
      • Name: To specify which characters need to be included in the module name to be included in the filtered results.
      • Category: To specify which type of modules should be included in the filtered results.
      • Author: To specify which characters need to be included in the author name to be included in the filtered results.
    3. If you selected the Name column in step 5.2, enter the characters you want to be included in the module name.

      4. The module catalog displays modules that contain the characters you entered.

    4. If you selected the Category column in step 5.2, select the check boxes next to the categories you want to be included in the results.

      5. The module catalog displays modules that are in the categories you selected.

    5. If you selected the Author column in step 5.2, enter the characters you want to be included in the author name.

      6. The module catalog displays modules that have author names containing the characters you entered.

  6. Click Add next to the modules you want to add.
  7. If you want to remove modules that you have added, click Remove next to the module.
  8. Click Exit Strategy Builder.
  9. Click OK.

    10. The web IDE displays the project with the module codes added.
