To apply module filters using the Strategy Builder, the Strategy Builder must be open. If the Strategy Builder is open, follow these steps:
  1. Click Filters.
  2. Select the column name you want the filter to be applied to:
    • Name: To specify which characters need to be included in the module name to be included in the filtered results.
    • Category: To specify which type of modules should be included in the filtered results.
    • Author: To specify which characters need to be included in the author name to be included in the filtered results.
  3. If you selected the Name column in step 2, enter characters you want to be included in the module name.
  4. If you selected the Category column in step 2, select the checkboxes next to the categories you want to be included in the results.
  5. If you selected the Author column in step 2, enter characters you want to be included in the module name.
