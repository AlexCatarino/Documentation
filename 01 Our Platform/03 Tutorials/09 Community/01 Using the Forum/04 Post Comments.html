You can only post comments on open forum threads.

Follow these steps to post a comment on a forum thread:

  1. At the bottom of the forum thread, enter your comment.
  2. (Optional) Follow these steps to attach a backtest:
    1. Click Attach Backtest.
    2. Click the Project dropdown and then select the project that contains the backtest that you want to attach.

      3. You can only attach projects that you own. If you don't own a project that you want to attach, clone the project and then attach the cloned version of the project.

    3. Click the Backtest dropdown and then select a backtest.

    When you attach a backtest, you're sharing all of the project files.

  3. Click Comment.
