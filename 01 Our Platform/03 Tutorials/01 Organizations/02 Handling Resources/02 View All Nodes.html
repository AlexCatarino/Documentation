Log in to the Algorithm Lab and then, in the left navigation bar, click Organization > Resources to view all of your organization's nodes.

The Resources page displays sections for your backtest, research, and live trading node clusters. Click the buttons in the top-right of the page to toggle between list and tile view. If the page is in list view, each section above includes a table with the following columns:

  • Name: Name of the node
  • Machine Type: The node model and specifications.
  • In Use By: The name of the member using the node.
  • Host: The live trading server name.
  • Assets: The maximum number of assets the node supports.
  • Actions: A list of possible actions.
