Follow these steps to view all of your organization's nodes:

  1. Log in to the Algorithm Lab.
  2. In the left navigation bar, click Organization > Resources.

    3. The Resources page displays.

The Resources page includes the following sections:

  1. Backtest Node Cluster: View all backtest nodes.
  2. Research Node Cluster: View all research nodes.
  3. Live Trading Node Cluster: View all live trading nodes.

Click the buttons in the top-right of the page to toggle between list and tile view. If the page is in list view, each section above includes a table with the following columns:

  1. Name: Name of the node
  2. Machine Type: The node model and specifications.
  3. In Use By: The name of the member using the node.
  4. Host: The live trading server name.
  5. Assets: The maximum number of assets the node supports.
  6. Actions: A list of possible actions.
