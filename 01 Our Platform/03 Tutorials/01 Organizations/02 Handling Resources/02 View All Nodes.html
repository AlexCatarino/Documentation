Log in to the Algorithm Lab and then, in the left navigation bar, click Organization > Resources to view all of your organization's nodes.

The Resources page displays sections for your backtesting, researching, and live trading node clusters. Click the buttons in the top-right of the page to toggle between list and tile view. If the page is in list view, each section above includes a table with the following columns:

  1. Name: Name of the node
  2. Machine Type: The node model and specifications.
  3. In Use By: The name of the member using the node.
  4. Host: The live trading server name.
  5. Assets: The maximum number of assets the node supports.
  6. Actions: A list of possible actions.
