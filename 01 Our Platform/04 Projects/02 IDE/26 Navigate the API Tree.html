Follow these steps to work with the API tree:

  1. Open a project.
  2. On the right side of the IDE, click API to open the API tree panel.
  3. In the search bar at the top of the API tree panel, enter the name of a QCAlgorithm base class member to search the API tree.
  4. Click on a member name in the API tree to expand and collapse the details of that member.

    5. If the member you select returns a class (for example, self.UniverseUniverse), then the members of the class are added to the API tree.
