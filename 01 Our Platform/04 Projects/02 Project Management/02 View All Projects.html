The My Projects page displays all of your QuantConnect projects in the organization you're connected as, including libraries and Boot Camp lessons. If you connect to a different organization, the list of projects changes.

The My Projects page shows the following information for each project:

  • Programming language
  • Name
  • The last date you modified the project

Click a project or directory on the page to open it.

Follow these steps to view the page:

  • Log in to the Algorithm Lab.
  • In the left navigation bar, click Projects.
  • Click Open Project.
