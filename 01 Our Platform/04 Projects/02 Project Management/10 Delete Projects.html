You can delete a project when it is open or closed.

Delete Open Projects

In the Project panel, click Delete, and then click OK to delete the project. "Project deleted" displays.

Delete Closed Projects

Follow these steps to delete the project:

  1. Log in to the Algorithm Lab.
  2. In the left navigation bar, click Projects.
  3. On the Projects page, click Open Project.
  4. If the project is in a directory, click the directory files to navigate to the project file.
  5. Hover over the project file and then click the trash can that appears.

    6. "Project deleted" displays.
