You need the Strategy Builder enabled to use it to auto-generate your algorithm code. To enable the Strategy Builder, on the Your Account page, in the Account Settings section, enable the Use Strategy Builder by Default switch.

Follow these steps to use the Strategy Builder to auto-generate your algorithm code:

  1. In the left navigation bar, click Projects.
  2. In the project panel, click Create New Algorithm.
  3. On the Strategy Builder page, click Add Module.

    4. The module catalog displays.

  4. Click Add on a module from each category.

    5. The categories are Universe, Alpha, Portfolio, Risk, and Execution. Use the pagination tools in the bottom-right corner to view all of the modules.

  5. Click Exit Strategy Builder.
  6. Click OK.

    7. The web IDE displays the project with the module codes added.
