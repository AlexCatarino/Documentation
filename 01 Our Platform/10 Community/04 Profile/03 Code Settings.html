Your profile's code settings control how some parts of the IDE operate. There are settings for the Strategy Builder, automatic builds, code completion, parameter highlighting, and meta-data sharing. You can edit your code settings from Your Account page. If you adjust your settings, they are saved to your profile so you don't have to set them every time that you log in from a new device.

Strategy Builder

The Strategy Builder setting lets you specify if the Strategy Builder should launch when you create new projects. If you are new to QuantConnect and want an easy way to generate algorithm code without writing any, enable the setting. If you don't use the Strategy Builder, disable the setting so that you can create new projects faster.

Automatic Builds

The Automatic Builds setting lets you specify if the IDE should build your project after each keystroke. To be notified immediately after you've made a coding error, enable the setting. If you don't want to see the red underlining and pop-ups in the IDE before you've finished writing an entire line of code, disable the setting.

Code Completion

The Code Completion setting lets you specify if autocomplete and Intellisense should be enabled in the IDE. To reduce the number of times that you need to use the API reference or API tree, enable the setting. If you find the autocomplete pop-ups distracting, disable the setting.

Parameter Highlighting

The Parameter Highlighting setting lets you specify if the IDE should mark hard-coded parameter values in your algorithms. If you want the IDE to show you where your algorithm may be overfit and where your algorithm can be made more general by removing parameters, enable the setting. If you find the parameter highlighting distracting, disable the setting.

Meta-Data Sharing

The Meta-Data Sharing setting lets you specify if your backtest data can be shared with Alpha Streams investors. The data that's shared is the performance statistics, significant holdings, and equity curves of your backtests. This information is just enough data for the funds to determine if they are interested in seeing your algorithm tested live in Alpha Streams. If you share the meta-data, your source code is not shared, you still own the intellectual property of your algorithm and you remain anonymous to the investors. If investors like the performance of your backtest, we may contact you for a licensing opportunity before you submit the algorithm to Alpha Streams. To share your meta-data with Alpha Streams investors, enable the setting. If you don't want to share your meta-data, disable the setting.

Edit Your Code Settings

Follow these steps to enable and disable your account's code settings:

  1. Log in to your account.
  2. In the top navigation bar, click yourUsername > My Account.
  3. On the Your Account page, in the Account Settings section, click the switches to enable and disable the following code settings:
    • Use Strategy Builder by Default
    • Automatic Builds
    • Code Completion
    • Parameter Highlighting
    • Attract Licensing Opportunities with Research Meta-Data Sharing

    "Updated!" displays.
