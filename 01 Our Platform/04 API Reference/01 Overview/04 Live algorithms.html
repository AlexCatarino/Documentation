What is live algorithm management with the API:
- The API gives you the ability to:
    - Create a live algorithm
    - Get a list of your live running algorithms
    - Read out a live algorithm (results)
           - Charts (equity curve)
           - Live performance statistics
           - Holdings
           - CashBook
           - Orders & Order events
           - Server statistics
    - Liquidate a live algorithm
    - Stop a live algorithm
    - Gets the logs of a specific live algorithm

Why retrieve live algorithm information:
-The algorithm results from the API method include more statistics than just that shown in the backtest results page in the GUI
-Download orders for deeper analysis
-Use equity curve charts to compute correlation with your other live algorithms
