The project/create API returns codes 200 on success, and 401 on authentication error.

200 Success

ProjectListResponse Model - Project list response.
projects Project Array
List of projects for the authenticated user.
success Boolean
Indicate if the API request was successful.
errors String Array
List of errors with the API call.
Example 
{
  "projects": [
    {
      "projectId": 0,
      "name": "string",
      "created": "2021-11-24T01:22:19.740Z",
      "modified": "2021-11-24T01:22:19.740Z",
      "language": "C#"
    }
  ],
  "success": true,
  "errors": [
    "string"
  ]
}
Project Model - Response from reading a project by id.
projectId integer
Project id.
name String
Name of the project.
created string(date-time)
Datetime the project was created.
modified string(date-time)
Datetime the project was modified.
language String Enum
Programming language of the project. Options: [C#, Py].
Example 
{
  "projects": [
    {
      "projectId": 0,
      "name": "string",
      "created": "2021-11-24T01:22:19.740Z",
      "modified": "2021-11-24T01:22:19.740Z",
      "language": "C#"
    }
  ],
  "success": true,
  "errors": [
    "string"
  ]
}
