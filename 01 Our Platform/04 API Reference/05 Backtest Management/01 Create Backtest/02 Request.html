The /backtests/create API accepts requests in the following format:

CreateBacktestRequest Model - Request to create a new backtest.
projectId integer
Project Id we sent for compile.
compileId string
Compile Id for the project to backtest.
backtestName string
Name for the new backtest.
Example 
{
  "projectId": 0,
  "compileId": "string",
  "backtestName": "string"
}
