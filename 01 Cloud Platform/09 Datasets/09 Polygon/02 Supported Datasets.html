The Polygon data provider provides the security price data directly from Polygon. Our integration supports securities from the following asset classes:

  • US Equity
  • US Equity Options
  • US Indices
  • US Index Options

If you use the Polygon data provider, Polygon only provides the security price data. We provide the following auxiliary datasets from the Dataset Market:

  • US Equity Security Master
  • Universe selection datasets
  • Non-streaming alternative datasets

For more information about the data source, see the Polygon API documentation.
