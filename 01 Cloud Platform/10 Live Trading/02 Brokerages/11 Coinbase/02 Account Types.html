Coinbase supports cash accounts. To set the account type in an algorithm, see the Coinbase brokerage model documentation.

Create an Account

Follow the Create a Coinbase account tutorial on the Coinbase website to create an account.

You will need API credentials to deploy live algorithms. After you have an account, open the API Key Management page on the Coinbase website and create API credentials. As you create the credentials, enable the View (read-only) permission check box and enable the Trade (execute trades on your behalf) permission check box. Paper Trading

The Coinbase brokerage doesn't support paper trading, but you can follow these steps to simulate it with QuantConnect:

  1. In the Initializeinitialize method of your algorithm, set the Coinbase brokerage model and your account type.
  2. Deploy your algorithm with the QuantConnect Paper Trading brokerage.
