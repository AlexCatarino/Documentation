Coinbase supports cash accounts. To set the account type in an algorithm, see the Coinbase brokerage model documentation.

Create an Account

Follow the Create a Coinbase account tutorial on the Coinbase website to create an account.

You will need API credentials to deploy live algorithms. After you have an account, follow these steps to create API credentials:

  1. Navigate to your API settings.
  2. Click the Create API key button.
  3. In the popup dialog configure:
    • API key nickname
    • Portfolio
    • Permissions: enable View and Trade permissions.
    • IP whitelist: enter our IP address, 146.59.85.21.
  4. Click Create & Download.
  5. Copy and secure your private/public key pair in a safe location.
  6. Click I've saved my key.

For more information about Coinbase Advanced Trade APIs, see Getting Started.

Paper Trading

The Coinbase brokerage doesn't support paper trading, but you can follow these steps to simulate it:

  1. In the Initializeinitialize method of your algorithm, set the Coinbase brokerage model and your account type.
  2. Deploy your algorithm with the QuantConnect Paper Trading brokerage.
