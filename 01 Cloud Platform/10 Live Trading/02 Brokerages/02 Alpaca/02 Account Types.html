Alpaca supports cash and margin accounts. To set the account type in an algorithm, see the Alpaca brokerage model documentation.

Create an Account

Follow the account creation wizard on the Alpaca website to create a Alpaca account.

You will API Key and Secret to deploy live algorithms. After you have an account, get your API Key and Secret from the Dashboard page on the Alpaca website.

Paper Trading

Alpaca supports paper trading, see Alpaca documentation for paper trading rules and assumptions.

Switch to your paper account dashboard, and get your API Key and Secret.

If you trade Equities or Crypto, you can use the QuantConnect data provider to get real-time data. If you trade Equity Options, you must use data from the Alpaca data provider. If you trade Equities and Equity Options, use both data providers.
