网页编程环境附带一些辅助编码的键盘快捷键。

快捷键 功能描述
Ctrl + Space 启动全自动完成
Ctrl + / 切换所选代码的评论
Ctrl + Up/Down 增加/减少字体大小
Ctrl + F 搜索
Ctrl + H 替换
Ctrl + S 保存文档并编译项目
Ctrl + D 删除行
Ctrl + Z 撤销
Ctrl + Y (or Ctrl + Shift + Z) 重做
Alt + Up/Down 向上/向下移动行
Alt + Shift + Up/Down 上下复制行
Alt + 0 (zero) 折叠所有代码块
Alt + Shift + 0 (zero) 展开所有代码块

完整的快捷键清单可以在ACE Editor开源项目中找到。
