Using parameters in an algorithm is unavoidable but an algorithm trends toward being overfitted as more parameters get added or fine-tuned. Optimizing parameters can be done well by using robust methodology, and the parameter counter shown in the backtest results page is just a general guide to inform you of how many parameters are present in the algorithm, reminding users that a good algorithm has more to it than just good backtest results.
