Symbols have several public properties you can use to uniquely identify the asset. The when serialized together this class allows unique identification of millions of different security objects, all within a self contained reference. 

Symbol.Market            # Market USA, Fxcm, Oanda. Liquidity venues.
Symbol.SecurityType      # Security Types include Equity, Option, Future, Forex, Crypto and Cfd
Symbol.OptionType        # American or European Option.
Symbol.OptionRight       # OptionRight indicates if a Put or Call
Symbol.Date              # Earliest listing date if equities, expiry for future/option.
