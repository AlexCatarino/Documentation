QuantConnect provides the following API methods to place orders. All of the methods return an $[OrderTicket, T:QuantConnect.Orders.OrderTicket], which you can use to update and cancel the orders. All orders should use the symbol object to place orders; the Symbol class uniquely identifies the security through ticker renames. Some order types are not supported by all brokerages. To confirm each order's support, click on the respective links below.
