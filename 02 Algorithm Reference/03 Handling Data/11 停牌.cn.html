$[Delisting,T:QuantConnect.Data.Market.Delisting]事件提供资产不再在交易所交易的通知。在股票停牌事件的最后一个交易日会发出停牌警告，为您的算法提供时间，以便在强制终止前顺利平仓。 $[Delisting,T:QuantConnect.Data.Market.Delisting] events provide notice that an asset is no longer trading on the exchange. A delisting warning is issued the final trading day for a stock delisting event to give you algorithm time to gracefully exit out of positions before forced termination. 

def Initialize(self):
    self.SetStartDate(2007, 5, 16)
    self.SetEndDate(2007, 5, 25)
    self.SetCash(100000);
    equity = self.AddEquity("AAA", Resolution.Daily)

def OnData(self, data):
    self.MarketOrder('AAA', 10)
    
    ## 打印停牌警告和通知
    if data.Delistings.ContainsKey('AAA'):
        delisting = data.Delistings['AAA']
            
        ## 记录停牌警告类型
        self.Log(delisting.ToString())
            
        if delisting.Type == 0:
            self.Log('AAA will be delisted EOD')
        if delisting.Type == 1:
            self.Log('AAA delisted')
