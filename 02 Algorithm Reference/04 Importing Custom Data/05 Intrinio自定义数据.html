Intrino是一个跟Quandl类似的第三方平台，能够提供付费数据。它们提供一个庞大的金融数据库，可能对您的算法很有用。 为了帮助您在算法中使用Intrino数据，我们创建了一个IntrinioEconomicData的实现方式，该实现方式允许从美联储经济数据（FRED）访问其经济数据库。

我们在IntrinioEconomicDataSources类 上已经构建了许多被请求的symbol对象的辅助方法。但如果您愿意，您可以在此处获得经济数据系列的完整列表。 

// 在Initialize方法中:
 AddData<IntrinioEconomicData>(IntrinioEconomicDataSources.Commodities.CrudeOilWTI, Resolution.Daily);
	
# 在Initialize方法中:
self.AddData(IntrinioEconomicData, "$DCOILWTICO", Resolution.Daily)
