在初始化阶段，您的算法需要使用AddData<T>(string ticker, Resolution resolution = Resolution.Daily)。这为LEAN提供了T类型“工厂”来创建对象，数据名称，以及接受数据和检查更新的分辨率。 self.AddData(Type class, string ticker, Resolution resolution = Resolution.Daily)。 这为LEAN提供了T类型“工厂”来创建对象，数据名称，以及接受数据和检查更新的分辨率。

该框架根据Resolution（分辨率）周期来检测新数据；即Resolution.Tick会持续地接收数据， Resolution.Second每秒接收一次数据，Resolution.Minute则是按每分钟。Hourly和Daily分辨率数据则是每三十分钟接收一次。 

// 在Initialize方法中:
AddData<Weather>("KNYC", Resolution.Daily);

# 在Initialize方法中:
self.AddData(Weather, "KNYC", Resolution.Minute)
