您可能希望引入一个静态的文件作为参考，例如加载一个日数据的对象（symbol）列表，或者是一个训练AI的文件。 通过$[Download(),M:QuantConnect.Algorithm.QCAlgorithm.Download]方法，这是可能的。 方法 Download下载一个URL同时将它返回成一个字符串(string)。 它可以采用标题设置进行身份验证，并通过用户名和密码进行基本身份验证。 

// If using dropbox remember to add the &dl=1 to trigger a download
var file = Download("https://www.dropbox.com?....&dl=1");

# If using dropbox remember to add the &dl=1 to trigger a download
file = self.Download("https://www.dropbox.com?....&dl=1")

这是从公共Dropbox文件下载数据的常见请求。 在这种情况下，您应确保下载直接文件链接 —— 而不是下载的HTML页面。 您可以通过将&dl=1添加到Dropbox下载URL的末尾来指定。
