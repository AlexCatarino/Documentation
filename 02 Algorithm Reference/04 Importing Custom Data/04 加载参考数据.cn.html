您可能希望导入一个静态的文件作为参考，例如加载一个每日更新的对象（symbol）列表，或者是一个训练AI的文件。这可以通过$[Download(),M:QuantConnect.Algorithm.QCAlgorithm.Download]方法来实现。 Download方法下载一个URL同时将它返回成一个字符串(string)。它可以采用标题设置进行身份验证，并通过用户名和密码进行基本身份验证。 

// 如果使用dropbox,记得加上&dl=1以促发下载
var file = Download("https://www.dropbox.com?....&dl=1");

# 如果使用dropbox,记得加上&dl=1以促发下载
file = self.Download("https://www.dropbox.com?....&dl=1")

从公共Dropbox文件下载数据是很常见的请求。在这种情况下，您应确保下载直接的文件链接 —— 而不是下载的HTML页面。您可以通过将&dl=1添加到Dropbox下载URL的末尾来指定。
