The IntrinioEconomicData implementation gives access to the massive repository of economic data from the Federal Reserve Economic Data system via the Intrinio API.

Some of the most requested symbols are available through the IntrinioEconomicDataSources class.

The full list of economic data series is available here. 

// In Initialize method:
 AddData(IntrinioEconomicDataSources.Commodities.CrudeOilWTI, Resolution.Daily);

# In Initialize method:
self.AddData(IntrinioEconomicData, "$DCOILWTICO", Resolution.Daily)
