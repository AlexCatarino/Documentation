您需要创建一个自定义类型来指导LEAN从何获取您的数据，以及怎么读取它。我们支持不同的数据类型与格式。您甚至可以改变回测和实盘交易的源位置。所有的数据必须拓展自$[BaseData,T:QuantConnect.Data.BaseData]同时覆盖原有的$[Reader,M:QuantConnect.Data.BaseData.Reader]和$[GetSource,M:QuantConnect.Data.BaseData.Reader]方法。

$[GetSource,M:QuantConnect.Data.BaseData.Reader]指导LEAN从哪里找到您的数据。它必定会返回一个$[SubscriptionDataSource,T:QuantConnect.Data.SubscriptionDataSource]对象。这个对象包括能找到您数据的URL，以及数据的格式(SubscriptionTransportMedium RemoteFile或者Rest)。当返回的数据源改变URL后，数据会被重新下载。这允许LEAN缓存大文件并只会在请求的时候下载新数据。这同样使您能够将大量的日内数据分解成小的日数据文件，以加速回测。

当使用$[SubscriptionTransportMedium.Rest,T:QuantConnect.SubscriptionTransportMedium]时，提供的url会在每个分辨率(Resolution)时段来读取，同时会被假定对一个数据点而言是充足的。这一般用于实时数据源。

$[Reader,M:QuantConnect.Data.BaseData.Reader]获取信息源提供的一行数据，并将其解析为您的一个自定义对象（例如在代码片段中的Weather）。除了设置自定义类型属性外，还应注意设置三个必需的属性：

  • Symbol（对象） - 应总是设定为config.Symbol
  • Time（时间） - 需要与自定义数据同步
  • Value（价值） - 是买卖以及计算投资组合所必需的

当在一行中没有可用的数据，您应得到空值。 

public class Weather : BaseData
{
    public decimal MaxC = 0;
    public decimal MinC = 0;
    public string errString = "";

    public override SubscriptionDataSource GetSource(
        SubscriptionDataConfig config,
        DateTime date,
        bool isLive)
    {
        var source = string.Format(
                "https://www.wunderground.com/history/airport/{0}/{1}/1/1/CustomHistory.html?dayend=31&monthend=12&yearend={1}&format=1",
                config.Symbol, date.Year);

          return new SubscriptionDataSource(source,
              SubscriptionTransportMedium.RemoteFile);
    }

    public override BaseData Reader(
        SubscriptionDataConfig config,
        string line,
        DateTime date,
        bool isLive)
    {
        if (string.IsNullOrWhiteSpace(line) ||
            char.IsLetter(line[0]))
            return null;

        var data = line.Split(',');

        return new Weather()
        {
            // 确保我们只会在交易日之后得到该数据 —— 不希望得到前瞻偏差
            Time = DateTime.Parse(data[0]).AddHours(20),
            Symbol = config.Symbol,
            MaxC = Convert.ToDecimal(data[1]),
            Value = Convert.ToDecimal(data[2]),
            MinC = Convert.ToDecimal(data[3]),
        };
    }
}

class Weather(PythonData):
    ''' Weather based rebalancing'''

    def GetSource(self, config, date, isLive):
        source = "https://www.wunderground.com/history/airport/{0}/{1}/1/1/CustomHistory.html?dayend=31&monthend=12&yearend={1}&format=1".format(config.Symbol, date.year);
        return SubscriptionDataSource(source, SubscriptionTransportMedium.RemoteFile);


    def Reader(self, config, line, date, isLive):
        # 如果第一个字母不是数字，则跳过
        if not (line.strip() and line[0].isdigit()): return None

        data = line.split(',')
        weather = Weather()
        weather.Symbol = config.Symbol
        weather.Time = datetime.strptime(data[0], '%Y-%m-%d') + timedelta(hours=20) # 确保我们只会在交易日之后得到该数据 —— 不希望得到前瞻偏差
        weather.Value = decimal.Decimal(data[2])
        weather["MaxC"] = float(data[1])
        weather["MinC"] = float(data[3])

        return weather
