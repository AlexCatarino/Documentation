购买力模型（也称为保证金模型）控制您的算法进行交易时的购买力（杠杆）。购买力的计算可以非常复杂，取决于许多因素，包括经纪商，甚至是一天中的时间。

购买力模型在$[IBuyingPowerModel, T:QuantConnect.Securities.IBuyingPowerModel]界面实现，并默认为$[BuyingPowerModel, T:QuantConnect.Securities.BuyingPowerModel]类。如果您想自定义购买力模型，可以在基类上覆盖您希望更改的方法。

我们还提供$[PatternDayTradingMarginModel，T：QuantConnect.Securities.PatternDayTradingMarginModel]（典型日内交易保证金模型）来模拟美国股票的日内模式日交易，其提供4倍日内杠杆和2倍隔夜杠杆。 

// 将指定股票设定为PDT(典型日内交易）保证金模型的示例：
// 您通常无需调整保证金模型
Securities["AAPL"].MarginModel = new PatternDayTradingMarginModel();

# 将指定股票设定为PDT(典型日内交易）保证金模型的示例：
# 您通常无需调整保证金模型
self.Securities["AAPL"].MarginModel =  PatternDayTradingMarginModel()

您可通过在Portfolio Level设定模型为$[Null,F:QuantConnect.Securities.MarginCallModel.Null]来禁用追加保证金通知模型。 

// 在初始化（Initialize）方法中
Portfolio.MarginCallModel = MarginCallModel.Null;

# 在初始化（Initialize）方法中
self.Portfolio.MarginCallModel = MarginCallModel.Null
