回测使用SetStartDate(int year, int month, int day) self.SetStartDate(int year, int month, int day) this.SetStartDate(int year, int month, int day) and SetEndDate(int year, int month, int day) self.SetEndDate(int year, int month, int day) this.SetEndDate(int year, int month, int day)方法来设置回测的时间范围。如果结束日期未被指定，默认值为昨天。 在.NET语言中您也可以使用一个DateTime对象来设置日期。 

SetStartDate(2013, 1, 1);         // 设置开始日期为指定日期
SetEndDate(2015, 1, 1);         // 设置结束日期为指定日期
SetEndDate(DateTime.Now.Date.AddDays(-1)); // 或者一个相对日期

self.SetStartDate(2013,1,1)
self.SetEndDate(2015,1,1)
self.SetEndDate(datetime.now() - timedelta(1)) # 或者使用一个相对日期

	
this.SetStartDate(2013, 10, 7)
this.SetEndDate(2013, 10, 11)
