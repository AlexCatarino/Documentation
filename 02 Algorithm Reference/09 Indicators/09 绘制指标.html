我们提供一个辅助方法来使绘制指标变得简单。有关绘制API的更多信息，请查阅绘制部分。 

Plot(string chart, Indicator[] indicators)
self.Plot(string chart, Indicator[] indicators)

请注意，并非所有指标都拥有相同的基本类型（T），因此可能无法被一同绘制，例如一些指标需要成交K线而另一些指标需要数据点。 

//绘制指标对象数组，基本类型为“Indicator”。
Plot("Indicators", sma, rsi); 

//绘制指标对象数组，基本类型为"TradeBarIndicator"。
Plot("Indicators", atr, aroon); 

//对于复杂的绘图，最简单的方法是单独绘制每一个指标。

# 绘制指标对象数组，基本类型为“Indicator”。
self.Plot("Indicators", sma, rsi); 

# 绘制指标对象数组，基本类型为"TradeBarIndicator"。
self.Plot("Indicators", atr, aroon); 

# 目前在每一次Plot方法的调用中我们最多可以绘制4个指标
# 对于复杂的绘图，最简单的方法是单独绘制每一个指标。
