我们为您提供超过100个技术指标。您可以通过两种方式使用它们：通过快捷创建方法，或者作为类对象。通过快捷创建方法创建的指标将被预先准备好接收数据。您可以在下方的参考列表中寻找我们支持的指标以及它们的属性。

$[Identity,T:QuantConnect.Indicators.Identity]是一个值得学习的关键指标。它仅仅返回资产的价格。在与其他指标组合使用时它非常有用。例如： 

var pep = Identity("PEP");    // Pepsi标签

var coke = Identity("KO");    // Coke标签
var delta = pep.Minus(coke);  // 它们之间的差值
