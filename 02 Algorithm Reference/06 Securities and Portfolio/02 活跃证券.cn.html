您可以通过ActiveSecurities（活跃证券）选择资产集(universe)中的当前资产。这对于从资产集(universe)中选择证券非常有用。它具有和Securities对象相同的属性。 

// 当前资产集中的证券:
ActiveSecurities["IBM"].HasData           // Security是否有数据
                 .Invested          // 是否持有
                 .LocalTime         // 交易所当地时间
                 .Holdings          // Portfolio对象
                 .Exchange          // 交易所
                 .FeeModel;         // 费用模型

# Securities currently in the universe:
self.ActiveSecurities["IBM"].HasData           # Security是否有数据
                 .Invested          # 是否持有
                 .LocalTime         # 交易所当地时间
                 .Holdings          # Portfolio对象
                 .Exchange          # 交易所
                 .FeeModel;         # 费用模型
