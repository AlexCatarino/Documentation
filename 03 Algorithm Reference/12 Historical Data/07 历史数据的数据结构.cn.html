QuantConnect平台托管一组特定数据，因此历史数据仅限于几种特定格式。有关证券类型的数据格式指南，请参阅下表：

证券类型 支持的数据类型
股票 成交K线(TradeBar)
Tick，秒，分钟，小时，每日
外汇，差价合约(CFD) 报价K线(QuoteBar)
Tick，秒，分钟，小时，每日
加密货币 成交K线(TradeBar)：默认值，报价K线(QuoteBar):可选
Tick，秒，分钟，小时，每日
期货合约 成交K线(TradeBar)：默认值，报价K线(QuoteBar):可选
Tick，秒，分钟
期权合约 成交K线(TradeBar)：默认值，报价K线(QuoteBar):可选
分钟

默认情况下会在成交K线(TradeBars)中返回历史数据，但对于期货，加密货币和期权报价K线(QuoteBars)也可用。要以报价K线(QuoteBars)请求数据，您必须在请求时指定数据类型。 

// 获取BTCUSD代号并使用它来请求历史数据
var btcusd = AddCrypto("BTCUSD", Resolution.Daily, Market.GDAX).Symbol;
var quotes = History<QuoteBar>(btcusd, 14, Resolution.Daily)
