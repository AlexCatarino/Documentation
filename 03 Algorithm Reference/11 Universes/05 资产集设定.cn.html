资产集是根据默认设置对象创建的。您可以配置这些设置对象来创建自定义的资产集。下面是UniverseSettings对象及其默认设置： 

    //热门的资产集设置:
    UniverseSettings.Resolution      // 添加的资产应当使用哪种分辨率
                    .Leverage        // 在资产集中资产应当使用多大的杠杆？
                    .Fillforward     // 资产数据是否向前填充？
                    .MinimumTimeInUniverse // 资产在资产集中的最小出现次数
                    .ExtendedMarketHours  // 资产是否应当提取盘后交易数据

    //热门的资产集设置:
    self.UniverseSettings.Resolution      // 添加的资产应当使用哪种分辨率
                    .Leverage        // 在资产集中资产应当使用多大的杠杆？
                    .Fillforward     // 资产数据是否向前填充？
                    .MinimumTimeInUniverse // 资产在资产集中的最小出现次数
                    .ExtendedMarketHours  // 资产是否应当提取盘后交易数据

在请求资产集之前，应该在Initialize()方法中配置这些设定。它们是全局变量，因此将适用于您创建的所有资产集。 

public class MyCustomSettingsUniverseAlgorithm : QCAlgorithm {
    public override void Initialize()
    {
        UniverseSettings.Resolution = Resolution.Second;
        AddUniverse(MySecondResolutionCoarseFilterFunction);
    }
}

class MyCustomSettingsUniverseAlgorithm(QCAlgorithm):
     def Initialize(self):
         # 请求秒级别分辨率的数据。这将会相当缓慢！
         self.UniverseSettings.Resolution = Resolution.Second
         self.AddUniverse(self.MySecondResolutionCoarseFilterFunction)
