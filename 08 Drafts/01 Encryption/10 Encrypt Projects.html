To encrypt a local project, run lean encrypt "<projectName>" --key "<path>".

$ lean encrypt "My Project" --key "C:\Users\john\qc-encryption-key.txt"
Local files encrypted successfully with key C:\Users\john\qc-encryption-key.txt

If you have an unencrypted copy of the project in QuantConnect Cloud, the cloud version remains unencrypted after you run the preceding command. To encrypt both the local and cloud version, add your encryption key to your browser and then push the project changes.

$ lean cloud push --project "My Project"
[1/1] Pushing 'My Project'
Successfully updated name, files, libraries, and encryption_key for 'My Project'
