To run a local backtest with Polygon data, open a terminal in your organization workspace and then run lean backtest <projectName> --data-provider Polygon --polygon-api-key <apiKey>.

$ lean backtest "My Project" --data-provider Polygon --polygon-api-key apiKey

The lean backtest command also accepts additional options for Polygon. If you provide any of the preceding options, your Lean configuration file saves them so that you only need to run lean backtest <projectName> to run another backtest with the same options.
