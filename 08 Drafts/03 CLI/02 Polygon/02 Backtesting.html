To run a backtest with Polygon data, open a terminal in the organization workspace that contains the project and then run lean backtest <projectName> --data-provider Polygon --polygon-api-key <apiKame>.

$ lean backtest "My Project" --data-provider Polygon --polygon-api-key dTB0asdfasdfasdf
Successfully ran 'My Project' in the 'backtesting' environment and stored the output in 'My Project\backtests\2024-01-26_15-29-12'

To get an API key, create a Polygon account and then see your API Keys page on the Polygon website.
