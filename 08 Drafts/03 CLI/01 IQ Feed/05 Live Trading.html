To deploy a local live algorithm that uses IQFeed as the data provider, open a terminal in your organization workspace and then run lean live deploy <projectName> --data-provider IQFeed --iqfeed-username <username> --iqfeed-password <password> --brokerage <brokerageName> <requiredBrokerageOptions>.

$ lean live deploy "My Project" --data-provider IQFeed --iqfeed-username username --iqfeed-password password --brokerage "Paper Trading"

The lean live deploy command also accepts additional options so that you can select IQFeed and run the command in non-interactive mode. If you provide any of the preceding options, your Lean configuration file saves them so that you only need to run lean live deploy <projectName> --brokerage <brokerageName> to deploy another live algorithm with the same options.
