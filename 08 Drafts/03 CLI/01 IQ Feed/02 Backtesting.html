To run a backtest with IQFeed data, open a terminal in your organization workspace and then run lean backtest <projectName> --data-provider IQFeed --iqfeed-username <username> --iqfeed-password <password>.

$ lean backtest "My Project" --data-provider IQFeed --iqfeed-username username --iqfeed-password password

The lean backtest command also accepts additional options for IQ Feed. If you provide any of the preceding options, your Lean configuration file saves them so that you only need to run lean backtest <projectName> to run another backtest with the same options.
