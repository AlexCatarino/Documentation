To access IQFeed data from the Research Environment, open a terminal in your organization workspace and then run lean research <projectName> --data-provider IQFeed --iqfeed-username <username> --iqfeed-password <password>.

$ lean research "My Project" --data-provider IQFeed --iqfeed-username username --iqfeed-password password

The lean research command also accepts additional options for IQFeed. If you provide any of the preceding options, your Lean configuration file saves them so that you only need to run lean research <projectName> to run another backtest with the same options.
