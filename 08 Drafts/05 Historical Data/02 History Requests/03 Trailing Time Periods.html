To get historical data for a trailing time period, pass a timedeltaTimeSpan to the historyHistory method. If there is no data in the time period, the result is empty. 

// Get the minute-resolution TradeBar data of an asset over the trailing 3 days.
var history = History<TradeBar>(symbol, TimeSpan.FromDays(3), Resolution.Minute);
# Get the minute-resolution TradeBar data of an asset over the trailing 3 days.
history = self.history(TradeBar, symbol, timedelta(3), Resolution.MINUTE)
