  1. Create a new project.

    2. The process to create a new project depends on if you use the Cloud Platform, Local Platform, or CLI.

  2. In the initializeInitialize method, add an asset.
    3. var spy = AddEquity("SPY");
    spy = self.add_equity('SPY')
  3. Call the historyHistory method with the asset's Symbol and a lookback period.
    4. var history = History(spy.Symbol, 10, Resolution.Daily);
