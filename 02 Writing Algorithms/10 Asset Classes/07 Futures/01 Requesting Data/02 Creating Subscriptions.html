What:
- Create a security subscription

Why:
-To add a data feed for the security to your algorithm
   - To enable trading / fills / quotes
   - To receive data updates in your algorithm + portfolio
   - To make trading decisions
-This adds the security to your "user-defined"

How:
- AddFuture() + SetFilter()
   - Save a reference to the canonical Symbol (the continuous future Symbol)
   - Usually called in OnData
- For more information on universes, see Universes > Futures.
-To remove a subscription, use RemoveSecurity()
