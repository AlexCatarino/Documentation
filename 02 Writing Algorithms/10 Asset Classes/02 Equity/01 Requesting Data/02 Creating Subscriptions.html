What:
- Create a security subscription

Why:
-To add a data feed for the security to your algorithm
   - To enable trading / fills / quotes
   - To receive data updates in your algorithm + portfolio
   - To make trading decisions
-This adds the security to your "user-defined" / static universe

How:
- AddEquity()
   - Save a reference to the Symbol
   - Usually called in OnData
- Alternatively, define an Equity universe or use an universe selection model. These types of universes can be dynamic.
