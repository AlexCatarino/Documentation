What is custom data?
- Your own external data; data that's not included in the Dataset Market.

What is possible?
- Can read the external data source to inject the data into your algorithm
- Use the data to make trading decisions
- Use the data to simulate trades on unsupported securities

What is required (custom type, implementing interface)
- You need to define a custom data type so that you can read the data source and convert it into a form that LEAN is familar with + can use.
