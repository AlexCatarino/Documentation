What is custom data?
- Your own external data; data that's not included in the Dataset Market.
- You can either bulk download a single static file or read a file line-by-line with a custom data reader

What is possible?
- Can read the external data source to inject the data into your algorithm
- Use the data to make trading decisions
- Use the data to simulate trades on unsupported securities

Common data sources:
-GitHub
    - Can use public or private repos. Use the raw file "https://raw.githubusercontent.com/<organization>/<repo>/<path>
    - Private repos include a token in the raw url
-DropBox
    - Ensure you download the direct file link, not the HTML page of the file. To specify this, add &dl=1 to the end of the Dropbox download URL.
-REST Endpoints
