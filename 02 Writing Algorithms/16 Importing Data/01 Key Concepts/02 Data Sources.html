You can import data from local files, remote files, or REST endpoints.

Local Files

To access local files in your algorithms, you must run local algorithms with the CLI. The file path is relative to the project root. For example, if you have a structure like <projectName>/custom-data/file.csv, use the following snippets:



# Import Technique 1: Bulk downloads
data = self.Download("custom-data/file.csv")

# Import Technique 2: Processing samples
def GetSource(self, config, date, isLive):
    return SubscriptionDataSource("custom-data/file.csv", SubscriptionTransportMedium.LocalFile)






Remote Files


The most common remote file providers to use are GitHub and Dropbox. To access remote files, add ?dl=1 to the end of the file URL. This parameter lets you download the direct file link, not the HTML page of the file.



GitHub


If you store your custom data in GitHub, you can use public or private repositories. When you download the data, use the raw file (for example, https://raw.githubusercontent.com/<organization>/<repo>/<path>). For instructions on accessing the raw file, see Viewing or copying the raw file content in the GitHub documentation.







# Import Technique 1: Bulk downloads
data = self.Download("https://raw.githubusercontent.com/<organization>/<repo>/<path>?dl=1")

# Import Technique 2: Processing samples
def GetSource(self, config, date, isLive):
    return SubscriptionDataSource("https://raw.githubusercontent.com/<organization>/<repo>/<path>?dl=1", SubscriptionTransportMedium.RemoteFile)





Dropbox


If you store your custom data in Dropbox, you need to create a link to the file. To create file links, see How to share files or folders in the Dropbox documentation.






# Import Technique 1: Bulk downloads
data = self.Download("https://www.dropbox.com/<filePath>?dl=1")

# Import Technique 2: Processing samples
def GetSource(self, config, date, isLive):
    return SubscriptionDataSource("https://www.dropbox.com/<filePath>?dl=1", SubscriptionTransportMedium.RemoteFile)





REST Endpoints


To access the data from REST endpoints, pass the endpoint URL to the Download method or the SubscriptionDataSource constructor.







# Import Technique 1: Bulk downloads
data = self.Download("<sourceURL>")

# Import Technique 2: Processing samples
def GetSource(self, config, date, isLive):
    return SubscriptionDataSource("<sourceURL>", SubscriptionTransportMedium.Rest)
