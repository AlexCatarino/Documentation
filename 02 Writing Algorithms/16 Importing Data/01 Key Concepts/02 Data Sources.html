Common data sources:
-Local Files
    - Via CLI
    - Relative to the project root. For example, if you have a structure like <projectName>/custom-data/file.csv, use self.Download("custom-data/file.csv")

-Remote files
  -GitHub
      - Use the raw file "https://raw.githubusercontent.com/<organization>/<repo>/<path>". For instructions on accessing the raw file, see Viewing or copying the raw file content in the GitHub documentation.
      - Can use public or private repos. Private repos include a token in the raw url
  -DropBox
      - Ensure you download the direct file link, not the HTML page of the file. To specify this, add &dl=1 to the end of the Dropbox download URL.

-REST Endpoints
