Follow these steps to transport binary files:

1) Add the following imports to your program:
import pickle
import base64
2) Serialize your object into bytes.
pickle_bytes = pickle.dumps(my_object)
3) Encode the bytes object using Base64 and then decode the result into ASCII.
pickle_bytes = pickle.dumps(my_object)
base64_str = base64.b64encode(pickle_bytes).decode('ascii')
4) Save the base64 string representation of your object into the ObjectStore or one of the supported external sources.

5) Load the base64 string representation of your object into your trading algorithm.

6) Restore the object.
base64_bytes = base64_str.encode('ascii')
model = base64.b64decode(base64_bytes)
restored_model = pickle.loads(model)
