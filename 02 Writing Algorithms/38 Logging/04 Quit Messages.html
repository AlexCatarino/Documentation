Algorithms can send error messages to the console using the $[Error(string message),M:QuantConnect.Algorithm.QCAlgorithm.Error] method. Error messages appear red in the console, so you can see them easily. Like debug messages, error messages are rate limited to avoid flooding your browser. 

Error("Volatility too high, terminating algorithm.");
Quit(); //Option: Instruct algorithm to stop.

self.Error("Volatility too high, terminating algorithm.")
self.Quit() # Optional: Instruct algorithm to stop.

this.Error("Volatility too high, triggering exit")
this.Quit()
