Symbol change events can be accessed via Slice events. Slice delivers unique events to your algorithm as they happen. We recommend saving the Symbol object when you add the data for easy access to the slice later. You can see an example of the slice accessor in the code below.

def OnData(self, data: Slice) -> None:
    symbolChangedEvent = data.SymbolChangedEvents.get(self.symbol)
    if symbolChangedEvent:
        self.Log(f"Symbol changed: {symbolChangedEvent.OldSymbol} -> {symbolChangedEvent.NewSymbol}")

public override void OnData(Slice data)
{
    if (data.SymbolChangedEvents.ContainsKey(_symbol))
    {
        var symbolChangedEvent = data.SymbolChangedEvents[_symbol];
        Log($"Symbol changed: {symbolChangedEvent.OldSymbol} -> {symbolChangedEvent.NewSymbol}");
    }
}
