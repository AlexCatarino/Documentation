Data can be accessed via Slice events. Slice delivers unique events to your algorithm as they happen. We recommend saving the symbol object when you add the data for easy access to slice later. Data is available in second resolution. You can see an example of the slice accessor in the code below.

def OnData(self, slice: Slice) -> None:
    if slice.ContainsKey(self.tiingo_symbol):
        description = slice[self.tiingo_symbol].Description
        self.Log(f"Received Tiingo News article at {self.Time} with description {description}")
