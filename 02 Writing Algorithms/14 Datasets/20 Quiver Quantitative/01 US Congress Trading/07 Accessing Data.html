Data can be accessed via Slice events. Slice delivers unique events to your algorithm as they happen. We recommend saving the symbol object when you add the data for easy access to slice later. Data is available in daily resolution. You can see an example of the slice accessor in the code below.

def OnData(self, slice: Slice) -> None:
    points = slice.Get(QuiverCongress)
    for point in points.Values:
        self.Log(f"Symbol: {point.Symbol.Underlying} Report date: {point.ReportDate}; Transaction date: {point.TransactionDate}; Representative: {point.Representative}; Transaction: {point.Transaction}; Amount: {point.Amount}; House: {point.House}")

public override void OnData(Slice slice)
{
    var points = slice.Get<QuiverCongress>();
    foreach (var point in points.Values)
    {
        Log($"Symbol: {point.Symbol.Underlying}; Report date: {point.ReportDate}; Transaction date: {point.TransactionDate}; Representative: {point.Representative}; Transaction: {point.Transaction}; Amount: {point.Amount}; House: {point.House}");
    }
}
