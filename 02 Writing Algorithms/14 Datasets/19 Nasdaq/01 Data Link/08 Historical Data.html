You can request historical data by using the data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no data in the period you request, the history result will be empty. The following example gets the historical data for Nasdaq Data Link by using the History API.

history = self.History(NasdaqDataLink, self.nasdaq_ibm, 60, Resolution.Daily)
var history = History(new[] {_nasdaqIbm}, 60, Resolution.Daily);
