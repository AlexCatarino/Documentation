The benchmark performance is input to calculate several statistics on your algorithm, including alpha and beta. To set a benchmark for your algorithm, call the $[SetBenchmark(), M:QuantConnect.Algorithm.QCAlgorithm.SetBenchmark] method. You can set the benchmark to a security, a constant value, or a value from a custom data source. By default, the benchmark is SPY.

// Set the benchmark to IBM
SetBenchmark("IBM");

// Set the benchmark to a constant value of 0
SetBenchmark(x => 0);

// Set the benchmark to a value from a custom data source
var symbol = AddData<CustomData>("CustomData", Resolution.Hour).Symbol;
SetBenchmark(symbol);
# Set the benchmark to IBM
self.SetBenchmark("IBM")

# Set the benchmark to a constant value of 0
self.SetBenchmark(lambda x: 0)

# Set the benchmark to a value from a custom data source
self.symbol = self.AddData(ExampleCustomData, "ExampleCustomData", Resolution.Hour).Symbol
self.SetBenchmark(self.symbol)
