After the Initialize method, the PostInitialize method performs post initialization routines, so don't override it. To be notified when the algorithm is ready to begin trading, define an OnWarmUpFinished event handler. This method is called even if you don't set a warm up period.

public override void OnWarmUpFinished()
{
    Log("Algorithm Ready");
}
def OnWarmUpFinished(self) -> None:
    self.Log("Algorithm Ready")
