You may need some historical data at the start of your algorithm to prime technical indicators or populate historical data arrays. The warm up period pumps data into your algorithm from before the start date. To set a warm up period, call the $[SetWarmUp(TimeSpan period),M:QuantConnect.Algorithm.QCAlgorithm.SetWarmUp] or $[SetWarmUp(int barCount),M:QuantConnect.Algorithm.QCAlgorithm.SetWarmUp] method. 

// Wind time back 7 days from start
SetWarmUp(TimeSpan.FromDays(7));

// Feed in 100 daily bars before the start date
SetWarmUp(100, Resolution.Daily);

// Feed in 100 bars before start date
SetWarmUp(100);

# Wind time back 7 days from start
self.SetWarmUp(timedelta(7))

# Feed in 100 daily bars before the start date
self.SetWarmUp(100, Resolution.Daily)

# Feed in 100 bars before start date
self.SetWarmUp(100)
