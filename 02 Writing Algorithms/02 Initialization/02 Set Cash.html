In backtests you can set your starting capital using the SetCash(decimal cash)self.SetCash(float cash)this.SetCash(decimal cash) method. By default, we set your starting cash to $100,000 USD. In live trading, the SetCash method is ignored and your brokerage cash is used instead. 

SetCash(100000)       // Set USD quantity to $100,000
SetCash("BTC", 10)    // Set Bitcoin quantity to 10
SetCash("EUR", 10000) // Set EUR quantity to 10,000
self.SetCash(100000)       # Set USD quantity to $100,000
self.SetCash("BTC", 10)    # Set Bitcoin quantity to 10
self.SetCash("EUR", 10000) # Set EUR quantity to 10,000
