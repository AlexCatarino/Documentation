-Some brokerages supports additional order properties to customize order execution.
-For example:
   - Financial Advisor group orders (only supported by IB)
   - OutsideRegularTradingHours (only configurable through IB; other brokerages support after-hours trading)
   - PostOnly: This flag will ensure the order add liquidity only
   - Hidden: The order does not appear in the order book; thus does not influence other market participants.
   - ReduceOnly: This order will only trade if it would decrease your position size.
   - FeeInBase: Prefer fee in base currency (default if selling)
   - FeeInQuote: Prefer fee in quote currency (default if buying, mutually exclusive with FeeInBase)
   - NoMarketPriceProtection: https://support.kraken.com/hc/en-us/articles/201648183-Market-Price-Protection (Only supported by Kraken)

-To see the order properties supported by the brokerage you're using, see the Orders section of the integration documentation for your brokerage.
