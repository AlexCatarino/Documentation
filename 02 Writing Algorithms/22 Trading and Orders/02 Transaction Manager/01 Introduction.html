The algorithm Transactions Manager ($[SecurityTransactionManager,T:QuantConnect.Securities.SecurityTransactionManager]) is a collection of helper methods for quick access to all your orders. It is located in the self.TransactionsTransactions property on your algorithm.

Why should you not really use orders, and only use order tickets?
- The order tickets are automatically updated as the order is processed by the brokerage. If you save a reference to your order tickets, you shouldn't need to use the TransactionManager.
