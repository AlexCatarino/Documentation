Using the Transaction Manager, you can fetch a list of all open orders for a symbol. This is helpful if you want to update multiple open orders for a specific symbol. The method returns a list of $[Order,T:QuantConnect.Orders.Order] objects. 

// Get all open orders
var openOrders = Transactions.GetOpenOrders();

// Get all open orders that pass a filter
var filteredOpenOrders = Transactions.GetOpenOrders(x => x.Quantity > 10);

// Get a list of all open orders for a symbol
var symbolOpenOrders = Transactions.GetOpenOrders(symbol);

# Get all open orders
