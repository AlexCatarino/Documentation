The Cancel helpers can cancel all open orders, or just those orders related with a specific symbol. The method returns a list of $[OrderTicket,T:QuantConnect.Orders.OrderTicket] objects. This is helpful if you are simulating an "OCA / One-Cancels-All" style of order where you want to cancel other related orders. 

// Cancel all open orders
var allCancelledOrders = Transactions.CancelOpenOrders();

// Cancel orders related to IBM, apply string tag.
var ibmCancelledOrders = Transactions.CancelOpenOrders("IBM", "Hit stop price");

 # Cancel all open orders
all_cancelled_orders = self.Transactions.CancelOpenOrders()

# Cancel orders related to IBM and apply a tag
ibm_cancelled_orders = self.Transactions.CancelOpenOrders("IBM", "Hit stop price")
