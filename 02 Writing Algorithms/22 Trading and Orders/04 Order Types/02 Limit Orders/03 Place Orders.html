To send a limit order, call the LimitOrder method with a symbol, quantity, and limit price. If you do not have sufficient capital for the order, it's rejected.

// Buy 1 Bitcoin at $40,000
var buyTicket = LimitOrder("BTCUSD", 1, 40000);

// Sell 1 Bitcoin at $40,000
var sellTicket = LimitOrder("BTCUSD", -1, 40000);
# Buy 1 Bitcoin at $40,000
buy_ticket = self.LimitOrder("BTCUSD", 1, 40000)

# Sell 1 Bitcoin at $40,000
sell_ticket = self.LimitOrder("BTCUSD", -1, 40000)

You can provide a tag and order properties to the LimitOrder method.

LimitOrder(symbol, quantity, limitPrice, tag, orderProperties);
self.LimitOrder(symbol, quantity, limitPrice, tag, orderProperties)

To place a marketable buy limit order, set the limit price to the current ask price or higher.

To place a marketable sell limit order, set the limit price to the current bid price or lower.

To place an unmarketable buy limit order, set the limit price below the current ask price.

To place an unmarketable sell limit order, set the limit price above the current bid price.
