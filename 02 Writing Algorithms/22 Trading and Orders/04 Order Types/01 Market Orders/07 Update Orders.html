Since market orders are transmitted to the brokerage almost immediately, you can only update the tag on the order. To update the order tag, call the UpdateTag method on the order ticket. The method returns an $[OrderResponse, T:QuantConnect.Orders.OrderResponse] that signals the success or failure of the update request. 

// Create a new order and save the order ticket
var ticket = MarketOrder("SPY", 100, tag: "original tag");

// Update the order tag
var response = ticket.UpdateTag("new tag");

// Check if the update was successful
if (response.IsSuccess) { 
     Debug("Order updated successfully");
}
