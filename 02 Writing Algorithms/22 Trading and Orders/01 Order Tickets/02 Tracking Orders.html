-As the state or your order updates over time, your order ticket automatically updates. Using just the order ticket, you can get the following properties of your order:
    - Status (Redirect to OrderEvents page for OrderStatus enum values)
    - Quantity
    - AverageFillPrice
    - QuantityFilled
    - Time (last order update)
    - OrderType
    - Tag
    - UpdateRequests
    - CancelRequests
    - OrderEvents
    - OrderClosed (a wait handle)
-To get an order field, use orderTicket.Get(<orderField>)
-To get most recent order response, use the GetMostRecentOrderResponse method

-Instead of checking the properties of the order ticket to view its state, every time an order is place, filled, canceled, updated, etc..., you receive order events through the OnOrderEvent event handler.
    -For more information about order events, see the Order Events page
