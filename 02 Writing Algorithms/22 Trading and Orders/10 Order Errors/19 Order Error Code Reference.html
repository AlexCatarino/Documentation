When an order fails to process it returns with a negative order-id. These error codes mean different things as described in the table below.

EnumError CodeDescription
0NoneNo error
-1ProcessingErrorUnknown error
-2OrderAlreadyExistsCannot submit because order already exists
-3InsufficientBuyingPowerNot enough money to to submit order
-4BrokerageModelRefusedToSubmitOrderInternal logic invalidated submit order
-5BrokerageFailedToSubmitOrderBrokerage submit error
-6BrokerageFailedToUpdateOrderBrokerage update error
-7BrokerageHandlerRefusedToUpdateOrderInternal logic invalidated update order
-8BrokerageFailedToCancelOrderBrokerage cancel error
-9InvalidOrderStatusOnly pending orders can be canceled
-10UnableToFindOrderMissing order
-11OrderQuantityZeroCannot submit or update orders with zero quantity
-12UnsupportedRequestTypeThis type of request is unsupported
-13PreOrderChecksErrorUnknown error during pre order request validation
-14MissingSecuritySecurity is missing. Probably did not subscribe.
-15ExchangeNotOpenSome order types require open exchange
-16SecurityPriceZeroZero security price is probably due to bad data
-17ForexBaseAndQuoteCurrenciesRequiredNeed both currencies in cashbook to trade a pair
-18ForexConversionRateZeroNeed conversion rate to account currency
-19SecurityHasNoDataShould not attempt trading without at least one data point
-20ExceededMaximumOrdersTransaction manager's cache is full
-21MarketOnCloseOrderTooLateNeed 11 minute buffer before exchange close
-22InvalidRequestRequest is invalid or null
-23RequestCanceledRequest was canceled by user
-24AlgorithmWarmingUpAll orders are invalidated while algorithm is warming up
-25BrokerageModelRefusedToUpdateOrderInternal logic invalidated update order
-26QuoteCurrencyRequiredNeed quote currency in cashbook to trade
-27ConversionRateZeroNeed conversion rate to account currency
-28NonTradableSecurityThe order's symbol references a non-tradable security
-29NonExercisableSecurityThe order's symbol references a non-exercisable security
-30OrderQuantityLessThanLoteSizeCannot submit or update orders with quantity that is less than lot size
-31ExceedsShortableQuantityThe order's quantity exceeds the max shortable quantity set by the brokerage
-32InvalidNewOrderStatusCannot update/cancel orders with OrderStatus.New
