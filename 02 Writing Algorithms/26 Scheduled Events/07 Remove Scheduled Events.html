- Remove Scheduled Events if you no longer need them to ensure your algorithm doesn't process unecessary logic.
- To remove a Scheduled Event, save a reference to the Scheduled Event when you create it. When you want to remove it, call the Remove method.

// Create a Scheduled Event
var event = Schedule.On(DateRules.EveryDay("SPY"), TimeRules.AfterMarketOpen("SPY", 10), () => {});

// Remove the Scheduled Event
Schedule.Remove(event);
