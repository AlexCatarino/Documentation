- Ensure you have a low enough data resolution to avoid stale fills.
    - Set at midnight => Any resolution
    - Set at an hour (not midnight) => Tick, Second, Minute, Hour
    - Set at a minute (not :00:00) => Tick, Second, Minute
    - Set at a second (not :00:00) => Tick, Second

Don't set at midnight/:00:00 above because the scheduled event is called before OnData
