The process to creating a consolidator depends on the type of consolidator you want to create. For instructions on the various consolidators we support, see Consolidator Types.

The most common error is to put braces "()" at the end of your function call when setting the event handler of the consolidator. If you use braces, the method executes and the result is passed as the event handler instead of the method itself. Remember to simply pass the name of your method to the event system. Specifically, it should be ConsolidationHandlerself.ConsolidationHandler, not ConsolidationHandler()self.ConsolidationHandler().
