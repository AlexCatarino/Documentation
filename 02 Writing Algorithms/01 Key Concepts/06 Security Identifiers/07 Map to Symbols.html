You can create Symbol from other industry-standard security identifiers like CUSIP, FIGI, ISIN, and SEDOL.

CUSIP

To convert a Committee on Uniform Securities Identification Procedures (CUSIP) number to a Symbol, call the CUSIP method.

-03783310 => AAPL R735QTJ8XC9X
-self.CUSIP()
