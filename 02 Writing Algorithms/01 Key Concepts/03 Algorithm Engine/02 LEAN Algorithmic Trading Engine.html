    4. What are the supported programming languages?

       - Python and C#

       - If you program in Python, LEAN uses Python.Net to bridge between the C# engine and your algorithm. It can be slow to move from Python to C#. If you access C# objects in your Python algorithm, it's fastest to only access them once and save a reference (for example, self.Securities[symbol].Close.)

       - If your Python algortihm subsclasses a C# class, the constructor must include `super().__init__`
