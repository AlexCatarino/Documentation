A particularly common and helpful use of the RollingWindow class is to store past indicator values. Indicators emit an Updated event after they have been updated. To create a RollingWindow of indicator points, attach an event handler function to Updated that adds the last value of the indicator to the rolling window. The value is an $[IndicatorDataPoint, C:QuantConnect.Indicators.IndicatorDataPoint] object that represents a piece of data at a specific time. The following examples create an indicator and add its values to a rolling window when the indicator is updated. 

// In Initialize, create the rolling windows
public override void Initialize()
{
    // Creates an indicator and adds to a rolling window when it is updated
   smaWindow = new RollingWindow<IndicatorDataPoint>(5);
   SMA("SPY", 5).Updated += (sender, updated) => smaWindow.Add(updated);
}

def Initialize(self):
    # Creates an indicator and adds to a rolling window when it is updated
    self.sma_window = RollingWindow[IndicatorDataPoint](5)
    self.SMA("SPY", 5).Updated += (lambda sender, updated: self.sma_window.Add(updated))
