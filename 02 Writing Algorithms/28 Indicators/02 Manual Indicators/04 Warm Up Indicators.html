Indicators compute their value based on historical data. Before you start trading with an indicator, warm it up. There are several ways to warm up automatic indicators.

Manual Indicator Warm Up

If you access to the QCAlgorithm object, you can manually warm up indicators with a history request. 
private SimpleMovingAverage _sma;

_sma = SimpleMovingAverage(20);
var history = algorithm.History(symbol, 20, Resolution.Daily);
foreach (var bar in history)
{
    sma.Update(bar.Time, bar.Close);
}

self.sma = SimpleMovingAverage(20)
history = algorithm.History(symbol, 20, Resolution.Daily)
if not history.empty:
    for time, row in history.loc[symbol].iterrows():
        self.sma.Update(time, row.close)

Algorithm Warm Up

If you create indicators at the beginning of your algorithm, you can set a algorithm warm up period to warm up the indicators. 
private SimpleMovingAverage _sma;

// In Initialize
var symbol = AddEquity("SPY").Symbol;
_sma = SimpleMovingAverage(20);
SetWarmUp(20, Resolution.Daily);

// In OnData
_sma.Update(data["SPY"]);
if (IsWarmingUp) {
    return;
}
# In Initialize
symbol = self.AddEquity("SPY").Symbol
self.sma = self.SMA(symbol, 20, Resolution.Daily)
self.SetWarmUp(20, Resolution.Daily)

# In OnData
self.sma.Update(data["SPY"])
if self.IsWarmingUp:
    return
