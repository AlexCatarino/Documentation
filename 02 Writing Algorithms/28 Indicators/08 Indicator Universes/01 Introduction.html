What is an indicator universe?

- A universe of securities that you select based on indicator values


To create an indicator universe,

(1) Define a SymbolData class that contains the selection indicators

(2) Define a universe that updates the SymbolData objects and selects the correponding symbols based on their indicator values


Limits:

- With this techinque, universe selection occurs once per day.

     - In backtesting, it occurs at midnight. To see the timing of live trading, open the Datafeeds documentation, select a data feed, and then check the Universe Selection section.

- You can't unsubscribe to the securities intraday

- The last data point you have in selecting a universe is yesterday's close. Pre-market data won't be available at the time of selection.


For more information about universes, see Universes.
