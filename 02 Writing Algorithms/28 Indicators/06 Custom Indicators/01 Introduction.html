What is a custom indicator?

- An indicator that you define. It receives input, performs some calculation, and sets the output value of the indicator


Why use a custom indicator?

- To use an indicator that we don't currently have implemented in LEAN

- To combine built-in indicators in ways that Indicator Extensions don't support

- To create your own unique indicator logic for trading
