Divide indicator chain by constant or indicator. The .Over(IIndicator) extension divides the .Current.Value of an indicator by the .Current.Value of another indicator or a fixed value.

// Sum the output of two indicators
var rsiShort = RSI("SPY", 14);
var rsiLong = RSI("SPY", 21);
var rsiDivision = rsiShort.Over(rsiLong);

// Sum the output of an indicator and a fixed value
var rsiAverage = rsiShort.Plus(rsiLong).Over(2);
# Sum the output of two indicators
rsi_short = self.RSI("SPY", 14)
rsi_long = self.RSI("SPY", 21)
rsi_division = rsiShort.Over(rsi_long)

# Sum the output of an indicator and a fixed value
rsi_average = rsiShort.Plus(rsi_long).Over(2)

If you pass an indicator to the .Over(IIndicator) extension, you can name the composite indicator.

var namedIndicator = rsiShort.Over(rsiLong, "RSI Division");
named_indicator = rsiShort.Over(rsi_long, "RSI Division")
