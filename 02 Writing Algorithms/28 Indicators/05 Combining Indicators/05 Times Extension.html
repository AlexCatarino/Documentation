The .Times(IIndicator) extension multiplies the .Current.Value of two indicators or multiplies a fixed value and the .Current.Value of an indicator.

// Subtract the output of two indicators
var emaShort = EMA("SPY", 14);
var emaLong = EMA("SPY", 21);
var emaDifference = emaShort.Times(emaLong);

// Sum the output of an indicator and a fixed value
var emaMinusValue = emaShort.Times(10)
# Sum the output of two indicators
ema_short = self.EMA("SPY", 14)
ema_long = self.EMA("SPY", 21)
ema_difference = ema_short.Times(ema_long)

# Sum the output of an indicator and a fixed value
ema_times_value = ema_short.Times(1.5)

If you pass an indicator to the .Times(IIndicator) extension, you can name the composite indicator.

var namedComposite = emaShort.Times(emaLong, "EMA Product");
named_composite = ema_short.Times(ema_long, "EMA Product")
