The Of extension feeds an indicator's .Current.Value into the input of another indicator. The first argument of the IndicatorExtensions.Of method must be an unregistered indicator. If you pass a registered indicator as the argument, that first indicator is updated twice. The first update is from the registered security and the second update is from the IndicatorExtensions class. To see how to create unregistered indicators, open the Indicator Reference, click an indicator, and scroll down to the Manual Usage section. The following example demonstrates how to create an unregistered SimpleMovingAverage indicator:

var rsi = RSI("SPY", 14);
var rsiSMA = IndicatorExtensions.Of(new SimpleMovingAverage(10), rsi);
rsi = self.RSI("SPY", 14)
self.rsi_sma = IndicatorExtensions.Of(SimpleMovingAverage(10), rsi)
