The WeightedBy extension calculates the average .Current.Value of an indicator over a lookback period, weighted by another indicator over the same looking such that

value = (x_1 * y_1 + x_2 * y_2 + ... + x_n * y_n) / sum(y_1, y_2, ..., y_n) 
var smaShort = SMA("SPY", 14); 
var smaLong = SMA("SPY", 21); 
var weightedSMA = smaShort.WeightedBy(smaLong, 3);

sma_short = self.SMA("SPY", 14)
sma_long = self.SMA("SPY", 21)
weighted_sma = sma_short.WeightedBy(sma_long, 3)
