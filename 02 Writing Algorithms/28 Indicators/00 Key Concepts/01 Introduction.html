Indicators translate a stream of data points into a numerical value. LEAN provides more than 100 pre-built technical indicators and candlestick patterns you can use in your algorithms to detect trading opportunities. To view a list of all the indicators, see the Indicator Reference. One simple indicator to learn is the Identity indicator, which just returns the value of the asset.

var pep = Identity("PEP"); // Pepsi ticker
pep = Identity("PEP") # Pepsi ticker
