To reset indicators, call the Reset method.

self.indicator.Reset() 

_indicator.Reset();

If you are live trading Equities or backtesting Equities without the adjusted data normalization mode, reset your indicators when splits and dividends occur. If a split or dividend occurs, the data in your indicators becomes invalid because it doesn't account for the price adjustments caused by the split or dividend. To replace your indicator history with the newly-adjusted prices, call the Reset method and then warm up the indicator with a History call.

# In OnData
if data.Splits.ContainsKey(self.symbol) or data.Dividends.ContainsKey(self.symbol):
    # Reset the indicator
    self.sma.Reset() 

    # Warm up the indicator
    history = self.History(self.symbol, self.sma.Period, Resolution.Daily)
    if not history.empty:
        for time, close in history.loc[self.symbol].close.iteritems():
            self.sma.Update(time, close)
// In OnData
if (data.Splits.ContainsKey(_symbol) || data.Dividends.ContainsKey(_symbol))
{
    // Reset the indicator
    _sma.Reset();

    // Warm up the indicator
    var history = History(_symbol, _sma.Period, Resolution.Daily);
    foreach (var bar in history)
    {
        _sma.Update(bar.EndTime, bar.Close);
    }
}
