There are two ways to create indicators. You can create indicators with the indicator constructor or use a helper method in the QCAlgorithm class. If you use the helper method, the indicator is automatically registered for updates as new data arrives to your algorithm. 
// Create a manual indicator with the indicator constructor
var manualSMA = SimpleMovingAverage(20, Resolution.Daily);

// Create an automatic indicator with the helper method
var symbol = AddCrypto("BTCUSD");
var autoSMA = SMA(symbol, 20, Resolution.Daily);
# Create a manual indicator with the indicator constructor
self.manual_sma = SimpleMovingAverage(20, Resolution.Daily)

# Create an automatic indicator with the helper method
symbol = self.AddCrypto("BTCUSD");
auto_sma = self.SMA(symbol, 20, Resolution.Daily)
