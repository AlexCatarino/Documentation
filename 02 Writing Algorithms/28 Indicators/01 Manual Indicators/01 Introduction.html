What are manual indicators?
- Manual indicators are indicators you create by calling the indicator constructor.

Why use manual indicators?
- Manual indicators are not automatically hooked up for updates from the underlying security
    - This lets you customize how you update the indicator
    - This lets you use the indicator in dynamic universes without causing your algorithm to slow down over time

Why not use manual indicators?
-It's not always necessary to create manual indicators. If your algorithm only uses a static universe and you want to update the indicator with data at the same resolution of the security subscription, create an automatic indicator. Automatic indicators are easier to create + update
