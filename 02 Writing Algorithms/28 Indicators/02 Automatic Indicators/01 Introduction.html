What are automatic indicators?
- Indicators that you create with the shortcut method from the QCAlgorithm API instead of using the indicator constructor
- Automatic indicators do everything that manual indicators do, but with automatic updates registered for you.

What use automatic indicators?
- Automatic indicators are automatically hooked up to be updated from the security price so you don't have to hook them up / update them yourself.

Why not use automatic indicators?
- If you have a dynamic universe, you shouldn't use automatic indicators because you can't remove the consolidator that keeps the indicator updated. Since you can't remove the consolidator, it slows down your algorithm.
