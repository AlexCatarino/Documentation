To get the exchange hours, use the Exchange.Hours property on a Security object.

var securityExchangeHours = Securities["SPY"].Exchange.Hours;
security_exchange_hours = self.Securities["SPY"].Exchange.Hours

The preceeding code snippet returns a SecurityExchangeHours object, which has the following attributes:

-Access the SecurityExchangeHours object through <security>.Exchange.Hours or through MarketHoursDatabase.GetExchangeHours()
-It has these properties: IsOpen, IsDateOpen, GetMarketHours, GetNextMarketOpen, GetNextMarketClose, GetPreviousTradingDay, GetNextTradingDay

- To get time zone of a security, use self.MarketHoursDatabase.GetDataTimeZone() - Also Security.Exchange.TimeZone
