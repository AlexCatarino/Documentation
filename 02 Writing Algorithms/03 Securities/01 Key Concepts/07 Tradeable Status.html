The IsTradable property shows whether you can trade a security. The property value is true when the security is in the universe, even if the data starts at a later day. Indices and canonical securities like the canonical Option Symbol are not tradable. In live mode, custom data objects are also not tradable, even if the custom data represents a tradable asset.

var tradable = Securities["SPY"].IsTradable;
tradable = self.Securities["SPY"].IsTradable
