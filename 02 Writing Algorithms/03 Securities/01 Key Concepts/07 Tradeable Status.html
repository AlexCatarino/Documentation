-IsTradable means that the security was not removed from the universe and/or delisted
-If the data is found in an universe, IsTradable is True even of the data only start later, since IsTradable doesn't know whether there is data or not.
-Indices and canonical Securities (those that represent a Future or Option chain) are not tradable
-In live mode, custom data, even if it represents a tradable asset (e.g. your own equity data) is not tradable.
