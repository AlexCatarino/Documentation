To create an Equity subscription, in the Initialize method, call the AddEquity method. The AddEquity method returns a Security object, which contains a Symbol. Save a reference to the Symbol so you can use it in OnData to access the security data in the Slice.

_symbol = AddEquity("SPY").Symbol;
self.symbol = self.AddEquity("SPY").Symbol

The AddEquity method creates a subscription for a single Equity asset and adds it to your user-defined universe. To create a dynamic collection of Equities, add an Equity universe or an Equity Universe Selection model.

To remove a subscription that you created with the AddEquity method, call the RemoveSecurity method.

RemoveSecurity(_symbol);
self.RemoveSecurity(self.symbol)

The RemoveSecurity method cancels your open orders for the security and liquidates your holdings.
