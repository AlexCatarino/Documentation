LEAN passes the data you request to the OnData method so you can make trading decisions.The default OnData method accepts a Slice objects, but you can define additional OnData methods that accept different data types (for example, a TradeBar). The Slice object that the OnData method receives groups all the data together at a single moment in time.1

All data formats use DataDictionary objects to group data by Symbol and provide easy access to information. The plural of the type denotes the collection of objects. The TradeBars DataDictionary is made up of TradeBar objects and the QuoteBars DataDictionary is made up of QuoteBar objects. To access individual data points in the dictionary, you can index the dictionary with the security ticker or Symbol, but we recommend you use the Symbol.
