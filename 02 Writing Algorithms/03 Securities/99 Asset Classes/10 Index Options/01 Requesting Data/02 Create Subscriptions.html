Before you can subscribe to an Index Option contract, you must get the contract Symbol.

Get Contract Symbols

To get Index Option contract Symbol objects, call the CreateOption method or use the OptionChainProvider. If you use the CreateOption method, you need to know the specific contract details.

var strike = 100m;
var expiry = DateTime.Today + TimeSpan.FromDays(30);
var index = Symbol.Create("VIX", SecurityType.Index, Market.USA);
var option = Symbol.CreateOption(index, Market.USA, OptionStyle.American, OptionRight.Put, strike, expiry);
strike = 100
expiry = datetime.today + timedelta(days=30)
index = Symbol.Create("CL", SecurityType.Index, Market.USA)
option = Symbol.CreateOption(index, Market.USA, OptionStyle.American, OptionRight.Put, strike, expiry)

Another way to get an Index Option contract Symbol is to use the OptionChainProvider. The GetOptionContractList method of OptionChainProvider returns a list of Option contracts that are available for a given underlying Index on a given date.

var index = Symbol.Create("VIX", SecurityType.Index, Market.USA);
var indexOptionContracts = OptionChainProvider.GetOptionContractList(index, Time);
index = Symbol.Create("CL", SecurityType.Index, Market.USA)
index_option_contracts = self.OptionChainProvider.GetOptionContractList(index, self.Time)

Subscribe to Contracts

To create an Index Option contract subscription, pass the contract Symbol to the AddOptionContract method. Save a reference to the contract Symbol so you can easily access the contract in the OptionChain that LEAN passes to the OnData method.

AddOptionContract(_contractSymbol);
self.AddOptionContract(self.contract_symbol)

The AddOptionContract method creates a subscription for a single Index Option contract and adds it to your user-defined universe. To create a dynamic universe of Index Option contracts, add an Index Option universe.

Warm Up Contract Prices

If you subscribe to an Index Option contract with AddOptionContract, you'll need to wait until the next Slice to receive data and trade the contract. To trade the contract in the same time step you subscribe to the contract, set the current price of the contract in a security initializer.

SetSecurityInitializer(security => {
    if (security.Type == SecurityType.IndexOption)
    {
        security.SetMarketPrice(GetLastKnownPrice(security));
    }
});
    
self.SetSecurityInitializer(self.CustomSecurityInitializer)

def CustomSecurityInitializer(self, security: Security) -> None:
    if security.Type == SecurityType.IndexOption:
        security.SetMarketPrice(self.GetLastKnownPrice(security))
