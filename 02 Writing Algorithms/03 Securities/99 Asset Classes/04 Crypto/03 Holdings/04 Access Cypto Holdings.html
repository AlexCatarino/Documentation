The CashBook stores the quantity of your Crypto holdings. To access your crypto holdings, key the CashBook with the currency ticker.

## Example code snippet
btc_quantity = self.Portfolio.CashBook['BTC'].Amount
btc_value = self.Portfolio.CashBook['BTC'].ValueInAccountCurrency

To access the virtual pair of your crypto trades, key the Portfolio object with the pair symbol

## Example code snippet
self.Portfolio["BTCUSD"]
