To create a Crypto subscription, in the Initialize method, call the AddEquity method. The AddCrypto method returns a Security object, which contains a Symbol. Save a reference to the Symbol so you can use it in OnData to access the security data in the Slice.

_symbol = AddCrypto("BTCUSD").Symbol;
self.symbol = self.AddEquity("BTCUSD").Symbol

The AddCrypto method creates a subscription for a single Crypto asset and adds it to your user-defined universe. To create a dynamic universe of Crypto assets, add an Crypto universe.

To view the supported assets, see the Crypto price datasets in the Dataset Market.
