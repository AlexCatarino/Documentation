There is no limit to the number of securities to which you can subscribe. However, some live data providers, e.g., Interactive Brokers, impose data subscription limits. To set a limit, assign an integer to DataSubscriptionLimit, preferentially on the Initialize method:
Settings.DataSubscriptionLimit = 500;
self.Settings.DataSubscriptionLimit = 500
All securities added with IAlgorithm.AddSecurity are counted as one subscription, with the exception of Options and Futures where every single contract in a chain counts as one subscription.
