What:
-The default account currency is USD
-The account currency impacts the self.Portfolio.ValueInAccountCurrency calculation
- If the account currency is not added to the algorithm. e.g. the algorithm only adds BTCETH and the account currency is USDT, the engine will add BTCUSDT and ETHUSDT as internal feeds to be able to calculate the ValueInAccountCurrency.

How:
-To set the account currency, use self.SetAccountCurrency
    - You can only set the account currency once. Each addition call is ignored.
-To get the account currency, use self.AccountCurrency or self.Portfolio.CashBook.AccountCurrency.
