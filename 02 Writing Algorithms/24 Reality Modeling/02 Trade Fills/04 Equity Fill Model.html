- If you are trading Equities, the default model to use for filling trades in backtests is the EquityFillModel
- The EquityFillModel fills orders at the following prices (before applying slippage) and assumes the trades fill completley
    - Market orders
         - Only fills during regular trading hours
         - Can fill on stale data
         - If buying, fills at best effort ask price
             - To get best effort ask price:
                 (1) If tick data, check the askPrice of the most recent quote tick in the slice
                 (2) If QuoteBar data, check the closing ask price
                 (3) If tick data, check the lastTrade price
                 (4) If TradeBar data, check the close price of the current bar
         - If selling, fills at the best effort bid price
            - To get best effort bid price:
                 (1) If tick data, check the bidPrice of the most recent quote tick in the slice
                 (2) If QuoteBar data, check the closing bid price
                 (3) If tick data, check the lastTrade price
                 (4) If TradeBar data, check the close price of the current bar
                

     - Limit orders
         - Same as ImmediateFillModel

     - LimitIfTouched orders
         - Can trigger/fill during pre/post market
         - Wont fill on stale data
         - Convert to limit order when...
             - Tick data
                   - If buying, when lastTradePrice <= triggerPrice
                   - If selling, when lastTradePrice >= triggerPrice
             - TradeBar data
                   - If buying, when lowPrice <= triggerPrice
                   - if selling, when highPrice >= triggerPrice
         - Fill limit order when...
              - If buying, when bestEffortAsk <= limitPrice, at min(bestEffortAsk, limitPrice)
              - if selling, when bestEffortBid >= limitPrice, at max(bestEffortBid, limitPrice)

    - StopMarket orders
         - Same as ImmediateFillModel

    - StopLimit orders
         - Same as ImmediateFillModel
