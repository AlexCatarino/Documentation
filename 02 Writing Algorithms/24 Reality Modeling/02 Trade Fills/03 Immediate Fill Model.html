-For most asset classes, the default model to use for filling trades in backtests is the ImmediateFillModel
-The ImmediateFillModel fills orders at the following prices (before applying slippage) and assumes the trades fill completely.
    - Market orders
         - If using tick data:
              - If the last tick is a quote
                   - Fills at the quote price
              - If the last tick is a trade tick, fills at the last trade price
         - If using quotebar data, fills at the closing bid/ask price of the current bar
         - If using tradebar data, fills at the closing price of the current bar
     - Limit orders
         - Tick data
               - If the last tick is a quote
                    - If buying, filled when quotePrice < limitPrice, filled at min(quotePrice, limitPrice)
                    - If selling, filled when quotePrice > limitPrice, filled at max(quotePrice, limitPrice)
               - If the last tick is a trade tick
                    - If buying, filled when lastTradePrice < limitPrice, filled at min(lastTradePrice, limitPrice)
                    - If selling, filled when lastTradePrice > limitPrice, filled at max(lastTradePrice, limitPrice)
         - QuoteBar data (most recent bar)
               - If buying, filled when askLowPrice < limitPrice, filled at min(askLowPrice, limitPrice)
               - If selling, filled when bidHighPrice > limitPrice, filled at max(bidHighPrice, limitPrice)
         - TradeBar data (most recent bar)
               - If buying, filled when lowPrice < limitPrice, filled at min(lowPrice, limitPrice)
               - If selling, filled when highPrice > limitPrice, filled at max(highPrice, limitPrice)
     - LimitIfTouched orders
         - Converted to limit orders when...
             - Tick data
                 - If the last tick is a quote tick
                     - If buying, converted to limit order when quotePrice <= triggerPrice
                     - If selling, converted to limit order when quotePrice >= triggerPrice
                 - If the last tick is a trade tick
                     - If buying, converted to limit order when lastTradePrice <= triggerPrice
                     - If selling, converted to limit order when lastTradePrice >= triggerPrice
             - TradeBar data
                 - If buying, converted to limit order when the low of the current bar <= triggerPrice
                 - If selling, converted to limit order when the high of the current bar >= triggerPrice
          - Limit order filled when...
                 -
    - StopMarket orders
         - Filled when the high/low breaks through the stop price
         - Buy orders filled at max(stopPrice, closePrice + slippage)
         - Sell orders filled at min(stopPrice, closePrice - slippage)
         - The lowPrice and highPrice is from the current QuoteBar, if it's available, otherwise it's the TradeBar
    - StopLimit orders
         - Converted to limit order when high/low breaks through the stop price, limit filled when close price breaks through limitPrice
         - Buy orders filled at min(highPrice, limitPrice)
         - Sell orders filled at max(lowPrice, limitPrice)
         - High, low, close are from the current QuoteBar, if it's available, otherwise it's the TradeBar
     - MOO
         - Filled at the market open price (QuoteBar if it's available, otherwise it's the TradeBar)
     - MOC
         - Filled at the market close (QuoteBar if it's available, otherwise it's the TradeBar)
