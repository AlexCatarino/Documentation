What short availability models are there:
-NullShortableProvider: Allows for all assets to be infinitely shortable, with no restrictions
-AtreyuShortableProvider: Provides the available shares for open short positions in the US Equity market. The data covers 10,500 US Equities, starts in January 2018, and is delivered on a daily frequency. This dataset is created by Atreyu using information from the exchanges.
-LocalDiskShortableProvider: Sources easy-to-borrow (ETB) data from the local disk for the given brokerage
