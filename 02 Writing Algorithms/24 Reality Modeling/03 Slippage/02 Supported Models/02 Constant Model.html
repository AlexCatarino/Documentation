The ConstantSlippageModel applies a constant percentage of slippage to each order. Its the default slippage model of the DefaultBrokerageModel.

security.SetSlippageModel(new ConstantSlippageModel(0));
security.SetSlippageModel(ConstantSlippageModel(0))

The following table describes the arguments the model accepts:

Argument Data Type Description Default Value
slippagePercent decimal The slippage percent for each order. The value must be in the interval (0, 1).

To view the implementation of this model, see the LEAN GitHub repository.
