-The brokerage model of your algorithm automatically sets the buying power model for each security.
-The default brokerage model is the DefaultBrokerageModel, which uses the following buying power models for each asset class:
   -Futures: FutureMarginModel
   -FutureOption: FuturesOptionsMarginModel
   -IndexOption: OptionMarginModel
   -Option: OptionMarginModel
   -All other asset classes: CashBuyingPowerModel if a cash account, SecurityMarginModel if a margin account
