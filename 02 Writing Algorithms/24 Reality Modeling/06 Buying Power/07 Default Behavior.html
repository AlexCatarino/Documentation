The brokerage model of your algorithm automatically sets the buying power model of each security. The default brokerage model is the DefaultBrokerageModel, which sets the buying power model based on the asset class of the security. The following table describes the default buying power model of each asset class:

Asset Class Model
Equity Options OptionMarginModel
Futures FutureMarginModel
Future Options FuturesOptionsMarginModel
Index Options OptionMarginModel
Other CashBuyingPowerModel for cash accounts and SecurityMarginModel for margin accounts.
