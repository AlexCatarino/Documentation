-The brokerage model of your algorithm automatically sets the fill model for each security.
-The default brokerage model is the DefaultBrokerageModel, which uses
   -Futures: FutureMarginModel
   -FutureOption: FuturesOptionsMarginModel
   -IndexOption: OptionMarginModel
   -Option: OptionMarginModel
   -All other asset classes: CashBuyingPowerModel if a cash account, SecurityMarginModel if a margin account
