LEAN supports the following QuantLib implementations of Option price models:

  • AdditiveEquiprobabilities
  • BaroneAdesiWhaley
  • BinomialCoxRossRubinstein
  • BinomialJarrowRudd
  • BinomialJoshi
  • BlackScholes
QuantLib(QL) implementations:
-
   - Pricing engine for European vanilla options using analytical formula.
   - http://quantlib.org/reference/class_quant_lib_1_1_analytic_european_engine.html
   - https://www.quantconnect.com/tutorials/introduction-to-options/options-pricing-black-scholes-merton-model
-
    - Barone-Adesi and Whaley pricing engine for American options (1987)
    - http://quantlib.org/referenceA /class_quant_lib_1_1_barone_adesi_whaley_approximation_engine.html
-BjerksundStensland
    - Bjerksund and Stensland pricing engine for American options (1993)
    - http://quantlib.org/reference/class_quant_lib_1_1_bjerksund_stensland_approximation_engine.html
- Integral
    - Pricing engine for European vanilla options using integral approach.
    - http://quantlib.org/reference/class_quant_lib_1_1_integral_engine.html
- CrankNicolsonFD
    - Pricing engine for European options using finite-differences.
    - http://quantlib.org/reference/class_quant_lib_1_1_f_d_european_engine.html
-
    - Pricing engine for vanilla options using binomial trees. Jarrow-Rudd model.
    - http://quantlib.org/reference/class_quant_lib_1_1_f_d_european_engine.html
-
    - Pricing engine for vanilla options using binomial trees. Cox-Ross-Rubinstein(CRR) model.
    - http://quantlib.org/reference/class_quant_lib_1_1_f_d_european_engine.html
- BinomialTrigeorgis
    - Pricing engine for vanilla options using binomial trees. Trigeorgis model.
    - http://quantlib.org/reference/class_quant_lib_1_1_f_d_european_engine.html
- BinomialTian
    - Pricing engine for vanilla options using binomial trees. Tian model.
    - http://quantlib.org/reference/class_quant_lib_1_1_f_d_european_engine.html
- BinomialLeisenReimer
    - Pricing engine for vanilla options using binomial trees. Leisen-Reimer model.
    - http://quantlib.org/reference/class_quant_lib_1_1_f_d_european_engine.html
-
    - Pricing engine for vanilla options using binomial trees. Joshi model.
    - http://quantlib.org/reference/class_quant_lib_1_1_f_d_european_engine.html
