LEAN supports the following Option price models. QLNet provides the underlying implementation of these models.

  • AdditiveEquiprobabilities
  • BaroneAdesiWhaley
  • BinomialCoxRossRubinstein
  • BinomialJarrowRudd
  • BinomialJoshi
  • BinomialLeisenReimer
  • BinomialTian
  • BinomialTrigeorgis
  • BjerksundStensland
  • BlackScholes
  • CrankNicolsonFD
  • Integral

The BlackScholes and Integral models currently return zero-value Greeks. For more information about these models, see GitHub Issue #4520.
