The WolverineFeeModel models the fees of Wolverine Execution Services. It's the default fee model of the WolverineBrokerageModel.

security.SetFeeModel(new WolverineFeeModel());
security.SetFeeModel(WolverineFeeModel())

The WolverineFeeModel charges fees in USD.

To view the implementation of this model, see the LEAN GitHub repository.
