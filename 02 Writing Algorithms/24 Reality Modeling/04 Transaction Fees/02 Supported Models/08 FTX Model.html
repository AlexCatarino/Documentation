The FTXFeeModel models the fees of FTX. It's the default fee model of the FTXBrokerageModel.

security.SetFeeModel(new FTXFeeModel());
security.SetFeeModel(FTXFeeModel())

The FTXFeeModel charges fees in the quote currency by default. It charges fees in the base currency if your order is a buy limit order where the PostOnly order property is enabled or the limit price is marketable.

To view the implementation of this model, see the LEAN GitHub repository.
