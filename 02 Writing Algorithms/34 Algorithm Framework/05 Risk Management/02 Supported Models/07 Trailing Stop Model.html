The TrailingStopRiskManagementModel monitors the drawdown of each security in the portfolio. When the drawdown exceeds a threshold (relative to the trailing max price since the position was opened), it liquidates the position.

- This model can operate even when the PCM provides an empty list of PortfolioTarget objects

The model accepts the following arguments:
-maximumDrawdownPercent: The maximum percentage drawdown allowed for algorithm portfolio compared with the highest unrealized profit; default 0.05 (5%) 

AddRiskManagement( new TrailingStopRiskManagementModel(maximumDrawdownPercent) );
self.AddRiskManagement( TrailingStopRiskManagementModel(maximum_drawdown_percent) )
