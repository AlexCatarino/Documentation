The MaximumDrawdownPercentPortfolio model monitors the portfolio drawdown. The drawdown can be relative to the starting portfolio value or the maximum portfolio value. When the portfolio value drops below a percentage threshold, the model liquidates the portfolio. After the portfolio is liquidated, the model resets.



The model accepts the following arguments:
-
-
- 
AddRiskManagement( new MaximumDrawdownPercentPerSecurity() );
self.AddRiskManagement( MaximumDrawdownPercentPerSecurity() )
