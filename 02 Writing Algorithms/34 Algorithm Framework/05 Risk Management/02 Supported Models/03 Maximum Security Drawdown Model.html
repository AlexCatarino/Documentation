The MaximumDrawdownPercentPerSecurity model monitors the unrealized profit percentage of each security in the portfolio. When the percentage drops below a threshold (relative to the opening price), it liquidates the position.

The model accepts the following arguments:
-maximumDrawdownPercent: The maximum percentage drawdown allowed for any single security holding; Default: 0.05 (5%) 

AddRiskManagement( new MaximumDrawdownPercentPerSecurity(maximumDrawdownPercent) );
self.AddRiskManagement( MaximumDrawdownPercentPerSecurity(maximum_drawdown_percent) )
