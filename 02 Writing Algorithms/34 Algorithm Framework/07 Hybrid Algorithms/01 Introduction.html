What is a hybrid algorithm?
- An algorithm that has the following characteristics:
    - uses the NullAlphaModel as the only Alpha model
    - Emits insights with the EmitInsights method
    - Has some other non-null framework model(s) set to execute/manage trades

Why use a hybrid algorithm:
- Some algorithms naturally violate the seperation of concerns design principle that the framework was designed for. In these cases, it's best to create a classic/hybrid algorithm
- If your classic algorithm has the NullPortfolioConstructionModel and
- If you want to use a risk management model with the default Alpha (NullAlphaModel) and PCM (NullPortfolioConstructionModel)
