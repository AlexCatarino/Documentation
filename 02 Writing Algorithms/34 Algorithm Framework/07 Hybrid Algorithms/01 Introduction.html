What is a hybrid algorithm?
- An algorithm that adds framework models to a classic algorithm.

- For example, some risk management models don't require the PCM to provide PortfolioTarget objects. If you add one of these risk management models to a classic algorithm, it's a hybrid algorithm. To see which Risk Management models don't need on the PCM to return PortfolioTarget objects, see Supported Models.

- Another example: Using Universe Selection model in your classic algorithm instead of just adding a user-defined Universe.

- Another example: You can add a PCM to your classic algorithm and use the EmitInsights method instead of placing manual orders with MarketOrder


Why use a hybrid algorithm:
- Some algorithms naturally violate the seperation of concerns design principle that the framework was designed for. In these cases, it's best to create a classic/hybrid algorithm
