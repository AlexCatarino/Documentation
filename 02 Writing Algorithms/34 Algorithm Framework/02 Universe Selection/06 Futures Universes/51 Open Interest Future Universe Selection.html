The OpenInterestFutureUniverseSelectionModel selects the Future contracts with the greatest open interest. The constructor accepts the following arguments:

-algorithm: Your QCAlgorithm object
-futureChainSymbolSelector: A function that receives a datetime and returns a list of Future Symbol objects. Selects symbols from the provided future chain
-(optional) chainContractsLookupLimit : Limit on how many contracts to query for open interest (cuts off oldest expiring contracts) ; default 6
-(optional) resultsLimit : Limit on how many contracts will be part of the universe 
public override void Initialize()
{
    var symbols = new[] {
        QuantConnect.Symbol.Create(Futures.Indices.SP500EMini, SecurityType.Future, Market.CME),
        QuantConnect.Symbol.Create(Futures.Metals.Gold, SecurityType.Future, Market.COMEX)
    }; 
    AddUniverseSelection(new OpenInterestFutureUniverseSelectionModel(this, dateTime => symbols));
}
