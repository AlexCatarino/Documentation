The UncorrelatedUniverseSelectionModel is a fundamental universe selection model for US Equities that selects assets that that currently have their correlation to a benchmark deviated from the mean. The model accepts the following arguments:
benchmark: Symbol of the benchmark; default Symbol.Create("SPY", SecurityType.Equity, Market.USA)
numberOfSymbolsCoarse: Number of coarse symbols; default 400
numberOfSymbols: Number of symbols selected by the universe model; default 10
windowLength: Rolling window length period for correlation calculation; default 5
historyLength: History length period; default 25
threshold: Threadhold for the minimum mean correlation between security and benchmark; default .05
