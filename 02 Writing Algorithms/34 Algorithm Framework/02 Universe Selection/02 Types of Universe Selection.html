The following table describes the types of universe selection models:

  • Manual Universes - Universes which use a fixed, static set of assets.
  • Fundamental Universes - Universes based on coarse price or fundamental data.
  • Scheduled Universes - Universes triggering on regular intervals.
  • Inception Date Universes - Universe that accepts a Dictionary of DateTime keyed by String that represent the Inception date for each ticker
