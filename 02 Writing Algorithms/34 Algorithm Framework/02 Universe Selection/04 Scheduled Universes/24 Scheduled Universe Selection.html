The universe selection helper works in the same way as the Scheduled Event API requiring a $[DateRule,T:QuantConnect.Scheduling.IDateRule], a $[TimeRule,T:QuantConnect.Scheduling.ITimeRule] to set the callback times, and a function to execute. The function receives a datetime object and should return a list of Symbol objects. The returning list of Symbol objects are the constituents of the universe. 

// Selection will run on mon/tues/thurs at 00:00/06:00/12:00/18:00
AddUniverseSelection(new ScheduledUniverseSelectionModel(
    DateRules.Every(DayOfWeek.Monday, DayOfWeek.Tuesday, DayOfWeek.Thursday),
    TimeRules.Every(TimeSpan.FromHours(6)),
    SelectSymbols // selection function in algorithm.
));
