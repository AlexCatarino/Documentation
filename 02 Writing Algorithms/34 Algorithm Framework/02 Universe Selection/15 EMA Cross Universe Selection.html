The EmaCrossUniverseSelectionModel is a fundamental universe selection model for US Equities. The model accepts the following arguments:
fastPeriod: Fast EMA period; Default 100
slowPeriod: Slow EMA period; Default 300
universeCount: Maximum number of members of this universe selection; Default 500
universeSettings: The settings used when adding symbols to the algorithm, specify null to use algorthm.UniverseSettings; Default: None

The model selects symbols with the larger delta by percentage between the two exponential moving averages. 
public override void Initialize()
{
    AddUniverseSelection(new EmaCrossUniverseSelectionModel());
}
