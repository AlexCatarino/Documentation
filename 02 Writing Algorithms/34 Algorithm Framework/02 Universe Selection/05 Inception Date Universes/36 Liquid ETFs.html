The LiquidETFUniverse is an inception date universe selection model that adds ETFs for the following categories at their inception date:
-Energy
-Metals
-Technology
-Treasuries
-Volatility
-S&P 500 Sectors 
public override void Initialize()
{
    AddUniverseSelection(new LiquidETFUniverse());
}
def Initialize(self):
    self.AddUniverseSelection(LiquidETFUniverse())

To view the implementation of this model, see the LEAN GitHub repository.
