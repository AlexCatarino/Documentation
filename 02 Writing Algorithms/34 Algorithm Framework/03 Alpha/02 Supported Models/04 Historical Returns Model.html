The HistoricalReturnsAlphaModel buys securities that have a positive trailing return and sells securities that have a negative trailing return. It sets the magnitude of the Insights to the the trailing rate of change.

The model accepts the following arguments:
-lookback: Historical return lookback period; default 1
-resolution: The resolution of historical data; default Resolution.Daily
// In Initialize
AddAlpha(HistoricalReturnsAlphaModel(lookback, resolution));
# In Initialize
self.AddAlpha(HistoricalReturnsAlphaModel(lookback=60, resolution=Resolution.Daily))
