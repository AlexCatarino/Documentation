The RsiAlphaModel generates insights based on the relative strength indicator values. The model uses the Wilder moving average type to create the RSI. When the RSI value passes above 70, the model emits a down insight. When the RSI value passes below 30, the model emits an up insight. The duration of the insight is resolution * period

The model accepts the following arguments:
-period: The RSI indicator period; default 14
-Resolution: resolution of the data; default Resolution.Daily 
// In Initialize
AddAlpha(new RsiAlphaModel(period, resolution));
# In Initialize
self.AddAlpha(RsiAlphaModel(period, resolution))
