The PearsonCorrelationPairsTradingAlphaModel ranks every pair combination by its pearson correlation and trades the pair with the highest correlation. This model follows the same insight logic as the BasePairsTradingModel.

The model accepts the following arguments:
-lookback: lookback period of the analysis; default 15
-resolution: analysis resolution; default Resolution.Minute
-threshold: The percent [0, 100] deviation of the ratio from the mean before emitting an insight; Default 1
-minimumCorrelation: The minimum correlation to consider a tradable pair; default 0.5 
AddAlpha(new PearsonCorrelationPairsTradingAlphaModel(lookback, resolution, threshold, minimumCorrelation));
self.AddAlpha(PearsonCorrelationPairsTradingAlphaModel(lookback, resolution, threshold, minimum_correlation))
