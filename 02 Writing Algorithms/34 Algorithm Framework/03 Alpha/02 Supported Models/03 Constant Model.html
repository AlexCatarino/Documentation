The ConstantAlphaModel always returns the same insight for each security.

The model accepts the following arguments:
-type: The type of insight
-direction: The direction of the insight
-period: The period over which the insight with come to fruition
-magnitude: The predicted change in magnitude as a +- percentage; Default None
-confidence: The confidence in the insight; Default None 
// In Initialize
AddAlpha(new ConstantAlphaModel(type, direction, period, magnitude, confidence));
# In Initialize
self.AddAlpha(ConstantAlphaModel(type, direction, period, magnitude, confidence))
