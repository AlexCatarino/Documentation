The MacdAlphaModel emits insights based on MACD crossovers. The MACD signal line is used to generate up/down insights. If the MACD signal is 1% above the security price, the model emits an up insight. If the MACD signal is 1% below the security price, the model emits a down insight. If the MACD signal is within 1% of the security price, the model emits a flat insight.

The model accepts the following arguments:
-fastPeriod: The MACD fast period; default 12
-slowPeriod: The MACD slow period; default 26
-signalPeriod: The smoothing period for the MACD signal; default 9
-movingAverageType: The type of moving average to use in the MACD; default MovingAverateType.Exponential
-resolution: The resolution of the data; Default Resolution.Daily
