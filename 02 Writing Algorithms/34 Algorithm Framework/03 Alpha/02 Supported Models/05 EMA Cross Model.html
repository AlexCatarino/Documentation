The EmaCrossAlphaModel uses an EMA cross to create insights. When the slow EMA crosses above the fast EMA, it buys. When the slow EMA crosses below the fast EMA, it sells. It sets the duration of insights to be resolution * fastPeriod.

The model accepts the following arguments:
-fastPeriod: The fast EMA period; Default 12
-slowPeriod: The slow EMA period; Default 26
-resolution: Resolution of the data; default Resolution.Daily

AddAlpha(new EmaCrossAlphaModel(fastPeriod, slowPeriod, resolution));
self.AddAlpha(EmaCrossAlphaModel(fast_period, slow_period, resolution))

The following table describes the arguments the model accepts:

Argument Data Type Description Default Value
maximumDrawdownPercent decimal The maximum percentage drawdown allowed for any single security holding 0.05 (5%)
