The $[NullPortfolioConstructionModel,T:QuantConnect.Algorithm.Framework.Portfolio.NullPortfolioConstructionModel] can be used to skip the execution phase of the algorithm, i.e. do nothing. It's the default portfolio construction model. This model is useful when you're trying to analyze the Alpha Model in isolation. 

SetPortfolioConstruction( new NullPortfolioConstructionModel() ); 
self.SetPortfolioConstruction( NullPortfolioConstructionModel() )
