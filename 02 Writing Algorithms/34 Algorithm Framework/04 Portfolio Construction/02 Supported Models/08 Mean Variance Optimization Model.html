The Mean Variance Portfolio Construction Model is an implementation of the classical model. It seeks to build a portfolio with the minimum volatility possible.

The model accepts the following arguments:
-
-
- 
SetPortfolioConstruction(new MeanVarianceOptimizationPortfolioConstructionModel(rebalance, portfolioBias, lookback, period, resolution, targetReturn, optimizer));
self.SetPortfolioConstruction(MeanVarianceOptimizationPortfolioConstructionModel(rebalance, portfolio_bias, lookback, period, resolution, target_return, optimizer))
