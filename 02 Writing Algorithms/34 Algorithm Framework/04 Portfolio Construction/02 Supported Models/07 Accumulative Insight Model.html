The AccumulativeInsightPortfolioConstructionModel generates target portfolio weights based on all the active insights of a security. For each active Insight of direction InsightDirection.Up, it increases the position size by a fixed percent. For each active Insights of direciton InsightDirection.Down, it decreases the position size by a fixed percent. For each active of direction InsightDirection.Flat, it moves the position size towards 0 by a fixed percent.

The model accepts the following arguments:
-rebalance: Rebalancing parameter. If it is a timedelta, date rules or Resolution, it will be converted into a function. If None will be ignored. The function returns the next expected rebalance time for a given algorithm UTC DateTime. The function returns null if unknown, in which case the function will be called again in the next loop. Returning current time will trigger rebalance.; Default None
- portfolioBias: Specifies the bias of the portfolio (Short, Long/Short, Long); default PortfolioBias.LongShort
- percent: percent of portfolio to allocate to each position; default 0.03 (3%)

SetPortfolioConstruction(new AccumulativeInsightPortfolioConstructionModel());
self.SetPortfolioConstruction(AccumulativeInsightPortfolioConstructionModel())
