The AccumulativeInsightPortfolioConstructionModel generates target portfolio weights based on all the active insights of a security. For each active Insight of direction InsightDirection.Up, it increases the position size by a fixed percent. For each active Insights of direction InsightDirection.Down, it decreases the position size by a fixed percent. For each active of direction InsightDirection.Flat, it moves the position size towards 0 by a fixed percent.

SetPortfolioConstruction(new AccumulativeInsightPortfolioConstructionModel());
self.SetPortfolioConstruction(AccumulativeInsightPortfolioConstructionModel())
