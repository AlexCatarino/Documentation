The MaximumSharpeRatioPortfolioOptimizer is an optimizer that maximizes the portfolio Sharpe Ratio.

The optimizer accepts the following arguments:
-minimum_weight(float): The lower bounds on portfolio weights; default -1
-maximum_weight(float): The upper bounds on portfolio weights; default 1
-risk_free_rate(float): The risk free rate; default 0 

var optimizer = new MaximumSharpeRatioPortfolioOptimizer();
optimizer = MaximumSharpeRatioPortfolioOptimizer()
