-Some portfolio construction models contain an optimizer that accepts the historical returns and return a list of optimized portfolio weights. Portfolio optimizer models implement the IPortfolioOptimizer interface. This interface has one method, Optimize.

The Optimize method accepts the following arguments:
-historicalReturns: Matrix of annualized historical returns where each column represents a security and each row returns for the given date/time (size: K x N)
-(optional) expectedReturns: Array of double with the portfolio annualized expected returns (size: K x 1).
-(optional) covariance: Multi-dimensional array of double with the portfolio covariance of annualized returns (size: K x K)

The optimize method returns an array of double with the portfolio weights (size: K x 1)
