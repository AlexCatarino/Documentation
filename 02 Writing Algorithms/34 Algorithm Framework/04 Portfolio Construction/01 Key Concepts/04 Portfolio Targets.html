The portfolio construction model returns PortfolioTarget objects, which are passed to the risk management model.

To create PortfolioTarget objects based on a quantity, pass the Symbol and quantity to the PortfolioTarget constructor.

// Create a new portfolio target for 1200 IBM shares.
var target = new PortfolioTarget("IBM", 1200);
# Create a new portfolio target for 1200 IBM shares.
target = PortfolioTarget("IBM", 1200)

If you have a margin account, you can use the $[Percent(algorithm, Symbol, percent),M:QuantConnect.Algorithm.Framework.Portfolio.PortfolioTarget.Percent] helper method to create PortfolioTarget objects based on a portfolio weight. 

// Calculate target equivalent to 10% of portfolio value
var target = PortfolioTarget.Percent(algorithm, "IBM", 0.1);
# Calculate target equivalent to 10% of portfolio value
target = PortfolioTarget.Percent(algorithm, "IBM", 0.1)

Your Portfolio Construction Model should return an array of PortfolioTarget objects from its  $[CreateTargets,M:QuantConnect.Algorithm.Framework.Portfolio.IPortfolioConstructionModel.CreateTargets] method. 

// Return an array of targets
return new PortfolioTarget[] {  new PortfolioTarget("IBM", 1200)  };
# Return an array of targets
return [ PortfolioTarget("IBM", 1200) ]
