You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets the historical data for SEC Fillings data by using the History API.

report8K_history = self.History(SECReport8K, self.report8K, 60, Resolution.Daily)
report10K_history = self.History(SECReport10K, self.report10K, 60, Resolution.Daily)
report10Q_history = self.History(SECReport10Q, self.report10Q, 60, Resolution.Daily)

var report8KHistory = History<SECReport8K>(_report8K, 60, Resolution.Daily);
var report10KHistory = History<SECReport10K>(_report10K, 60, Resolution.Daily);
var report10QHistory = History<SECReport10Q>(_report10Q, 60, Resolution.Daily);
