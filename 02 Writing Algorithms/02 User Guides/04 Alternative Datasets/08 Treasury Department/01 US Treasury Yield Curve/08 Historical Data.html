You can request historical custom data in your algorithm using the custom data Symbol object. To learn more about historical data requests, please visit the Historical Data<\/a> documentation. If there is no custom data in the period you request, the history result will be empty. The following example gets the historical data for the US Treasury Yield Curve by using the History API.<\/p> 

history = self.History(USTreasuryYieldCurveRate, yield_curve_symbol, 60, Resolution.Daily)<\/code><\/pre>
