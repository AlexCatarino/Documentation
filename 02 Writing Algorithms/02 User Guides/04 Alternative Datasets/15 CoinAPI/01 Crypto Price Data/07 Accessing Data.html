Data can be accessed via Slice events. Slice delivers unique events to your algorithm as they happen. We recommend saving the symbol object when you add the data for easy access to slice later. Data is available in every resolution. You can see an example of the slice accessor in the code below.

def OnData(self, data):
    bitfinex_btc_price = data[self.bitfinex_btcusd].Price
    coinbase_btc_price = data[self.coinbase_btcusd].Price
    self.Log(f”Time: {self.Time}; Bitfinex price: {bitfinex_btc_price}; Coinbase price: {coinbase_btc_price}”)
public override void OnData(Slice data)
{
    var bitfinexBtcPrice = data[_bitfinexBtcusd].Price;
    var coinbaseBtcPrice = data[_coinbaseBtcusd].Price;
    Log($”Time: {Time}; Bitfinex price: {bitfinexBtcPrice}; Coinbase price: {coinbaseBtcPrice}”);
}
