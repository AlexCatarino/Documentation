You can request historical data in your algorithm using the data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no data in the period you request, the history result will be empty. The following example gets the historical data for CoinAPI by using the History API.

bitfinex_history = self.History(self.bitfinex_btcusd, 30, Resolution.Daily)
coinbase_history = self.History(self.coinbase_btcusd, 30, Resolution.Daily)
var bitfinexHistory = History(_bitfinexBtcusd, 30, Resolution.Daily);
var coinbaseHistory = History(_coinbaseBtcusd, 30, Resolution.Daily);
