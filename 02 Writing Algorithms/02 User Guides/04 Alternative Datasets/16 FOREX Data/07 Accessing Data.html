Quote data can be accessed via Slice events. Slice delivers unique events to your algorithm as they happen. We recommend saving the symbol object when you add the data for easy access to slice later. Quote data is available in daily down to tick resolution. You can see an example of the slice accessor in the code below.

class ForexAlgorithm(QCAlgorithm):
    def OnData(self, data):
        # The marketData object can be a QuoteBar or a List of Tick
        marketData = data[self.symbol]

        # If it's QuoteBar:
        self.Log(f'Symbol: {self.symbol} - Data: {marketData}')

        # If it's Tick data:
        for tick in marketData:
            self.Log(f"Time: {tick.EndTime}; Symbol: {self.symbol}; Tick: {tick}")

namespace QuantConnect.Algorithm.CSharp
{
    public class ForexAlgorithm : QCAlgorithm
    {
        public override void OnData(Slice data)
        {
            // The marketData object can be a QuoteBar or a List of Tick
            var marketData = data[_symbol];

            // If it's QuoteBar:
            Log($"Symbol: {_symbol} - Data: {marketData}");

            // If it's Tick data:
            foreach (var tick in marketData)
            {
                 Log($"Time: {tick.EndTime}; Symbol: {_symbol} Tick: {tick}");
            }
        }
    }
}
