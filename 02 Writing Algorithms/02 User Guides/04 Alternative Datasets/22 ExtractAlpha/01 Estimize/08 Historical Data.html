You can request the alternative dataset’s historical data by using the data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no data in the period you request, the history result will be empty. The following example gets the alternative dataset’s historical data by using the History API.

consensus_history = self.History(estimize_consensus_symbol, 10, Resolution.Daily)
estimate_history = self.History(estimize_estimate_symbol, 10, Resolution.Daily)
release_history = self.History(estimize_release_symbol, 10, Resolution.Daily)

history = self.History([estimize_consensus_symbol,
                        estimize_estimate_symbol,
                        estimize_release_symbol],
                       10,
                       Resolution.Daily)

var consensusHistory = History(new[]{estimizeConsensusSymbol}, 10, Resolution.Daily);
var estimateHistory = History(new[]{estimizeEstimateSymbol}, 10, Resolution.Daily);
var releaseHistory = History(new[]{estimizeReleaseSymbol}, 10, Resolution.Daily);

var history = History(new[]{estimizeConsensusSymbol,
                            estimizeEstimateSymbol,
                            estimizeReleaseSymbol},
                           10,
                           Resolution.Daily)
