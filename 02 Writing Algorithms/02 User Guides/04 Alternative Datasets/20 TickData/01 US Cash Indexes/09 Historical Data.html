You can request historical US Cash Indexes data by using the data Symbol object. To learn more about historical data requests, please visit the Historical Data documentation. If there is no data in the period you request, the history result will be empty. The following example gets the historical data for US Equity Indexes by using the History API.

history = self.History(self.vix, 10, Resolution.Minute)
var history = History(_vix, 10, Resolution.Minute);
