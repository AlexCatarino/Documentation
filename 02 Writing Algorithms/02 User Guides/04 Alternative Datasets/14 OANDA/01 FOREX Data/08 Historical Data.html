You can request historical quote data in your algorithm using the FOREX price data Symbol object. To learn more about historical data requests, please visit the Historical Data<\/a> documentation. If there is no FOREX price data in the period you request, the history result will be empty. The following example gets the historical data for FOREX price data by using the History API.<\/p> 

history = self.History(self.symbol, 60, Resolution.Daily)<\/code><\/pre>

var history = History(_symbol, 60, Resolution.Daily);<\/code><\/pre>
