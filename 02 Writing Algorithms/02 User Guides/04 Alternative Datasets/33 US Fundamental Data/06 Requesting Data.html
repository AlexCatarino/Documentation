To add US Fundamental data to your algorithm, you need to provide a fine universe selection method to AddUniverse() or AddUniverseSelection(). For detailed documentation on using fundamental data, see Fundamentals Selection.

class MorningStarDataAlgorithm(QCAlgorithm):

    def Initialize(self):
        self.SetStartDate(2021, 1, 1)
        self.SetEndDate(2021, 7, 1)
        self.SetCash(100000) 
        
        self.AddUniverse(self.SelectCoarse, self.SelectFine)
