To add Bitcoin Metadata dataset to your algorithm, use the AddData() method to request the data. Note that the data has to be accessed by the symbol “BTCUSD”. As with all datasets, you should save a reference to your symbol for easy use later in your algorithm. For detailed documentation on using custom data, see Importing Custom Data<\/a>.<\/p>

{
    public class BlockchainBitcoinMetadataAlgorithm: QCAlgorithm

        {
            SetStartDate(2019, 1, 1);
            SetEndDate(2020, 6, 1);
            SetCash(100000);

            var btcusd = AddCrypto("BTCUSD", Resolution.Daily, Market.Bitfinex).Symbol;
