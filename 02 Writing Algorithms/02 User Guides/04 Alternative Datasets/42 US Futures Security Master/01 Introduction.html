US Futures Security Master by QuantConnect provides mapping reference data for the most liquid contracts of the CME Group exchanges, calculated with popular rolling techniques. The data covers 75 root Future contracts, starting in 2012, and is delivered on a daily frequency with a zip file with all the contract mappings. This dataset is created by daily processing of the US historical Future chains.

Using this data source paired with raw Futures data, multiple rolling techniques are supported including; ForwardPanamaCanal, BackwardsPanamaCanal, and Backwards Ratio. The specific date of rolling can also be configured with options including; LastTradingDay, FirstDayMonth, or the greatest OpenInterest.
