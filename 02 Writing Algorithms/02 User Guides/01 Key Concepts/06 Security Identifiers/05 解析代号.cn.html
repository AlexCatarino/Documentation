当Symbol被序列化为字符串时，它将如下所示：SPY R735QTJ8XC9X。这个由两部分组成的字符串是base64编码的数据集。将所有属性编码为短格式允许密集通信，而无需第三方列表或查找。

大多数情况下，您不需要使用这些编码的字符串。但是，QuantConnect提供了一种方法，用于将Symbol对象反序列化为易于使用的对象以供API使用。您可以使用此方法，如下所示： 

# Python 
google = self.Symbol("GOOCV VP83T1ZUHROL")
print(google.Market)                                # USA （美国）
print(google.SecurityType)                          # Equity （证券）
print(google.Symbol)                                # GOOCV （代号）

# CSharp
var google = Symbol("GOOCV VP83T1ZUHROL");
google.Market                       # Market.USA （美国）
google.SecurityType                 # SecurityType.Equity （证券）
google.Symbol                       # GOOCV （代号）

Market属性用于区分代表不同基础资产但拥有相同字符串值的代码。一个典型的例子是各种做市商对欧元兑美元的价格不同。QuantConnect分别存储这些数据，由于它们具有不同的成交价，我们将执行场所视为不同的市场。
