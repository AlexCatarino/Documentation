标准差执行模型试图在价格比给定时期的正常股票价格低2个标准差时成交订单。目的是在市场中逢低进行交易。不幸的是，在市场走势强劲的情况下，这可能导致交易延迟，因为在下一次价格下跌前可能需要一段时间。

要在算法中使用预先制定的执行模型，您应该在Initialize()中设置它： 

SetExecution( new StandardDeviationExecutionModel() );
self.SetExecution( StandardDeviationExecutionModel() )

这个模型有以下可选参数： 

StandardDeviationExecutionModel( deviations = 2, period = 60, resolution=Resolution.Minute )

  • 偏差 - 交易前与平均值的最小偏差。
  • 周期 - 标准差指标的周期。
  • 分辨率 - 标准差指标的分辨率。
