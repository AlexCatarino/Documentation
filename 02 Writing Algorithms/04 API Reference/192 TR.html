TR

TR(
 Symbol symbol,
 Resolution resolution,

Creates a new TrueRange indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose TR we want.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

TrueRange - The TrueRange indicator for the requested symbol.
