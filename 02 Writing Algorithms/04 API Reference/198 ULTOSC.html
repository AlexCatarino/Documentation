ULTOSC()

ULTOSC(
                                           Symbol symbol,
                                                  Int32 period1,
                                                  Int32 period2,
                                                  Int32 period3,
                                                  Resolution resolution,
                                                  IBaseDataBar selector
                                                  )

Creates a new UltimateOscillator indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose ULTOSC we want.
Int32period1The first period over which to compute the ULTOSC.
Int32period2The second period over which to compute the ULTOSC.
Int32period3The third period over which to compute the ULTOSC.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

UltimateOscillator - The UltimateOscillator indicator for the requested symbol over the specified period
