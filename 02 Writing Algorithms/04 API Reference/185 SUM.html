SUM

SUM(
 Symbol symbol,
 Int32 period,
 Resolution resolution,
 Decimal selector
)

Creates a new Sum indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Sum we want.
Int32periodThe period over which to compute the Sum.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

Sum - The Sum indicator for the requested symbol over the specified period
