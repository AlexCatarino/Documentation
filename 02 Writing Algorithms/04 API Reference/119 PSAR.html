PSAR

PSAR(
 Symbol symbol,
 Decimal afStart,
 Decimal afIncrement,
 Decimal afMax,
 Resolution resolution,
 IBaseDataBar selector
)

Creates a new Parabolic SAR indicator

Parameters

Type Name Description
SymbolsymbolThe symbol whose PSAR we seek.
DecimalafStartAcceleration factor start value. Normally 0.02.
DecimalafIncrementAcceleration factor increment value. Normally 0.02.
DecimalafMaxAcceleration factor max value. Normally 0.2.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

ParabolicStopAndReverse - A ParabolicStopAndReverse configured with the specified periods
