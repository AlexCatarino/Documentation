ATR()

ATR(
                                              Symbol symbol,
                                                  Int32 period,
                                                  MovingAverageType type,
                                                  Resolution resolution,
                                                  IBaseDataBar selector
                                                  )

Creates a new AverageTrueRange indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose ATR we want.
Int32periodThe smoothing period used to smooth the computed TrueRange values.
MovingAverageTypetypeThe type of smoothing to use.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

AverageTrueRange - A new AverageTrueRange indicator with the specified smoothing type and period
