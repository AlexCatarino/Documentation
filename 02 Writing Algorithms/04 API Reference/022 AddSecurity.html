ResolutionresolutionResolution of the MarketType required: MarketData, Second or Minute.
StringmarketThe market the requested security belongs to, such as 'usa' or 'fxcm'.
BooleanfillDataForwardIf True, returns the last available data even if none in that timeslice.
Decimalleverageleverage for this security.
BooleanextendedMarketHoursExtendedMarketHours send in data from 4am - 8pm, not used for FOREX.
DataMappingModedataMappingModeThe contract mapping mode to use for the security.
DataNormalizationModedataNormalizationModeThe price scaling mode to use for the security.

Set a required SecurityType-symbol and resolution for algorithm

Parameters

Type Name Description
SymbolsymbolThe security Symbol.
ResolutionresolutionResolution of the MarketType required: MarketData, Second or Minute.
BooleanfillDataForwardIf True, returns the last available data even if none in that timeslice.
Decimalleverageleverage for this security.
BooleanextendedMarketHoursExtendedMarketHours send in data from 4am - 8pm, not used for FOREX.
DataMappingModedataMappingModeThe contract mapping mode to use for the security.
DataNormalizationModedataNormalizationModeThe price scaling mode to use for the security.
Int32contractDepthOffsetThe continuous contract desired offset from the current front month. For example, 0 (default) will use the front month, 1 will use the back month contract.

Returns
