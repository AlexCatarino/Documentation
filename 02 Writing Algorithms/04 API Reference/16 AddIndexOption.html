AddIndexOption

AddIndexOption(
 String ticker,
 Resolution resolution,
 String market,
 Boolean fillDataForward
)

Creates and adds index options to the algorithm.

Parameters

Type Name Description
StringtickerThe ticker of the Index Option.
ResolutionresolutionResolution of the index option contracts, i.e. the granularity of the data.
StringmarketMarket of the index option. If no market is provided, we default to USA.
BooleanfillDataForwardIf True, this will fill in missing data points with the previous data point.

Returns

Option - Canonical Option security
