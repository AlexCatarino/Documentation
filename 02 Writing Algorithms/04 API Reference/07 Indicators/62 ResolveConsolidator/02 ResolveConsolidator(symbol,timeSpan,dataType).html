ResolveConsolidator(symbol,timeSpan,dataType)

Gets the default consolidator for the specified symbol and resolution

Parameters

Type Name Description
SymbolsymbolThe symbol whose data is to be consolidated.
TimeSpantimeSpanThe requested time span for the consolidator, if None, uses the resolution from subscription.
TypedataTypeThe data type for this consolidator, if None, uses TradeBar over QuoteBar if present.

Returns

IDataConsolidator- The new default consolidator
