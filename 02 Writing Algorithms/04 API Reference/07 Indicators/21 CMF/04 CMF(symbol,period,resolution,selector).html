CMF(symbol,period,resolution,selector)

Creates a new ChaikinMoneyFlow indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose CMF we want.
Int32periodThe period over which to compute the CMF.
ResolutionresolutionThe resolution.
TradeBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

ChaikinMoneyFlow - The ChaikinMoneyFlow indicator for the requested symbol over the specified period
