self.MASS(symbol,emaPeriod,sumPeriod,resolution,selector)

Creates a new Mass Index indicator. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Mass Index we want.
Int32emaPeriodThe period used by both EMA.
Int32sumPeriodThe sum period.
ResolutionresolutionThe resolution.
TradeBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

MassIndex - The Mass Index indicator for the requested symbol over the specified period
