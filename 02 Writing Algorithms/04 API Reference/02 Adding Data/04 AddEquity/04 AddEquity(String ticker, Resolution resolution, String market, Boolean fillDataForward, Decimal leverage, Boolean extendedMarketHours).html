AddEquity(String ticker, Resolution resolution, String market, Boolean fillDataForward, Decimal leverage, Boolean extendedMarketHours)

Creates and adds a new Equity security to the algorithm

Parameters

Type Name Description
StringtickerThe equity ticker symbol.
ResolutionresolutionThe Resolution of market data, Tick, Second, Minute, Hour, or Daily. Default is Minute.
StringmarketThe equity's market, Market. Default value is None and looked up using BrokerageModel.DefaultMarkets in AddSecurity``1.
BooleanfillDataForwardIf True, returns the last available data even if none in that timeslice. Default is True.
DecimalleverageThe requested leverage for this equity. Default is set by SecurityInitializer.
BooleanextendedMarketHoursTrue to send data during pre and post market sessions. Default is False.

Returns

Equity - The new security
