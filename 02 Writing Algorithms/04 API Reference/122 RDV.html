RDV

RDV(
 Symbol symbol,
 Int32 period,
 Resolution resolution,
 TradeBar selector
)

Creates an RelativeDailyVolume indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose RDV we want.
Int32periodThe period of the RDV.
ResolutionresolutionThe resolution.
TradeBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

RelativeDailyVolume - The Relative Volume indicator for the given parameters
