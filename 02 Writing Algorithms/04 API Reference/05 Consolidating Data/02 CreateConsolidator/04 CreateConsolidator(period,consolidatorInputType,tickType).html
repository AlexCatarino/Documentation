CreateConsolidator(period,consolidatorInputType,tickType)

Creates a new consolidator for the specified period, generating the requested output type.

Parameters

Type Name Description
TimeSpanperiodThe consolidation period.
TypeconsolidatorInputTypeThe desired input type of the consolidator, such as TradeBar or QuoteBar.
TickTypetickTypeTrade or Quote. Optional, defaults to trade.

Returns

IDataConsolidator- A new consolidator matching the requested parameters
