AddSeries()

AddSeries(
                                        String chart,
                                                  String series,
                                                  SeriesType seriesType,
                                                  String unit
                                                  )

Add a series object for charting. This is useful when initializing charts with series other than type = line. If a series exists in the chart with the same name, then it is replaced.

Parameters

Type Name Description
StringchartThe chart name.
StringseriesThe series name.
SeriesTypeseriesTypeThe type of series, i.e, Scatter.Options: ['Line', 'Scatter', 'Candle', 'Bar', 'Flag', 'StackedArea', 'Pie', 'Treemap']
StringunitThe unit of the y axis, usually $.

Returns

Void
