SWISS

SWISS(
 Symbol symbol,
 Int32 period,
 Double delta,
 SwissArmyKnifeTool tool,
 Resolution resolution,
 Decimal selector
)

Creates Swiss Army Knife transformation for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol to use for calculations.
Int32periodThe period of the calculation.
DoubledeltaThe delta scale of the BandStop or BandPass.
SwissArmyKnifeTooltoolThe tool os the Swiss Army Knife.Options: ['Gauss', 'Butter', 'HighPass', 'TwoPoleHighPass', 'BandPass']
ResolutionresolutionThe resolution.
Decimalselectorelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

SwissArmyKnife - The calculation using the given tool
