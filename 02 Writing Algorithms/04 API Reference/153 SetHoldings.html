SetHoldings

SetHoldings(
 PortfolioTarget PortfolioTarget,
 Boolean liquidateExistingHoldings,
 String tag,
 IOrderProperties orderProperties
)

Sets holdings for a collection of targets. The implementation will order the provided targets executing first those that reduce a position, freeing margin.

Parameters

Type Name Description
PortfolioTargetPortfolioTargetProvides an implementation of IPortfolioTarget that specifies a specified quantity of a security to be held by the algorithm.
BooleanliquidateExistingHoldingsTrue will liquidate existing holdings.
StringtagTag the order with a short string.
IOrderPropertiesorderPropertiesThe order properties to use. Defaults to DefaultOrderProperties.

Returns

Void

SetHoldings

SetHoldings(
 Symbol symbol,
 Decimal percentage,
 Boolean liquidateExistingHoldings,
 String tag,
 IOrderProperties orderProperties
)

Automatically place a market order which will set the holdings to between 100% or -100% of *PORTFOLIO VALUE*. E.g. SetHoldings("AAPL", 0.1); SetHoldings("IBM", -0.2); -> Sets portfolio as long 10% APPL and short 20% IBM E.g. SetHoldings("AAPL", 2); -> Sets apple to 2x leveraged with all our cash. If the market is closed, place a market on open order.

Parameters

Type Name Description
SymbolsymbolSymbol indexer.
Decimalpercentagedecimal fraction of portfolio to set stock.
BooleanliquidateExistingHoldingsbool flag to clean all existing holdings before setting new faction.
StringtagTag the order with a short string.
IOrderPropertiesorderPropertiesThe order properties to use. Defaults to DefaultOrderProperties.

Returns

Void
