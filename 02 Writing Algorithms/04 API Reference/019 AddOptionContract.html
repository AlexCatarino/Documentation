AddOptionContract()

AddOptionContract(                                Symbol symbol,
                                                  Resolution resolution,
                                                  Boolean fillDataForward,
                                                  Decimal leverage
                                                  )

Creates and adds a new single Option contract to the algorithm

Parameters

Type Name Description
SymbolsymbolThe option contract symbol.
ResolutionresolutionThe Resolution of market data, Tick, Second, Minute, Hour, or Daily. Default is Minute.
BooleanfillDataForwardIf True, returns the last available data even if none in that timeslice. Default is True.
DecimalleverageThe requested leverage for this equity. Default is set by SecurityInitializer.

Returns

Option - The new security
