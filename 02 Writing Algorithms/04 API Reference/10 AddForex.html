AddForex

AddForex(
 String ticker,
 Resolution resolution,
 String market,
 Boolean fillDataForward,
 Decimal leverage
)

Creates and adds a new Forex security to the algorithm

Parameters

Type Name Description
StringtickerThe currency pair.
ResolutionresolutionThe Resolution of market data, Tick, Second, Minute, Hour, or Daily. Default is Minute.
StringmarketThe foreign exchange trading market, Market. Default value is None and looked up using BrokerageModel.DefaultMarkets in AddSecurity``1.
BooleanfillDataForwardIf True, returns the last available data even if none in that timeslice. Default is True.
DecimalleverageThe requested leverage for this equity. Default is set by SecurityInitializer.

Returns

Forex - The new security
