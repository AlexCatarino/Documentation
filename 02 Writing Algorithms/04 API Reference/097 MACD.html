MACD()

MACD(                                             Symbol symbol,
                                                  Int32 fastPeriod,
                                                  Int32 slowPeriod,
                                                  Int32 signalPeriod,
                                                  MovingAverageType type,
                                                  Resolution resolution,
                                                  Decimal selector
                                                  )

Creates a MACD indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose MACD we want.
Int32fastPeriodThe period for the fast moving average.
Int32slowPeriodThe period for the slow moving average.
Int32signalPeriodThe period for the signal moving average.
MovingAverageTypetypeThe type of moving average to use for the MACD.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

MovingAverageConvergenceDivergence - The moving average convergence divergence between the fast and slow averages
