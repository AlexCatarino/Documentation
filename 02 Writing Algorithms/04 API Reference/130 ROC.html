ROC

ROC(
 Symbol symbol,
 Int32 period,
 Resolution resolution,
 Decimal selector
)

Creates a new RateOfChange indicator. This will compute the n-period rate of change in the security. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose RateOfChange we want.
Int32periodThe period over which to compute the RateOfChange.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

RateOfChange - The RateOfChange indicator for the requested symbol over the specified period
