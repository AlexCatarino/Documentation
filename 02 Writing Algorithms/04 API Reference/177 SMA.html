SMA

SMA(
 Symbol symbol,
 Int32 period,
 Resolution resolution,
 Decimal selector
)

Creates an SimpleMovingAverage indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose SMA we want.
Int32periodThe period of the SMA.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

SimpleMovingAverage - The SimpleMovingAverage for the given parameters
