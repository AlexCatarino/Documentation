StopMarketOrder()

StopMarketOrder(                                  Symbol symbol,
                                                  Int32 quantity,
                                                  Decimal stopPrice,
                                                  String tag,
                                                  IOrderProperties orderProperties
                                                  )

Create a stop market order and return the newly created order id; or negative if the order is invalid

Parameters

Type Name Description
SymbolsymbolString symbol for the asset we're trading.
Int32quantityQuantity to be traded.
DecimalstopPricePrice to fill the stop order.
StringtagOptional string data tag for the order.
IOrderPropertiesorderPropertiesThe order properties to use. Defaults to DefaultOrderProperties.

Returns

OrderTicket - The order ticket instance.
