AddSecurity(securityType,ticker,resolution,fillDataForward,extendedMarketHours,dataMappingMode,dataNormalizationMode)

Add specified data to our data subscriptions. QuantConnect will funnel this data to the handle data routine.

Parameters

Type Name Description
SecurityTypesecurityTypeMarketType Type: Equity, Commodity, Future, FOREX or Crypto. Options: ['Base', 'Equity', 'Option', 'Commodity', 'Forex', 'Future', 'Cfd', 'Crypto', 'FutureOption', 'Index', 'IndexOption']
/tickerThe security ticker.
/resolutionResolution of the Data Required.
/fillDataForwardWhen no data available on a tradebar, return the last data that was generated.
/extendedMarketHoursShow the after market data as well.
/dataMappingModeThe contract mapping mode to use for the security.
/dataNormalizationModeThe price scaling mode to use for the security.

Returns

This method provides no return.
