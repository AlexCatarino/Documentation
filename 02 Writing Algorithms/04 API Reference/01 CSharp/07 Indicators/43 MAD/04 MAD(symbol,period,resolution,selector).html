MAD(symbol,period,resolution,selector)

Creates a new MeanAbsoluteDeviation indicator.

Parameters

Type Name Description
/symbolThe symbol whose MeanAbsoluteDeviation we want.
/periodThe period over which to compute the MeanAbsoluteDeviation.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(MeanAbsoluteDeviation) - The MeanAbsoluteDeviation indicator for the requested symbol over the specified period
