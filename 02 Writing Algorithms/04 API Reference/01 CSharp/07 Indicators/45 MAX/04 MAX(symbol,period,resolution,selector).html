MAX(symbol,period,resolution,selector)

Creates a new Maximum indicator to compute the maximum value

Parameters

Type Name Description
/symbolThe symbol whose max we want.
/periodThe look back period over which to compute the max value.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None and the symbol is of type TradeBar defaults to the High property, otherwise it defaults to Value property of BaseData (x => x.Value).

Returns

(Maximum) - A Maximum indicator that compute the max value and the periods since the max value
