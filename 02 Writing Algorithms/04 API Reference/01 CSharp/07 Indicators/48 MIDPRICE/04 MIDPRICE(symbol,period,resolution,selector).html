MIDPRICE(symbol,period,resolution,selector)

Creates a new MidPrice indicator.

Parameters

Type Name Description
/symbolThe symbol whose MIDPRICE we want.
/periodThe period over which to compute the MIDPRICE.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(MidPrice) - The MidPrice indicator for the requested symbol over the specified period
