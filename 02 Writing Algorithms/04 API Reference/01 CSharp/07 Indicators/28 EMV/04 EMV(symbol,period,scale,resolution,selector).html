EMV(symbol,period,scale,resolution,selector)

Creates an EaseOfMovementValue indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
/symbolThe symbol whose EMV we want.
/periodThe period of the EMV.
/scaleThe length of the outputed value.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(EaseOfMovementValue) - The EaseOfMovementValue indicator for the given parameters
