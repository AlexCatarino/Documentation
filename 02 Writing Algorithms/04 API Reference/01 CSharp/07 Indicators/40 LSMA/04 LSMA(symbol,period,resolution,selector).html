LSMA(symbol,period,resolution,selector)

Creates and registers a new Least Squares Moving Average instance.

Parameters

Type Name Description
/symbolThe symbol whose LSMA we seek.
/periodThe LSMA period. Normally 14.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

(LeastSquaresMovingAverage) - A LeastSquaredMovingAverage configured with the specified period
