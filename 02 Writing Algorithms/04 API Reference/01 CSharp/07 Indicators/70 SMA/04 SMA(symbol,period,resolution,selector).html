SMA(symbol,period,resolution,selector)

Creates an SimpleMovingAverage indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
/symbolThe symbol whose SMA we want.
/periodThe period of the SMA.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(SimpleMovingAverage) - The SimpleMovingAverage for the given parameters
