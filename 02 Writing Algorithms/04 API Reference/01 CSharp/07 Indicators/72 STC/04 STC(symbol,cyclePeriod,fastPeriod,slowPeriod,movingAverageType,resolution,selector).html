STC(symbol,cyclePeriod,fastPeriod,slowPeriod,movingAverageType,resolution,selector)

Creates a new Schaff Trend Cycle indicator

Parameters

Type Name Description
/symbolThe symbol for the indicator to track.
/cyclePeriodThe signal period.
/fastPeriodThe fast moving average period.
/slowPeriodThe slow moving average period.
/movingAverageTypeThe type of moving average to use.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(SchaffTrendCycle) - The SchaffTrendCycle indicator for the requested symbol over the specified period
