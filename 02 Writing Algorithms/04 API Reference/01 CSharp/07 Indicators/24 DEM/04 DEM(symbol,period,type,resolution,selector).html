DEM(symbol,period,type,resolution,selector)

Creates a new DeMarker Indicator (DEM), an oscillator-type indicator measuring changes in terms of an asset's High and Low tradebar values.

Parameters

Type Name Description
/symbolThe symbol whose DEM we seek.
/periodThe period of the moving average implemented.
/typeSpecifies the type of moving average to be used.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

(DeMarkerIndicator) - The DeMarker indicator for the requested symbol.
