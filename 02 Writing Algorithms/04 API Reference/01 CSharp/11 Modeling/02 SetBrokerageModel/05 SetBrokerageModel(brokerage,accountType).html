SetBrokerageModel(brokerage,accountType)

Sets the brokerage to emulate in backtesting or paper trading. This can be used for brokerages that have been implemented in LEAN

Parameters

Type Name Description
BrokerageNamebrokerageThe brokerage to emulate. Options: ['Default', 'QuantConnectBrokerage', 'InteractiveBrokersBrokerage', 'TradierBrokerage', 'OandaBrokerage', 'FxcmBrokerage', 'Bitfinex', 'Binance', 'GDAX', 'Alpaca', 'AlphaStreams', 'Zerodha', 'Samco', 'Atreyu', 'TradingTechnologies', 'Kraken', 'FTX']
AccountTypeaccountTypeThe account type (Cash or Margin). Options: ['Margin', 'Cash']

Returns

This method provides no return.
