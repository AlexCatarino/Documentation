self.ROCR(symbol,period,resolution,selector)

Creates a new RateOfChangeRatio indicator.

Parameters

Type Name Description
/symbolThe symbol whose ROCR we want.
/periodThe period over which to compute the ROCR.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(RateOfChangeRatio) - The RateOfChangeRatio indicator for the requested symbol over the specified period
