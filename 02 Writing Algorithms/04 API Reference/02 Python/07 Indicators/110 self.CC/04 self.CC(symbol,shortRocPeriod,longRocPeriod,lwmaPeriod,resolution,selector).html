self.CC(symbol,shortRocPeriod,longRocPeriod,lwmaPeriod,resolution,selector)

Initializes a new instance of the CoppockCurve" indicator

Parameters

Type Name Description
/symbolThe symbol whose Coppock Curve we want.
/shortRocPeriodThe period for the short ROC.
/longRocPeriodThe period for the long ROC.
/lwmaPeriodThe period for the LWMA.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

(CoppockCurve) - The Coppock Curve indicator for the requested symbol over the specified period
