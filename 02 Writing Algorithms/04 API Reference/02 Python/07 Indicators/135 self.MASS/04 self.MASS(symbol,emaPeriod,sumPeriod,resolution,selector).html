self.MASS(symbol,emaPeriod,sumPeriod,resolution,selector)

Creates a new Mass Index indicator. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
/symbolThe symbol whose Mass Index we want.
/emaPeriodThe period used by both EMA.
/sumPeriodThe sum period.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(MassIndex) - The Mass Index indicator for the requested symbol over the specified period
