self.SWISS(symbol,period,delta,tool,resolution,selector)

Creates Swiss Army Knife transformation for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
/symbolThe symbol to use for calculations.
/periodThe period of the calculation.
/deltaThe delta scale of the BandStop or BandPass.
SwissArmyKnifeTooltoolThe tool os the Swiss Army Knife. Options: ['Gauss', 'Butter', 'HighPass', 'TwoPoleHighPass', 'BandPass']
/resolutionThe resolution.
/selectorelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

(SwissArmyKnife) - The calculation using the given tool
