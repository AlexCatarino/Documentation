self.KCH(symbol,period,k,movingAverageType,resolution,selector)

Creates a new Keltner Channels indicator. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
/symbolThe symbol whose Keltner Channel we seek.
/periodThe period over which to compute the Keltner Channels.
/kThe number of multiples of the from the middle band of the Keltner Channels.
/movingAverageTypeSpecifies the type of moving average to be used as the middle line of the Keltner Channel.
/resolutionThe resolution.
/selectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

(KeltnerChannels) - The Keltner Channel indicator for the requested symbol.
