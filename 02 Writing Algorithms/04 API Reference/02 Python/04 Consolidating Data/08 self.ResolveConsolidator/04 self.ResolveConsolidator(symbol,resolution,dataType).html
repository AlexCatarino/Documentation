self.ResolveConsolidator(symbol,resolution,dataType)

Gets the default consolidator for the specified symbol and resolution

Parameters

Type Name Description
/symbolThe symbol whose data is to be consolidated.
/resolutionThe resolution for the consolidator, if None, uses the resolution from subscription.
/dataTypeThe data type for this consolidator, if None, uses TradeBar over QuoteBar if present.

Returns

- The new default consolidator
