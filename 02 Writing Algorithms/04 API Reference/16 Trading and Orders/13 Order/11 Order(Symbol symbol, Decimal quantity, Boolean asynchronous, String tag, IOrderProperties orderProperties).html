Order(Symbol symbol, Decimal quantity, Boolean asynchronous, String tag, IOrderProperties orderProperties)

Wrapper for market order method: submit a new order for quantity of symbol using type order.

Parameters

Type Name Description
SymbolsymbolSymbol of the MarketType Required.
DecimalquantityNumber of shares to request.
BooleanasynchronousSend the order asynchronously (False). Otherwise we'll block until it fills.
StringtagPlace a custom order property or tag (e.g. indicator data).
IOrderPropertiesorderPropertiesThe order properties to use. Defaults to .

Returns

OrderTicket - The order ticket instance.
