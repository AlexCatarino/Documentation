LOGR()

LOGR(                                             Symbol symbol,
                                                  Int32 period,
                                                  Resolution resolution,
                                                  Decimal selector
                                                  )

Creates a new LogReturn indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose log return we seek.
Int32periodThe period of the log return.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

LogReturn - log return indicator for the requested symbol.
