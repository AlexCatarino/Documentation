self.AddUniverse(T,name,resolution,universeSettings,selector)

Creates a new universe and adds it to the algorithm. This will use the default universe settings specified via the property. This universe will use the defaults of SecurityType.Equity, and Market.USA

Parameters

Type Name Description
PyObjectTThe data type.
StringnameA unique name for this universe.
ResolutionresolutionThe epected resolution of the universe data.
UniverseSettingsuniverseSettingsThe settings used for securities added by this universe.
PyObjectselectorFunction delegate that performs selection on the universe data.

Returns

Universe
