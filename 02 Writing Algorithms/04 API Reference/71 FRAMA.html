FRAMA

FRAMA(
 Symbol symbol,
 Int32 period,
 Int32 longPeriod,
 Resolution resolution,
 IBaseDataBar selector
)

Creates an FractalAdaptiveMovingAverage (FRAMA) indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose FRAMA we want.
Int32periodThe period of the FRAMA.
Int32longPeriodThe long period of the FRAMA.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

FractalAdaptiveMovingAverage - The FRAMA for the given parameters
