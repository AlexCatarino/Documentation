RSI(symbol,period,movingAverageType,resolution,selector)

Creates a new RelativeStrengthIndex indicator. This will produce an oscillator that ranges from 0 to 100 based on the ratio of average gains to average losses over the specified period.

Parameters

Type Name Description
SymbolsymbolThe symbol whose RSI we want.
Int32periodThe period over which to compute the RSI.
MovingAverageTypemovingAverageTypeThe type of moving average to use in computing the average gain/loss values.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

RelativeStrengthIndex - The RelativeStrengthIndex indicator for the requested symbol over the specified period
