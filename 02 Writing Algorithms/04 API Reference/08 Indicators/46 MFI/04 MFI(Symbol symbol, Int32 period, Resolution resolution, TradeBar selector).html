MFI(Symbol symbol, Int32 period, Resolution resolution, TradeBar selector)

Creates a new MoneyFlowIndex indicator. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose MFI we want.
Int32periodThe period over which to compute the MFI.
ResolutionresolutionThe resolution.
TradeBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

MoneyFlowIndex - The MoneyFlowIndex indicator for the requested symbol over the specified period
