DCH(Symbol symbol, Int32 period, Resolution resolution, IBaseDataBar selector)

Overload shorthand to create a new symmetric Donchian Channel indicator which has the upper and lower channels set to the same period length.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Donchian Channel we seek.
Int32periodThe period over which to compute the Donchian Channel.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

DonchianChannel - The Donchian Channel indicator for the requested symbol.
