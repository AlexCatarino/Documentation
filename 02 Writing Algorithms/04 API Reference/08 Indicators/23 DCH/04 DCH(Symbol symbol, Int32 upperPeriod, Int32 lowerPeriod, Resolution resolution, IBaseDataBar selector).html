DCH(Symbol symbol, Int32 upperPeriod, Int32 lowerPeriod, Resolution resolution, IBaseDataBar selector)

Creates a new Donchian Channel indicator which will compute the Upper Band and Lower Band. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Donchian Channel we seek.
Int32upperPeriodThe period over which to compute the upper Donchian Channel.
Int32lowerPeriodThe period over which to compute the lower Donchian Channel.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

DonchianChannel - The Donchian Channel indicator for the requested symbol.
