ARIMA(symbol,arOrder,diffOrder,maOrder,period,resolution)

Creates a new ARIMA indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose ARIMA indicator we want.
Int32arOrderAR order (p) -- defines the number of past values to consider in the AR component of the model.
Int32diffOrderDifference order (d) -- defines how many times to difference the model before fitting parameters.
Int32maOrderMA order (q) -- defines the number of past values to consider in the MA component of the model.
Int32periodSize of the rolling series to fit onto.
ResolutionresolutionThe resolution.

Returns

AutoRegressiveIntegratedMovingAverage - The ARIMA indicator for the requested symbol over the specified period
