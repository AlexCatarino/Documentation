DEM(Symbol symbol, Int32 period, MovingAverageType type, Resolution resolution, TradeBar selector)

Creates a new DeMarker Indicator (DEM), an oscillator-type indicator measuring changes in terms of an asset's High and Low tradebar values.

Parameters

Type Name Description
SymbolsymbolThe symbol whose DEM we seek.
Int32periodThe period of the moving average implemented.
MovingAverageTypetypeSpecifies the type of moving average to be used.
ResolutionresolutionThe resolution.
TradeBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

DeMarkerIndicator - The DeMarker indicator for the requested symbol.
