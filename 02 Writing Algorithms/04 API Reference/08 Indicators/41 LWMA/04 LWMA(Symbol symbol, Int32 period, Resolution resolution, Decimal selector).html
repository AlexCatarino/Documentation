LWMA(Symbol symbol, Int32 period, Resolution resolution, Decimal selector)

Creates a new LinearWeightedMovingAverage indicator. This indicator will linearly distribute the weights across the periods.

Parameters

Type Name Description
SymbolsymbolThe symbol whose LWMA we want.
Int32periodThe period over which to compute the LWMA.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

LinearWeightedMovingAverage
