ADX(Symbol symbol, Int32 period, Resolution resolution, IBaseDataBar IBaseDataBar)

Creates a new Average Directional Index indicator. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Average Directional Index we seek.
Int32periodThe period over which to compute the Average Directional Index.
ResolutionresolutionThe resolution.
IBaseDataBarIBaseDataBarRepresents a type that is both a bar and base data.

Returns

AverageDirectionalIndex - The Average Directional Index indicator for the requested symbol.
