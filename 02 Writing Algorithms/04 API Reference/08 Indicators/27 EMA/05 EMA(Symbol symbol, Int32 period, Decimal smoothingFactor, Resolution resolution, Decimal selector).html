EMA({'Description': 'The symbol whose EMA we want.', 'Type': 'Symbol'} symbol, {'Description': 'The period of the EMA.', 'Type': 'Int32'} period, {'Description': 'The percentage of data from the previous value to be carried into the next value.', 'Type': 'Decimal'} smoothingFactor, {'Description': 'The resolution.', 'Type': 'Resolution'} resolution, {'Description': 'Selects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).', 'Type': 'Decimal'} selector)

Creates an ExponentialMovingAverage indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose EMA we want.
Int32periodThe period of the EMA.
DecimalsmoothingFactorThe percentage of data from the previous value to be carried into the next value.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

ExponentialMovingAverage - The ExponentialMovingAverage for the given parameters
