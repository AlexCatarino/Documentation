T3(symbol,period,volumeFactor,resolution,selector)

Creates a new T3MovingAverage indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose T3 we want.
Int32periodThe period over which to compute the T3.
DecimalvolumeFactorThe volume factor to be used for the T3 (value must be in the [0,1] range, defaults to 0.7).
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

T3MovingAverage - The T3MovingAverage indicator for the requested symbol over the specified period
