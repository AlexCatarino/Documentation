STC(Symbol symbol, Int32 cyclePeriod, Int32 fastPeriod, Int32 slowPeriod, MovingAverageType movingAverageType, Resolution resolution, Decimal selector)

Creates a new Schaff Trend Cycle indicator

Parameters

Type Name Description
SymbolsymbolThe symbol for the indicator to track.
Int32cyclePeriodThe signal period.
Int32fastPeriodThe fast moving average period.
Int32slowPeriodThe slow moving average period.
MovingAverageTypemovingAverageTypeThe type of moving average to use.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

SchaffTrendCycle - The SchaffTrendCycle indicator for the requested symbol over the specified period
