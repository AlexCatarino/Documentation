Creates an Market Profile indicator for the symbol with Volume Profile (VOL) mode. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose VP we want.
Int32periodThe period of the VP.
DecimalvalueAreaVolumePercentageThe percentage of volume contained in the value area.
DecimalpriceRangeRoundOffHow many digits you want to round and the precision. i.e 0.01 round to two digits exactly.
ResolutionresolutionThe resolution.
TradeBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

VolumeProfile - The Volume Profile indicator for the given parameters
