PPHL(symbol,lengthHigh,lengthLow,lastStoredValues,resolution,selector)

Creates a new PivotPointsHighLow indicator

Parameters

Type Name Description
SymbolsymbolThe symbol whose PPHL we seek.
Int32lengthHighThe number of surrounding bars whose high values should be less than the current bar's for the bar high to be marked as high pivot point.
Int32lengthLowThe number of surrounding bars whose low values should be more than the current bar's for the bar low to be marked as low pivot point.
Int32lastStoredValuesThe number of last stored indicator values.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

PivotPointsHighLow - The PivotPointsHighLow indicator for the requested symbol.
