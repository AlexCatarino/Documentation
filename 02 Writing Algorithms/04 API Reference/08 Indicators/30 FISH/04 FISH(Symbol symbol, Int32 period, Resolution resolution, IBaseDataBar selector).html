FISH(Symbol symbol, Int32 period, Resolution resolution, IBaseDataBar selector)

Creates an FisherTransform indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose FisherTransform we want.
Int32periodThe period of the FisherTransform.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

FisherTransform - The FisherTransform for the given parameters
