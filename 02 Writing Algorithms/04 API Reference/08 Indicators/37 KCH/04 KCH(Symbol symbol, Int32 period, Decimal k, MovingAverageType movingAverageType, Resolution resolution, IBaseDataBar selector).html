KCH(Symbol symbol, Int32 period, Decimal k, MovingAverageType movingAverageType, Resolution resolution, IBaseDataBar selector)

Creates a new Keltner Channels indicator. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Keltner Channel we seek.
Int32periodThe period over which to compute the Keltner Channels.
DecimalkThe number of multiples of the from the middle band of the Keltner Channels.
MovingAverageTypemovingAverageTypeSpecifies the type of moving average to be used as the middle line of the Keltner Channel.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

KeltnerChannels - The Keltner Channel indicator for the requested symbol.
