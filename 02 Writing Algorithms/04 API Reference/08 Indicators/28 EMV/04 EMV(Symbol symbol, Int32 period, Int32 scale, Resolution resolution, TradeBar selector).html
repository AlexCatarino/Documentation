EMV(Symbol symbol, Int32 period, Int32 scale, Resolution resolution, TradeBar selector)

Creates an EaseOfMovementValue indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose EMV we want.
Int32periodThe period of the EMV.
Int32scaleThe length of the outputed value.
ResolutionresolutionThe resolution.
TradeBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

EaseOfMovementValue - The EaseOfMovementValue indicator for the given parameters
