Identity(Symbol symbol, TimeSpan resolution, Decimal selector, String fieldName)

Creates a new Identity indicator for the symbol The indicator will be automatically updated on the symbol's subscription resolution

Parameters

Type Name Description
SymbolsymbolThe symbol whose values we want as an indicator.
TimeSpanresolutionThe desired resolution of the data.
DecimalselectorSelects a value from the BaseData, if None defaults to the .Value property (x => x.Value).
StringfieldNameThe name of the field being selected.

Returns

Identity - A new Identity indicator for the specified symbol and selector
