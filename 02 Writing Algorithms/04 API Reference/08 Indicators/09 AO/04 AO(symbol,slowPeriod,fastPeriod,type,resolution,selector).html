AO(symbol,slowPeriod,fastPeriod,type,resolution,selector)

Creates a new Awesome Oscillator from the specified periods.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Awesome Oscillator we seek.
Int32slowPeriodThe period of the slow moving average associated with the AO.
Int32fastPeriodThe period of the fast moving average associated with the AO.
MovingAverageTypetypeThe type of moving average used when computing the fast and slow term. Defaults to simple moving average.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

AwesomeOscillator
