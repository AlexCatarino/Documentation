KAMA(Symbol symbol, Int32 period, Resolution resolution, Decimal selector)

Creates a new KaufmanAdaptiveMovingAverage indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose KAMA we want.
Int32periodThe period of the Efficiency Ratio (ER) of KAMA.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

KaufmanAdaptiveMovingAverage - The KaufmanAdaptiveMovingAverage indicator for the requested symbol over the specified period
