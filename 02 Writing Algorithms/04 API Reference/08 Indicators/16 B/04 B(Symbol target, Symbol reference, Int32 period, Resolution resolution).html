B(Symbol target, Symbol reference, Int32 period, Resolution resolution)

Creates a Beta indicator for the given target symbol in relation with the reference used. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymboltargetThe target symbol whose Beta value we want.
SymbolreferenceThe reference symbol to compare with the target symbol.
Int32periodThe period of the Beta indicator.
ResolutionresolutionThe resolution.

Returns

Beta - The Beta indicator for the given parameters
