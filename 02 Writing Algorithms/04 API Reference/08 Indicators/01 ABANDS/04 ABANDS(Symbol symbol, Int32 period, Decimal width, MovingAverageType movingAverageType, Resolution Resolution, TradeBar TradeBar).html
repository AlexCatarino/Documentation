ABANDS(Symbol symbol, Int32 period, Decimal width, MovingAverageType movingAverageType, Resolution Resolution, TradeBar TradeBar)

Creates a new Acceleration Bands indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Acceleration Bands we want.
Int32periodThe period of the three moving average (middle, upper and lower band).
DecimalwidthA coefficient specifying the distance between the middle band and upper or lower bands.
MovingAverageTypemovingAverageTypeType of the moving average.
Options: ['Simple', 'Exponential', 'Wilders', 'LinearWeightedMovingAverage', 'DoubleExponential', 'TripleExponential', 'Triangular', 'T3', 'Kama', 'Hull', 'Alma']
ResolutionResolutionResolution of data requested. Always sort the enum from the smallest to largest resolution.
Options: ['Tick', 'Second', 'Minute', 'Hour', 'Daily']
TradeBarTradeBarTradeBar class for second and minute resolution data: An OHLC implementation of the QuantConnect BaseData class with parameters for candles.

Returns

AccelerationBands
