NATR(symbol,period,resolution,selector)

Creates a new NormalizedAverageTrueRange indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose NATR we want.
Int32periodThe period over which to compute the NATR.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

NormalizedAverageTrueRange - The NormalizedAverageTrueRange indicator for the requested symbol over the specified period
