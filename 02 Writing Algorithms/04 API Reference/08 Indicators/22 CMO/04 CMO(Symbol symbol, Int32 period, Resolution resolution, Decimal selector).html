CMO(Symbol symbol, Int32 period, Resolution resolution, Decimal selector)

Creates a new ChandeMomentumOscillator indicator.

Parameters

Type Name Description
SymbolsymbolThe symbol whose CMO we want.
Int32periodThe period over which to compute the CMO.
ResolutionresolutionThe resolution.
DecimalselectorSelects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Returns

ChandeMomentumOscillator - The ChandeMomentumOscillator indicator for the requested symbol over the specified period
