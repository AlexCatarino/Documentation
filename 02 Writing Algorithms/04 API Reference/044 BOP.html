BOP()

BOP(
                                              Symbol symbol,
                                                  Resolution resolution,
                                                  IBaseDataBar selector
                                                  )

Creates a new Balance Of Power indicator. The indicator will be automatically updated on the given resolution.

Parameters

Type Name Description
SymbolsymbolThe symbol whose Balance Of Power we seek.
ResolutionresolutionThe resolution.
IBaseDataBarselectorSelects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Returns

BalanceOfPower - The Balance Of Power indicator for the requested symbol.
