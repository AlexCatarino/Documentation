self.Consolidate(symbol,period,TickType,handler)

Registers the to receive consolidated data for the specified symbol

Parameters

Type Name Description
SymbolsymbolThe symbol who's data is to be consolidated.
ResolutionperiodThe consolidation period.
TickTypeTickTypeTypes of tick data QuantConnect currently only has trade, quote, open interest tick data.
Options: ['Trade', 'Quote', 'OpenInterest']
PyObjecthandlerData handler receives new consolidated data when generated.

Returns

IDataConsolidator- A new consolidator matching the requested parameters with the handler already registered
