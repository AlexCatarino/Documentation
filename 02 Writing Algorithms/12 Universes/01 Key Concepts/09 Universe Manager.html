The universe manager tracks all the universes in your algorithm. If you add multiple universes to your algorithm, you can access the constituents of each individual universe. To access the constituents of a universe in a multi-universe algorithm, save references to each universe when you add them.

private Universe _universe;

// In Initialize
_universe = AddUniverse(MyCoarseFilterFunction);

// In OnData
var universeMembers = UniverseManager[_universe.Symbol].Members;
# In Initialize
self.universe = self.AddUniverse(self.MyCoarseFilterFunction)

# In OnData
universe_members = self.UniverseManager[self.universe.Symbol].Members
