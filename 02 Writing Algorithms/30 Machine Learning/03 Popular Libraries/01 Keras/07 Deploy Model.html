Predict Labels

To predict the labels of new data, get the most recent set of features and then call the predict method.

features = self.get_features()[-1]
prediction = float(self.model.predict(features)[-1])

After you predict the latest label, you can place orders.

if prediction > 0:
    self.SetHoldings(self.spy, 1)
elif prediction < 0:            
    self.SetHoldings(self.spy, -1)
