The Chart object contains a collection of Series objects, which display data on the chart. To add a chart to an algorithm, create a Chart object and then call the AddChart method.

var chart = new Chart("Chart Name");
AddChart(chart);
chart = Chart("Chart Name")
self.AddChart(chart)

The Chart constructor expects a name argument. The following chart names are reserved:

  • Assets Sales Volume
  • Exposure
  • Alpha Assets
  • Alpha Asset Breakdown
