The QC Cloud deployment environment is a collection of servers that the QuantConnect team manages. It's the same deployment environment you use if you create projects, spin up research nodes, and deploy algorithms on the QuantConnect website. Follow these steps to set the deployment environment of a project on the local platform to QC Cloud:

  1. Create a project open an exisiting one.
  2. In the Project panel, click the Deployment Environment field and then click Cloud from the drop-down menu.
