To deploy a private cloud, use the following steps:

Setting up a Master Work Server:

  1. Follow the following tutorial to install and authenticate the LEAN CLI.
  2. Run the LEAN CLI command lean deploy private-cloud --master to start a work queue and listen for jobs from the team.
    If you are setting up a single server, this is all you need to do.
  3. To expand the private cluster further so work is distributed across many servers, install the LEAN CLI into the second server and run: lean deploy private-cloud --slavecode>.
    This will spin up the second work-server to consume jobs.
