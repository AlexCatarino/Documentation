Follow these steps to install the local platform:

  1. Install Docker.
  2. Open a terminal and download the latest LEAN image.
    3. $ docker pull quantconnect/lean

    It takes about an hour to download the image. While it's downloading, continue to the next step.

  3. Install Visual Studio Code.
  4. Install Local Platform.

If you open Visual Studio Code and it asks you to log in to QuantConnect, you successfully installed the local platform.
