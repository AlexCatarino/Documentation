The Exchangeexchange property of a Security object contains information about the exchange that lists the security. Notably the Hourshours property

contains information on the trading days and hours. 
var exchange = Securities["SPY"].Exchange;
var hours = exchange.Hours;
var isOpenNow = hours.isOpen(Time, extendedMarketHours: false);
