The holdingsHoldings property of a Security object contains information about the investment state and history of the security.

var holdings = Securities["SPY"].Holdings;
var quantity = holdings.Quantity;
holdings = self.securities["SPY"].holdings
quantity = holdings.quantity

Alternatively, you can access the current holdings of a security with the Portfolioportfolio dictionary.

var holdings = Portfolio["SPY"];
var quantity = holdings.Quantity;
holdings = self.portfolio["SPY"]
quantity = holdings.quantity
