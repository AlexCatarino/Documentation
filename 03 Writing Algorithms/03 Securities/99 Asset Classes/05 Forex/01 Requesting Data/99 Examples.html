The following examples demonstrate some common practices for requesting Forex data.

Example 1: Request All Oanda Forex

The following algorithm aims to add subscription to all Forex pairs traded in the Oanda exchange. By making use of the GetSymbolPropertiesListget_symbol_properties_list method of the symbol properties database with the designated market and security type, we can obtain and request data for all symbols of the wanted security type traded in the desired market. It will be particularly useful for spotting arbitration opportunities among forex pairs.

public ForexExampleAlgorithm : QCAlgorithm
{
    private List<Symbol> _forex;

    public override void Initialize()
    {
        // To get all symbols and properties of Oanda traded Forex, retrieve data from the symbol property database.
        var symbolPropertiesList = SymbolPropertiesDatabase.GetSymbolPropertiesList(Market.Oanda, SecurityType.Forex);
        // Subscribe to all the returned Forex, note that you do not need to specify the market as the symbol object already does.
        _forex = symbolPropertiesList.Select(x => AddForex(x.Key.Symbol).Symbol).ToList();
    }
}
class ForexExampleAlgorithm(QCAlgorithm):
    def initialize(self) -> None:
        # To get all symbols and properties of Oanda traded Forex, retrieve data from the symbol property database.
        symbol_properties_list = self.symbol_properties_database.get_symbol_properties_list(Market.OANDA, SecurityType.FOREX)
        # Subscribe to all the returned Forex, note that you do not need to specify the market as the symbol object already does.
        self._forex = [self.add_forex(x.key.symbol).symbol for x in symbol_properties_list]
