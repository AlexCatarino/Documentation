The following tables describe where you can access the logs and runtime statistics, depending on how to deploy your algorithms.

QuantConnect Cloud Deployments

The following table describes how you can access the logs and runtime statistics of algorithms you deploy in QuantConnect Cloud:

Execution Type Result Type Access Tools
Backtest Logs
  • Backtest results page in the Algorithm Lab
  • Backtest results page on Local Platform
Backtest Runtime Statistics
  • Backtest results page in the Algorithm Lab
  • Backtest results page on Local Platform
  • ReadBacktest API method
  • /backtests/read endpoint
Live Trading Logs
  • Live results page in the Algorithm Lab
  • Live results page on Local Platform
  • /live/read/log endpoint
Live Trading Runtime Statistics
  • Live results page in the Algorithm Lab
  • Live results page on Local Platform
  • ReadLiveAlgorithm API method
  • /live/read endpoint

On-Premise Deployments

The following table describes how you can access the logs and runtime statistics of algorithms you deploy on-premise:

Execution Type Result Type Access Tools
Backtest Logs
  • Backtest results page on Local Platform
  • <organizationWorkspace> / <projectName> / backtests / <unixTimestamp> / <algorithmId>-log.txt
  • lean logs CLI command
Backtest Runtime Statistics
  • Backtest results page on Local Platform
  • <organizationWorkspace> / <projectName> / backtests / <unixTimestamp> / <algorithmId>.json
Live Trading Logs
  • Live results page on Local Platform
  • <organizationWorkspace> / <projectName> / live / <timeStamp> / / L-<deploymentId>-log.txt
  • lean logs CLI command
Live Trading Runtime Statistics
  • Live results page on Local Platform
  • <organizationWorkspace> / <projectName> / live / <timeStamp> / / L-<deploymentId>.json
