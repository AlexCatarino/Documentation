When LEAN returns an enumerable, the object should be converted to a list() to be easily read by python classes. Popular API methods which return an enumerable are shown below:

Historical Data

While using research or building algorithms you might see types like the MemoizingEnumerable that is a raw C# type accessible in python. 

qb = QuantBook()
start = datetime(2023, 2, 1)
end = datetime(2023, 2, 4)
ibm = qb.AddEquity('IBM', Resolution.Minute)
bars = qb.History[TradeBar](ibm.Symbol, start, end, Resolution.Minute)
bars
<QuantConnect.Util.MemoizingEnumerable[TradeBar] object at 0x7f1dea829f40>

You can directly enumerate this object with standard python for loops, or to convert it to a native type, wrap the object in the list():

# Convert to a standard python list
bars = list(bars)
len(bars)
