If you update the Object Store from outside of your live trading node and try to read the new content from your live node, it may look like the data you read never updates. This occurs because the Object Store caches data to speed up execution. To get the latest value for a specific Object Store key, clear the cache and then read the dataa>.

ObjectStore.Clear();
if (ObjectStore.ContainsKey(key))
{
    var value = ObjectStore.Read(key);
}
self.ObjectStore.Clear()
if self.ObjectStore.ContainsKey(key):
    value = self.ObjectStore.Read(key)
