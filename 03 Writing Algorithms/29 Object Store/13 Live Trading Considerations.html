If you update the Object Store from outside of your live trading node and then try to read the new content from inside your live trading algorithm, you may not get the latest data because the Object Store caches data to speed up execution. To get the latest value for a specific Object Store key, clear the cache and then read the data.

ObjectStore.Clear();
if (ObjectStore.ContainsKey(key))
{
    var value = ObjectStore.Read(key);
}
self.ObjectStore.Clear()
if self.ObjectStore.ContainsKey(key):
    value = self.ObjectStore.Read(key)
