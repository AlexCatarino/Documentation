To get the historical data in a DataFrame, use Python.

When you request historical data for a universe in the format of a DataFrame, the default format bundles all of the daily data into a list. The following code block and image sho an example of this format:

universe = self.add_universe(self.universe.dollar_volume.top(3))
history = self.history(universe, 2)

To flatten the data into a DataFrame where there is one column for each data point attribute, set the flatten argument to True.

history = self.history(universe, 2, flatten=True)
