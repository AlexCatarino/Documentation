You can get historical fundamental data in an algorithm and in the Research Environment.

Historical Data in Algorithms

To get historical fundamental data in an algorithm, call the History method with Fundamental type and the Equity Symbol. The method returns fundamental data for the specific assets you provide.

var ibm = QuantConnect.Symbol.Create("IBM", SecurityType.Equity, Market.USA);

// Fundamental objects
var fundamentalHistory = History<Fundamental>(ibm, TimeSpan.FromDays(30));
ibm = Symbol.Create("IBM", SecurityType.Equity, Market.USA)

# DataFrame objects
asset_df_history = self.History(Fundamental, ibm, timedelta(30))

# Fundamental objects
fundamental_history = self.History[Fundamental](ibm, timedelta(30))

To get fundamental data for all US Equities, including delisted companies, call the History method with the Fundamentals type.

// Fundamentals objects for all US Equities (including delisted companies)
var fundamentalsHistory = History<Fundamentals>(TimeSpan.FromDays(30));
# DataFrame objects
all_df_history = self.History(Fundamentals, timedelta(30))

# Fundamentals objects
fundamentals_history = self.History[Fundamentals](timedelta(30))

If there is no data in the period you request, the history result is empty.

Historical Data in the Research Environment

To get historical data in the Research Environment, call any of the preceding methods or call the GetFundamental method with the Equity Symbol, a Fundamental property, a start date, and an end date. If there is no data in the period you request, the history result is empty.

history = qb.GetFundamental(symbol, "ValuationRatios.PERatios", datetime(2021, 1, 1), datetime(2021, 7, 1))
var history = qb.GetFundamental(symbol, "ValuationRatios.PERatios", new DateTime(2021, 1, 1), new DateTime(2021, 7, 1));

For more information about historical US Equity fundamental data, see Equity Fundamental Data.
