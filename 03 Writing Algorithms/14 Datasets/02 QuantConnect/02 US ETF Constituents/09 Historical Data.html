There are several types of historical data you can request.

Get Constituent Data History

To get historical ETFConstituentData objects, add a warm-up period so your selection function receives the ETFConstituentData objects from before the start date of your algorithm.

Get Price History

To get historical prices of the ETF constituents, use the Symbol member of each object to request historical market data. If there is no data in the period you request, the history result is empty.

def ETFConstituentsFilter(self, constituents: List[ETFConstituentData]) -> List[Symbol]:
    # Get the 20 largest weighting constituents
    selected = sorted([c for c in constituents if c.Weight],
        key=lambda c: c.Weight, reverse=True)[:20]
    selectedSymbols = [c.Symbol for c in selected]

    history = History(selectedSymbols, 10, Resolution.Daily)
                
    return selectedSymbols 

private IEnumerable<Symbol> ETFConstituentsFilter(IEnumerable<ETFConstituentData> constituents)
{
    // Get the 20 largest weighting constituents
    var selectedSymbols = constituents
        .OrderByDescending(c => c.Weight)
        .Take(10)
        .Select(c => c.Symbol)

    var history = History(selectedSymbols, 10, Resolution.Daily);

    return selectedSymbols;
}
