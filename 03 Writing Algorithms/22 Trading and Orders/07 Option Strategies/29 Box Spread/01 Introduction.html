A Box Spread is the combination of a bull call spread and a bear put spread. It consist of buying an ITM call at strike $A$, selling an OTM put at strike $A$, buying an ITM put at strike $B < A$, and selling an OTM call at strike $B$. This strategy serves as an delta-neutral arbitration from Option mispricing. Note that it only attains a true profit when the risk-free return is greater than the risk-free interest rate.
