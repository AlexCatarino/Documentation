A Box Spread is the combination of a bull call spread and a bear put spread. It consist of buying an ITM call and an ITM put, while selling an OTM put and an OTM call with the same strike as the ITM call and ITM put respectively. This strategy serves as an delta-neutral arbitration from option mispricing. Note that it only attains a true profit when the risk-free return is greater than the risk free interest rate.
