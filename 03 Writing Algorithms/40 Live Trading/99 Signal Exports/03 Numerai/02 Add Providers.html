To export signals to Collective2 from your algorithm, during initialization, add a Collective2 signal export provider by calling the AddSignalExportProviders method with a NumeraiSignalExport object.

SignalExport.AddSignalExportProviders(new NumeraiSignalExport(publicId, secretId, modelId, fileName));

self.SignalExport.AddSignalExportProviders(NumeraiSignalExport(publicId, secretId, modelId, fileName))

The NumeraiSignalExport constructor accepts the following arguments:

Argument Data Type Description Default Value
publicId stringstr Your Numerai API key. To create and view your API keys, see the Automation section of your Account page on the Numerai website.
secretId stringstr Your Numerai API secret. You get your API secret when you create a new API key.
modelId stringstr The Id of the Numerai model. Follow these steps to get the model Id:
  1. Open the Models page on the Numerai website.
  2. In the Models section, click the three dots next to a model.
  3. Click Copy Model ID.
 ""
fileName stringstr The name for the signal file. If you use a file name that already exists, the new file overwrites the old one. "predictions.csv"

You can add multiple signal export providers to a single algorithm.
