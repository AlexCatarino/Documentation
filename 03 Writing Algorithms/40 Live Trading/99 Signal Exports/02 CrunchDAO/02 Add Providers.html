To export signals to Collective2 from your algorithm, during initialization, add a Collective2 signal export provider.

SignalExport.AddSignalExportProviders(new CrunchDAOSignalExport(apiKey, model, submissionName, comment));

self.SignalExport.AddSignalExportProviders(CrunchDAOSignalExport(apiKey, model, submissionName, comment))

The CrunchDAOSignalExport constructor accepts the following arguments:

Argument Data Type Description Default Value
apiKey stringstr Your CrunchDAO API key. To get your API key, see the Profile > Settings page on the CrunchDAO website.
model stringstr The Id of your CrunchDAO model. To create a new model or to get Id of an existing model, see the Profile > Alpha page on the CrunchDAO website.
submissionName stringstr A name for the submission to distiguish it from your other submissions. ""
comment stringstr A comment for the submission. ""

You can add multiple signal export providers to a single algorithm.
