The BinanceFeeModel models the fees of Binance and Binance US. It's the default fee model of the BinanceBrokerageModel and BinanceUSBrokerageModel.

security.SetFeeModel(new BinanceFeeModel());
security.SetFeeModel(BinanceFeeModel())

The following table describes the arguments the model accepts:

Argument Data Type Description Default Value
mFee decimalfloat Maker fee value 0.001
tFee decimalfloat Taker fee value 0.001

The BinanceFeeModel charges fees in the currency you receive from a trade. For example, if you buy ETHBTC, you pay fees in ETH. If you sell ETHBTC, you pay fees in BTC.

To view the implementation of this model, see the LEAN GitHub repository.
