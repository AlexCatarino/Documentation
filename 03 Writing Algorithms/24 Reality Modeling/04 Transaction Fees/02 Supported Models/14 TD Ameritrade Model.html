The TDAmeritradeFeeModel models the fees of TD Ameritrade. It's the default fee model of the TDAmeritradeBrokerageModel.

security.SetFeeModel(new TDAmeritradeFeeModel());
security.SetFeeModel(TDAmeritradeFeeModel())

The TDAmeritradeFeeModel charges fees in USD.

To view the implementation of this model, see the LEAN GitHub repository.
