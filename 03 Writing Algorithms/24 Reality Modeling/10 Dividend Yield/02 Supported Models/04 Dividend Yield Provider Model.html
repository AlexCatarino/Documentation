The DividendYieldProvider estimated annualized continuous dividend yield at given date from historical dividend payments.

var symbol = AddEquity("MSFT").Symbol;
_dividendYieldModel = new DividendYieldProvider(symbol);
symbol = self.AddEquity("MSFT").Symbol
self.dividendYieldModel = DividendYieldProvider(symbol);

The following table describes the arguments the model accepts:

Argument Data Type Description
symbol Symbol The symbol of a dividend-paying security

To view the implementation of this model, see the LEAN GitHub repository.
