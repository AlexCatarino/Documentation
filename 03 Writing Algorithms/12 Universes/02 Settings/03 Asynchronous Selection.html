To improve the algorithm performance, set Asynchronous property to trueTrue in the Initialize method before you add a universes.

UniverseSettings.Asynchronous = true;
    AddUniverse(MyFundamentalFilterFunction);
    var option = AddOption("SPY");
    option.SetFilter(u => u.Strikes(-10, 10).Expiration(0, 30));
self.UniverseSettings.Asynchronous = True
    self.AddUniverse(self.MyFundamentalFilterFunction)
    option = self.AddOption("SPY")
    option.SetFilter(lambda u: u.Strikes(-10, 10).Expiration(0, 30))

The filter function cannot use external data

  1. Historical data requests
  2. Porfolio or security state. For example, Securities[_underlyingSymbol].Priceself.Securities[self.underlying_symbol].Price
