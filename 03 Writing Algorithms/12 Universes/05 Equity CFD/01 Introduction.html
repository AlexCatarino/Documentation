An Equity CFD universe lets you select a set of Equity CFD assets. To select the underlying Equities, you can use a Fundamental universe to select assets based on corporate fundamental data or you can use an ETF constituents universe to select the CFD versions of stocks in an exchange-traded fund. This feature is powerful for European clients seeking to trade US ETF products.

Only the Interactive Brokers CFD integration supports trading Stock-CFD products. QuantConnect doesn't have historical data for Interactive Brokers CFD products; however, you can use the LiveMode flag to swap to the CFD equivalents for live trading. In live trading, you must include Interactive Brokers as a data provider to trade Equity-CFD products.
