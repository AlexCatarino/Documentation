The following examples demonstrate some common liquidity universes for US Equities.

Example 1: Daily Liquid Universe

The following example selects the 100 most liquid US Equities and subscribes to daily data:

public class DailyLiquidUniverseAlgorithm : QCAlgorithm
{
    public override void Initialize()
    {
        UniverseSettings.Resolution = Resolution.Daily;
        AddUniverse(Universe.DollarVolume.Top(100));
    }
}
class DailyLiquidUniverseAlgorithm(QCAlgorithm):

    def initialize(self):
        self.universe_settings.resolution = Resolution.DAILY
        self.add_universe(self.universe.dollar_volume.top(100))

Example 2: Asynchronous Liquid Universe

The following example asynchronously selects the 1,000 most liquid stocks:

public class AsynchronousLiquidUniverseAlgorithm : QCAlgorithm
{
    public override void Initialize()
    {
        UniverseSettings.Asynchronous = true;
        AddUniverse(Universe.DollarVolume.Top(1000));
    }
}
class AsynchronousLiquidUniverseAlgorithm(QCAlgorithm):

    def initialize(self):
        self.universe_settings.asynchronous = True
        self.add_universe(self.universe.dollar_volume.top(1_000))
