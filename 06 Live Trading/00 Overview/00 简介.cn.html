自2015年以来，QuantConnect一直支持实盘交易。我们拥有经过实战考验的共置服务器，可为数千个实盘交易客户提供服务。实盘交易需要稳定性和正常运行时间，现在我们可以实现6个月的不间断运行时间。为实现这一目标，我们购买了现代化的服务器并将其存放在Equinix中，来为QuantConnect社区提供最佳的稳定性。

QuantConnect致力于在回测与实盘交易之间尽可能煎炒差异以保持一致性。在大部分情况下，您不会注意到任何差异。在以下部分中，我们将为您介绍一些实盘交易策略的功能。

您可以通过$[LiveMode,P:QuantConnect.Algorithm.QCAlgorithm.LiveMode]布尔属性来区分回测以及实盘交易。 

if (LiveMode) {
      Debug("实盘交易！");
}
if self.LiveMode:
      self.Debug("实盘交易！")
