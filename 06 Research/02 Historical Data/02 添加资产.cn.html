您必须将要分析的资产添加到QuantBook以请求它们的历史数据。添加资产使用与QCAlgorithm相同的API。有关这些方法的详细信息，请参阅初始化算法部分。 

qb = QuantBook()
spy = qb.AddEquity("SPY")    # 添加股票数据
eur = qb.AddForex("EURUSD")  # 添加外汇数据
btc = qb.AddCrypto("BTCUSD") # 添加虚拟货币数据
goog = qb.AddOption("GOOG")  # 添加GOOG的期权
es  = qb.AddFuture("ES")     # 添加SP500期货
xau = qb.AddCfd("XAUUSD", Resolution.Minute, Market.Oanda)  # 添加差价合约。需要指定Market.Oanda。
