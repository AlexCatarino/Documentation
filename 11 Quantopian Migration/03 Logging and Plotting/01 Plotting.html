In QuantConnect, plotting is usually done with the self.Plot("chart name", "series name", value), while plotting in Quantopian is simply record(series1_name=value, series2_name=value, etc.). While creating multiple plotting series through one line is possible in Quantopian, in QuantConnect, a separate call is needed for each series. Furthermore, all series with the same chart name string will be on the same chart, and new charts can be created by calling self.Plot with a different chart name. For example:

    class Algorithm(QCAlgorithm):
        ...
        def OnData(self, data):
            self.Plot('Chart1', 'Series1', 1)
            self.Plot('Chart1', 'Series2', 2)
            self.Plot('Chart2', 'c2 Series1', 3)

Will create two charts, one named "Chart1" with two series of data, and another named "Chart2" with one series of data. However, for users who wish to add the chart under the equity curve, simply use self.Plot("series name", value). More information on plotting can be found in the Charting section of our Documentation.
