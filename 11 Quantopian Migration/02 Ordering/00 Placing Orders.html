Placing orders with Quantopian is done with the following methods:

  • order_optimal_portfolio
  • order
  • order_value
  • order_percent
  • order_target
  • order_target_value
  • order_target_percent

On QuantConnect, can use the Buy, Sell, or SetHoldings methods. The Buy and Sell methods accept a quantity to buy or sell, while the SetHoldings method accepts a percentage of the portfolio to allocate. All of these methods execute trades with market orders. 

class MyAlgorithm(QCAlgorithm):
    def Initialize(self):
        self.SetStartDate(2020, 4, 28)
        self.SetCash(100000)
        self.AddEquity("SPY", Resolution.Minute)

    def OnData(self, data):
        if not self.Portfolio.Invested:
            self.SetHoldings("SPY", 1)

Instead of using Buy, Sell, or SetHoldings, we can also create orders of a particular type. We have the following methods available:

  • MarketOrder
  • LimitOrder
  • StopMarketOrder
  • StopLimitOrder
  • MarketOnOpenOrder
  • MarketOnCloseOrder

For more information on order types, refer to our Trading and Orders documentation.

Quantopian's order_optimal_portfolio method computes the optimial portfolio weights using an objective and contraints, then places the orders to achieve the desired portfolio. To acheive the same functionality with QuantConnect, we utilize portfolio construction models.
